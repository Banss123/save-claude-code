# Step 3: 스킬/에이전트 탐색 및 제안

업무가 확정되면 로컬에 설치된 skills, agents, plugins를 탐색한다.

- 활용 가능한 스킬/에이전트가 있으면 "이 업무에 활용 가능한 스킬이 있습니다:" 형태로 목록 제시
- 각 스킬의 역할을 한 줄로 설명
- 사용자가 선택하거나, 추천 조합 승인 시 사용
- 적합한 스킬이 없으면 없다고 말하고 일반 워크플로우 진행

## 스킬 실행 실패 시
1. 에러 내용 보고
2. 스킬 없이 수동 처리 가능한지 판단
3. 가능하면 대안 제시, 불가능하면 보고

## 자율 생성 허용
기존 스킬/에이전트로 커버 안 되는 업무라면 필요한 스킬이나 에이전트를 직접 생성해도 된다. 별도 승인 불필요.
