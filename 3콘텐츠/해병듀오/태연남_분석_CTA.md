- 멘트 템플릿 B (전화/연락처형):

## 반복 엔딩 공식

### 표준 엔딩 시퀀스 (80% 이상 영상 적용)

```
[본문 마무리] → [상담 CTA] → [구독/좋아요 (선택)] → [클로징 멘트]
```

**클로징 멘트 3가지 패턴:**

A. 기본형 (가장 많음):
"오늘 영상은 여기까지고요. 지금까지 태양광을 연구한 남자 태연남이었습니다. 시청해 주셔서 감사합니다."

B. 다음 영상 예고형 (초기 영상):
"오늘 영상은 여기까지고요. 태양광을 연구하는 남자 태연남이었습니다. 다음 영상에서 만나뵙겠습니다. 감사합니다."

C. 응원형:
"지금까지 시청해 주셔서 감사합니다. 태양광을 연구한 남자 현남이었습니다."
(초기에는 "연구하는", 후기에는 "연구한"으로 변화)

### 본문 마무리 → CTA 연결 패턴

**페인포인트 환기 후 CTA 연결 (가장 효과적인 구조):**
"[어려운 상황 요약] → 이런 경우 아래 고정 댓글 연락처로 연락 주시면 제가 직접 확인해서 알려 드리도록 하겠습니다."

- "이 과정이 혼자 번거롭거나 어렵다고 느껴지신 분들은 아래에 고정 댓글이나 영상 설명난 링크로 문의 주시면 제가 솔로 용량을 조회해서 알려 드리도록 하겠습니다."

## 터진 영상 vs 안 터진 영상의 CTA 차이

### 터진 영상 (10만+ 조회수) — 5개
| 영상 | 조회수 | CTA 특징 |
|---|---|---|
| 평생 후회하는 선택 10가지 | 625,012 | 상담 링크 + 감사 인사. CTA 매우 간결 |
| 주택용 태양광 사기 피해사례 | 295,012 | 공유 유도 + 고정 댓글 연락처 |
| 가정용태양광 불법건축물 합법화 | 281,423 | 전화번호 CTA + 구독 좋아요 |
| 주택용태양광 에어컨 몇시간 | 245,884 | 링크 CTA만 (간결) |
| 주택 8년 살아보고... | 213,194 | 댓글 경험 공유 + 링크 CTA |

**터진 영상 CTA 공통점:**
1. CTA가 짧고 명확 — 길게 설명하지 않음
2. "궁금하신 분들은 아래 댓글/링크 확인" 한 문장으로 마무리
3. 감정적 마무리 멘트가 CTA 앞에 먼저 배치 (클로징 감성 → 행동 유도 순서)
4. 구독/좋아요 언급 없거나 매우 가볍게 처리
5. 본문 마지막에 핵심 인사이트 한 줄 요약 후 CTA ("오늘 내용을 한 줄로 정리하면 이겁니다." 식)

### 안 터진 영상 (1만 이하) — 14개
**CTA 공통점:**
1. 상담 CTA가 길고 자세함 — 업체 자랑 포함
2. "저희는 연간 980건 이상 시공", "전국에 세 개의 지사가 있어" 등 업체 PR이 섞임
3. 구독/좋아요 유도가 더 자주, 더 강하게 등장
4. "지금 구독하지 않으면 이 채널을 다시 찾기 힘들 수도 있습니다" 등 압박성 멘트
5. CTA 직전 본문과 CTA 사이 연결이 어색 — 내용이 끝나자마자 갑자기 상담 유도

## 해병듀오 적용 CTA 제안

### 핵심 학습: 태연남 성공 공식
터진 영상의 패턴 = **"쓴소리로 신뢰 획득 → 감정적 공감 마무리 → 짧고 명확한 1개 행동 유도"**

안 터진 영상의 실수 = 업체 자랑 + 긴 상담 CTA + 구독 압박의 혼합

---

### 해병듀오 적용 CTA 시퀀스

**[엔딩 구조]**
```
본문 핵심 인사이트 1줄 요약
↓
공감/감정 마무리 1-2문장 ("이런 실수 저도 했습니다 / 이 정보가 000 준비하시는 분들께 도움이 되었으면 좋겠습니다")
↓
단일 CTA 1개만 (DM 또는 링크 또는 댓글)
↓
채널 시그니처 마무리 ("해병 두 명이 알려드렸습니다. 감사합니다.")
```

**CTA 멘트 템플릿 (태연남 패턴 변용):**

A. 상담 유도형:
"[주제]에 대해 더 자세한 상황 점검이 필요하신 분들은 아래 링크로 문의 주시면 직접 확인해서 알려드리도록 하겠습니다."

B. 공유 유도형 (폭로/경고 콘텐츠):
"이런 상황에 놓이신 분이 주변에 계신다면 이 영상 꼭 공유해 주세요."

C. 댓글 유도형 (경험 콘텐츠):
"여러분의 경험도 댓글로 공유해 주시면 비슷한 상황의 분들께 큰 힘이 될 겁니다."

**피해야 할 것:**
- 구독 좋아요 압박 ("지금 구독 안 하면 못 찾습니다" 류)
- 업체 자랑 섞기 ("저희는 몇 건 시공했고...")
- CTA 2개 이상 동시에 (링크 + 구독 + 공유 한꺼번에 X)
- 길고 설명적인 CTA (2-3문장 이내로)
