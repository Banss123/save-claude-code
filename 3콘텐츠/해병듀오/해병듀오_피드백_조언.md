## 🎯 병용형님 피드백 (핵심)

### 해병듀오 장점
- 후킹이 잘됨
- 해병대 컨셉 주목도 확실
- 영상미·컨셉 재밌음
- 첫 작품 치고 퀄리티 좋음

### 핵심 문제점
> **사업적 메시지가 없다 = 전환이 불가능하다**
>
> **트래픽 모아도 설득 못하면 날리는 것.**

### 첫 영상 문제점
1. 설득 구조 없음
2. 기능 중심 메시지 → 피로도 증가
3. 키워드가 안 좋음 (해외마케팅)
4. 감정 자극 없음
5. CTA(행동 유도) 없음

### 피로도 높은 메시지 (금지)
- "구글 상위노출"
- "샤오홍슈 마케팅"
- "일본인 마케팅"

업주는 이미 100번 들음 → "그래서 어쩌라고?" 상태.

### 고객이 진짜 듣고 싶은 것 = **결과**
- 매출
- 사례
- 사회적 증거
- 남들이 실제로 얼마 벌었는지
- 진짜로 가능해 보이는지

**업주 감정 움직이는 단 하나의 키워드 = 매출**

## 📝 권장 콘텐츠 구조

### 3단 구조

**1. 문제 제시**
```
"요즘 외국인 매출 안 늘죠?"
"상권 좋은데 매출이 안 터지는 이유가 있습니다."
```

**2. 사회적 증거 / 사례**
```
A족발집: 900만 → 1,700만
B횟집: 1,200만 → 2,400만
C한식집: 500만 → 1,200만
"이런 사례만 100개 넘습니다."
```

**3. CTA**
```
외국인을 잡아준다 → 과정
결과 → 매출 올려준다
"매출 무료 진단"
"100만원짜리 전자책 무료 제공"
```

## 🔍 25.12.09 문제점 분석

### 업주들의 기본 상태
- 마케팅 자체의 피로도 높음
- "~하셔야 합니다" → 피로도 높은 메시지 → 뻔한 소리 취급
- 업주도 이미 앎: "네이버플레이스 상위노출, 샤오홍슈 체험단, 구글 상위노출하면 매출 오른다"

### 피로도 없는 정보 (필요)
- 업주들에게 **필요한/모르는/도움되는 정보**
- **뾰족하게 찔러주기**
- 예: 구체적으로 구글맵을 어떻게 세팅해주는지

### 이성 < 감성
- 이성 - 청소나 일 열심히 하는 것
- 감성 - 돈 벌고 싶다
- **이성 설득보다 감정 공감이 구매 결정에 더 큼**

## 🎯 감정 자극 전략

### 사례로 후킹 (3초 / 7초)
```
"월세 걱정에 잠 못 이루던 사장님"
"얼음장같이 차가운 방바닥에서 성공을 꿈꾸던 시절"
```
→ 구체적·생생한 묘사로 동경/페인 자극

### 동경 vs 페인 → **1 : 3 비율**
- 동경: 매출 올라가는 것 (이득)
- 페인: 매출 못 올리면 끝장 (손해)

### 권위 (전문가 신뢰성)
```
"200개 매장 관리 중"
"파마 전문 미용실 800건 시술 경험"
"임플란트 시술 횟수 5,000건"
```

### 사회적 증거 (실제 성공 사례)
```
"매출이 2배 상승한 A 음식점 사례"
"월세 밀리던 가게가 대박 난 B 매장 사례"
```

## 📹 영상 구성 전략 (기승전결 뒤집기)

### 후킹 (0~3초)
- 3초 안에 시선 사로잡기
- 인사말·뻔한 질문 금지
- 강력한 결과/충격적 사실 먼저
- 예: "망해가던 가게가 이렇게 변했습니다", "매출 3배 상승 비결 공개"

### 구체성
- 모호한 표현 금지
- 구체 수치·사례 사용
- "매출이 많이 올랐습니다" ❌
- "월 매출 1,000만 원 → 3,000만 원, 200% 상승" ✅

### 간결성
- 짧고 간결하게 끊기
- 한 문장 **40~50자 이내**
- 불필요한 접속사·수식어 제거

## 📊 메타 광고 vs 자연 트래픽

- 초기: 유튜브 쇼츠, 인스타 릴스 등 자연 트래픽 활용
- 광고 효율(ROAS) 고려
- **콘텐츠 자체의 매력**이 중요
- **초반: 메타광고 + 브랜딩**으로 터질 때까지
- **월 계약 30개 목표** 그냥 딸 것으로 예상

## 💰 사장님 설득 핵심 멘트 (3배 법칙)

```
"사장님, 외국인은 오늘 아니면 못 먹는다는 생각으로 지갑을 엽니다.
내국인은 단골이 되어야 매출이 쌓이지만,
외국인은 한 번의 방문으로 단골 3인분의 매출을 한 번에 꽂아줍니다.
이 3배의 기회를 잡으시겠습니까, 아니면 그냥 지나가게 두시겠습니까?"
```

### 의료 마케팅 3배 법칙 (의료 관광)
- 보건복지부 외국인 환자 유치 실적 근거
- 내국인: 건강보험·일상 진료 → 객단가 낮음
- 외국인: 성형·피부·검진 비급여 중심 → **1인당 평균 진료비 3.2~5배 높음**

```
"원장님, 한국 환자 3명 진료할 시간에
외국인 환자 1명만 제대로 잡으면 수익 구조가 바뀝니다."
```

### 쇼핑·부가 지출 시너지
- 외국인 관광객 카드 지출: 식비 + 쇼핑 긴밀 연결
- **외국인 전용 메뉴판 / 다국어 SEO 매장**: 주류·추가 메뉴 주문율 **2.8배 높음** (카드사 빅데이터 분석)

## ✍️ 스레드 운영 가이드

### 기본 원칙
- 게시물 하루 10~15개
- 댓글 하루 10~15개

### 글 작성 공식
1. **글자수**: 짧은 게 트렌드 (한 줄 20자 이하 / 총 120자 이하)
2. **표본 넓히기**: "역삼동 맛집" ❌ → "서울 맛집" ✅
3. **사진/영상 필수** (주제 관련 뭐든 OK)
4. **첫 줄 = 후킹**
5. **숫자로 말하기**: "난 월에 1천만원 벌어" / "~하는 법 5단계"
6. **권위자 이용**: "일론머스크가 가장 중요시 생각하는 ~~"
7. 과정만 올리고 이득 없는 영상 ❌
8. **댓글 유도로 이득 주기** (반응·체류시간 확보)
