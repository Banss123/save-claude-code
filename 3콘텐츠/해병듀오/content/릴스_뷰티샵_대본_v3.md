# 릴스 대본 — 뷰티샵 타겟 광고 소재 v3

> **목적:** 메타 광고 소재 → 인스턴트 양식 → 리드 수집
> **타겟:** 뷰티샵 원장 (미용실, 반영구, 네일, 속눈썹 등 통칭)
> **출연:** 민재 or 재원 (1인 토킹헤드)
> **길이:** 25~35초
> **톤:** 해병듀오 — 직설, 자신감, 명령조
> **참고:** 현재 집행 중인 병의원 광고 소재 구조

---

## 구조 원칙

병의원 대본에서 검증된 흐름을 뷰티샵에 맞게 적용.

```
[훅] 질문 + 전복 ("틀렸습니다" 패턴)
  ↓ 자연스럽게
[시장 기회] 외국인 수요가 있다는 팩트 → "근데 왜 안 오나?"
  ↓ 자연스럽게
[경쟁 공포] 이미 움직이는 원장님들 존재 → FOMO
  ↓ 자연스럽게
[권위 + 사례] 구체 수치 + 실제 결과
  ↓ 자연스럽게
[CTA] 무료 진단 + 자격 제한 + "실시!"
```

**핵심:** 각 구간이 다음 구간의 이유가 되어야 함. "그래서?" → "왜?" → "어떻게?" 가 자동으로 이어지도록.

## 대본 A — 전복형 (메인 추천)

> **훅 심리:** 확증 편향 깨기 + 손실 회피
> **길이:** 약 32초
> **참고:** 병의원 대본의 "100만원 아끼는 게 이득?" 패턴 차용

---

**[훅 — 0~5초]**

> "원장님, 외국인 손님 없어도 괜찮다고 생각하십니까?
> 틀렸습니다.
> 지금도 외국인들은 구글 켜서 옆 가게 예약 넣고 있습니다."

---

**[시장 기회 — 5~12초]**

> "지금 외국인들 사이에서 케이뷰티는 난리입니다.
> 반영구, 네일, 헤어 받으려고 한국행 비행기 끊는 사람들이 쏟아지고 있어요.
> 근데 이 사람들이 원장님 매장에 왔습니까?
> 안 왔다면 이유는 하나입니다.
> 원장님이 못하는 게 아닙니다. 외국인이 쓰는 플랫폼에 안 보이는 겁니다."

---

**[경쟁 공포 — 12~18초]**

> "이미 빠르게 움직이는 원장님들은
> 외국인이 실제로 검색하는 플랫폼에서
> 노출, 리뷰, 예약 이 세 가지를 전부 잡고 계십니다.
> 늦으면 늦을수록, 그 손님들은 전부 옆 가게로 갑니다."

---

**[권위 + 사례 — 18~28초]**

> "저희 해병듀오마케팅은 현재 300개가 넘는 매장 데이터로
> 이 세 가지를 직접 세팅하고 관리해드리고 있습니다.
>
> 실제로 저희와 함께한 홍대 A뷰티샵은
> 세 달 만에 외국인 유입 3배 증가,
> 월 매출 1,000만원 추가 달성하셨습니다."

---

**[CTA — 28~33초]**

> "해병듀오가 무료로 진단해드립니다.
> 진짜 매출 올릴 자신 있는 원장님만,
> 아래 클릭해서 지금 당장 문의한다. 실시!"

## 촬영 큐시트 (v3 공통)

### 촬영 핵심 — 자연스러운 흐름

v2까지는 구간별로 컷이 뚝뚝 끊겼음. v3는 **한 호흡으로 말하는 느낌** 유지.

| 원칙 | 적용 |
|------|------|
| 원테이크 기본 | 전체를 한 번에 쭉 말하는 테이크 최소 3회 |
| 점프컷은 편집에서 | 촬영은 끊지 않고, 편집에서 뜸만 제거 |
| 앵글 고정 | 미디엄샷 하나로 통일 (훅/CTA만 살짝 줌인 편집) |
| 표정 전환 자연스럽게 | 훅(진지) → 기회(밝아짐) → 공포(다시 진지) → 사례(자신감) → CTA(에너지) |

### 촬영 순서

```
1. 대본 A 풀 테이크 × 3회
2. 대본 B 풀 테이크 × 3회
3. 대본 C 풀 테이크 × 3회
4. CTA "실시!" + "아-잇!" 별도 3회
5. 여유 있으면: 훅만 따로 촬영 (다른 훅으로 교체 테스트용)
```

### 촬영 체크리스트

- [ ] 조명: 자연광 or 링라이트
- [ ] 배경: 깔끔한 단색 벽 or 사무실
- [ ] 오디오: 핀마이크 필수
- [ ] 의상: 검정 or 진한 색 (빨간 자막과 대비)
- [ ] 세로 9:16 확인
- [ ] **원테이크로 쭉 말하기 연습 후 촬영**
- [ ] 대본 A/B/C 각 3테이크 이상
- [ ] "실시!" / "아-잇!" 별도 녹음

### 상단 해병듀오 로고 (상시 노출)

실제 편집본 기준 상단 로고가 영상 내내 고정 노출됨. 표준 스펙으로 확정:

| 항목 | 값 |
|------|-----|
| 위치 | **상단 중앙** (필요 시 조정) |
| 크기 | 화면 폭의 20~25% (프로필 인식 가능 수준) |
| 상하 여백 | 상단에서 80~120px (4K 기준) |

> ⚠ 로고 구체 위치/크기는 실제 편집본 확인 후 최종 확정 필요. 현재 값은 일반 권장치.

### BGM

| 항목 | 설정 |
|------|------|
| 분위기 | 긴장감 트랩 비트 → 사례 구간에서 밝아짐 |
| 볼륨 | 음성 대비 -12~-15dB |
| 훅 | 서스펜스 / 비트 없이 시작 |
| "틀렸습니다" | 비트 인 |
| 시장 기회 | 비트 유지 |
| 경쟁 공포 | 비트 올림 |
| 사례 | 비트 밝아짐 (동경 구간) |
| CTA | 비트 유지 → "실시!"에서 컷 |
