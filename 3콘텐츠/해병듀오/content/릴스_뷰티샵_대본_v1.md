# 릴스 대본 — 뷰티샵 타겟 광고 소재 v1

> **목적:** 메타 광고 소재 → 인스턴트 양식 리드 수집
> **타겟:** 미용실, 반영구, 뷰티샵 원장 (병원 제외)
> **출연:** 민재 or 재원 (1인 토킹헤드)
> **길이:** 20~30초 (메타 광고 최적 구간)
> **촬영:** 세로 9:16

---

## 훅 후보 (택 1)

| # | 훅 문장 | 타입 | 비고 |
|---|---------|------|------|
| A | "인스타 매일 올리는데 예약은 왜 안 느나요?" | 공감형 | 가장 보편적 고통 |
| B | "블로그 6개월 했는데 신규 예약 0건이요?" | 충격형 | 숫자로 현실 자극 |
| C | "소개 손님만 기다리는 원장님, 이거 보세요" | 지목형 | 타겟 직접 호명 |
| D | "마케팅 대행사 3군데 맡겨봤는데 다 똑같았다고요?" | 공감+도발형 | 경쟁사 불만 공략 |
| E | "월 110만원으로 예약 40건 만든 방법" | 결과 제시형 | 호기심 유발 |

**추천:** A 또는 C (뷰티샵 원장의 일상 언어에 가장 가까움)

---

## 대본 (훅 A 버전 기준)

### 구조: 훅(3초) → 공감(5초) → 해결(10초) → CTA(5초) = 약 23초

---

**[1] 훅 — 0~3초**

> "인스타 매일 올리는데 예약은 왜 안 느나요?"

- 화면: 카메라 정면 응시, 말하면서 살짝 다가옴
- 톤: 진지 + 약간 도발적

---

**[2] 공감 — 3~8초**

> "네이버 블로그도 하고, 인스타도 올리고,
> 근데 신규 손님은 여전히 소개뿐이죠."

- 화면: 상체 미디엄샷 유지
- 톤: 공감, 고개 살짝 끄덕이며

---

**[3] 해결 — 8~20초**

> "저희가 뷰티샵 전문으로 광고를 세팅해드리는데요,
> 실제로 3개월 만에 월 신규 예약 40건 넘게
> 만들어드린 적 있습니다.
> 블로그 말고, 광고로 '진짜 예약할 사람'만
> 데려오는 거예요."

- 화면: 중간에 성과 수치 텍스트 오버레이 삽입
- 톤: 자신감 있게, 속도 약간 올림
- 포인트: "진짜 예약할 사람" 강조

---

**[4] CTA — 20~25초**

> "지금 아래 버튼 눌러서 무료 상담 신청해보세요.
> 딱 3분이면 됩니다."

- 화면: 카메라에 더 가까이, 아래쪽 가리키는 제스처
- 톤: 가볍고 부담 없게

## 촬영 큐시트

| 컷 # | 구간 | 화면 구성 | 출연자 액션 | 비고 |
|-------|------|----------|------------|------|
| 1 | 훅 (0~3초) | 바스트샷 → 살짝 줌인 느낌 | 카메라 정면, 한 발짝 다가옴 | 첫 마디 전 0.5초 무음 정적 (편집에서 활용) |
| 2 | 공감 (3~8초) | 미디엄샷 고정 | 고개 끄덕, 손 제스처 가볍게 | 배경 깔끔한 곳 (카페/사무실) |
| 3 | 해결 (8~20초) | 미디엄샷 유지 | 자신감 있는 표정, 손가락으로 포인트 짚기 | "40건" 말할 때 손가락 강조 |
| 4 | CTA (20~25초) | 클로즈업 | 아래쪽 가리키며 미소 | 마지막 1초 정지 프레임 활용 가능 |

### 촬영 체크리스트

- [ ] 조명: 자연광 or 링라이트 (얼굴에 그림자 없이)
- [ ] 배경: 깔끔한 단색 or 사무실 (어수선한 배경 X)
- [ ] 오디오: 핀마이크 필수 (메타 광고는 음소거 시작이 많으므로 자막이 핵심이지만, 음질도 중요)
- [ ] 의상: 깔끔한 비즈니스 캐주얼 (신뢰감)
- [ ] 테이크: 각 컷 최소 3테이크 (편집 선택지 확보)
- [ ] 세로 촬영 확인 (9:16)

## 프리미어 프로 편집 가이드

### 시퀀스 설정

| 항목 | 값 |
|------|-----|
| 해상도 | 1080 x 1920 (세로 9:16) |
| 프레임레이트 | 30fps |
| 타임베이스 | 29.97fps도 OK |

### 편집 구조 (타임라인)

```
V3: [텍스트/자막 레이어]
V2: [그래픽 오버레이 - 성과 수치, 하단 CTA 바]
V1: [메인 영상 클립]
A1: [출연자 음성]
A2: [BGM]
```

### 컷별 편집 디테일

| 구간 | 편집 포인트 | 프리미어 적용 |
|------|------------|--------------|
| **훅 (0~3초)** | 0.5초 무음 → 첫 마디에 줌인 키프레임 | Position/Scale 키프레임: 100% → 105% (0.3초간) |
| **전환 (3초)** | 훅→공감 넘어갈 때 점프컷 | 단순 컷 (디졸브 쓰지 말 것) |
| **공감 (3~8초)** | 핵심 키워드 자막 강조 | "소개뿐" 부분 색상 변경 (노란색 or 주황색) |
| **해결 (8~20초)** | 성과 수치 팝업 | "월 신규 예약 40건" 모션그래픽 (에센셜 그래픽스) |
| **해결 중간** | 리듬 유지용 점프컷 2~3회 | 말 사이 뜸 제거, 0.5초 이상 공백 컷 |
| **CTA (20~25초)** | 하단 CTA 바 + 아래 화살표 | 깜빡이는 화살표 애니메이션 (1초 루프) |
| **아웃트로 (25초~)** | 1초 정지 프레임 + 페이드 | 마지막 프레임 Freeze Frame → 0.5초 페이드 아웃 |

### 자막 스타일

| 항목 | 설정 |
|------|------|
| 폰트 | Pretendard Bold (or 프리텐다드) |
| 크기 | 60~70pt (모바일 가독성) |
| 위치 | 화면 하단 1/3 (Safe Zone) |
| 배경 | 반투명 검정 박스 (opacity 60~70%) |
| 강조 | 핵심 키워드만 노란색 (#FFD700) or 주황색 |
| 방식 | 전체 문장 X → 2~3단어씩 끊어서 표시 |

**자막 끊기 예시:**
```
[0.0~1.5초] 인스타 매일 올리는데
[1.5~3.0초] 예약은 왜 안 느나요?
[3.0~5.0초] 블로그도 하고 인스타도 올리고
[5.0~8.0초] 신규 손님은 여전히 소개뿐
[8.0~11.0초] 저희가 뷰티샵 전문으로
[11.0~14.0초] 광고를 세팅해드리는데요
[14.0~17.0초] 3개월 만에 월 신규 예약 40건
[17.0~20.0초] 진짜 예약할 사람만 데려옵니다
[20.0~23.0초] 아래 버튼 눌러서
[23.0~25.0초] 무료 상담 신청하세요
```

### BGM

| 항목 | 설정 |
|------|------|
| 분위기 | 밝고 경쾌한 로파이 or 미니멀 비트 |
| 볼륨 | 음성 대비 -15~-20dB (음성이 확실히 들려야 함) |
| 시작 | 훅 시작과 동시에 페이드인 (0.5초) |
| 드롭 | 해결 파트 시작(8초)에서 비트 살짝 올림 |
| 종료 | CTA 끝나며 페이드아웃 (1초) |
| 추천 소스 | Artlist, Epidemic Sound (저작권 안전) |

### 컬러/톤 보정

| 항목 | 설정 |
|------|------|
| 전체 톤 | 밝고 따뜻한 톤 (뷰티 타겟이므로 차갑지 않게) |
| Lumetri | 색온도 살짝 웜 (+5~10), 대비 약간 올림 |
| 피부톤 | 자연스럽게 (과도한 보정 X, 신뢰감 저하) |

### 내보내기 설정

| 항목 | 값 |
|------|-----|
| 포맷 | H.264 |
| 해상도 | 1080 x 1920 |
| 프레임레이트 | 30fps |
| 비트레이트 | VBR 2패스, 타겟 15Mbps / 최대 20Mbps |
| 오디오 | AAC 320kbps |

## A/B 테스트 제안

메타 광고에 올릴 때 최소 2개 변형을 만들어서 테스트:

| 변형 | 차이점 | 테스트 목적 |
|------|--------|------------|
| **v1-A** | 훅 A "인스타 매일 올리는데~" | 공감형 훅 성과 |
| **v1-C** | 훅 C "소개 손님만 기다리는 원장님~" | 지목형 훅 성과 |

나머지 구간은 동일하게, **훅만 다르게** 촬영 (같은 세션에서 훅만 2번 찍으면 됨).
CPL 비교 후 승자 훅 스타일로 다음 소재 방향 결정.
