# 릴스 대본 — 뷰티샵 타겟 광고 소재 v2 (최종)

> **목적:** 메타 광고 소재 → 인스턴트 양식 → 리드 수집
> **타겟:** 미용실, 반영구, 뷰티샵 원장 (병원 제외)
> **출연:** 민재 or 재원 (1인 토킹헤드)
> **길이:** 20~25초
> **촬영:** 세로 9:16
> **톤:** 해병듀오 — 직설, 명령조, 자신감, 군기

---

## 적용 원칙 (병용형님 피드백 + 심리학 코어)

| 원칙 | 적용 |
|------|------|
| 결론 먼저 (기승전결 뒤집기) | 훅에서 결과/충격부터 제시 |
| "해외마케팅" 직접 사용 금지 | 결과(외국인 예약, 매출 수치)로 표현 |
| 감정 > 이성 | 월세 걱정, 돈 욕구, 경쟁심 자극 |
| 구체적 수치 필수 | "매출 올랐다" X → "800에서 2,400" O |
| 사례 스토리텔링 | 데이터보다 구체적 사례 이야기 |
| 손실 회피 (2.5배 법칙) | 이득보다 "놓치고 있다/흘리고 있다" 프레임 |
| 사회적 증거 + 권위 | "200개 매장", "검증된 방법" |
| 문장 40~50자 이내 | 짧고 끊어서, 명령조 |
| 페인:동경 = 1:3 | 영상 전체에서 동경(매출 상승) 위주 |

## 대본 A — 결과 제시형 (훅 #1)

> **구조:** 결과(3초) → 상황 설명(5초) → 방법(8초) → 사례 증거(4초) → CTA(3초) = 약 23초
> **톤:** 자신감, 단언, 빠른 템포

---

**[1] 훅 — 0~3초**

> "월 매출 800이던 반영구샵.
> 3개월 만에 2,400 찍었습니다."

- 화면: 정면 클로즈업, 눈 맞춤
- 자막: "800 → 2,400" 빨간색 굵은 숫자 강조
- 편집: 첫 0.3초 무음 → 첫 마디에 줌인 펀치

---

**[2] 상황 — 3~8초**

> "국내 손님만 받다가 한계 왔었거든요.
> 근데 외국인 손님이 들어오기 시작하니까
> 게임이 바뀌었습니다."

- 화면: 미디엄샷, 점프컷 2회
- 자막: "외국인 손님" / "게임이 바뀌었습니다" 강조
- 편집: "게임이 바뀌었습니다"에서 효과음 (타격음/비트 드롭)

---

**[3] 방법 — 8~16초**

> "외국인들이 검색하는 플랫폼에
> 원장님 가게가 뜨게 만들어드립니다.
> 네이버 말고. 외국인이 실제로 보는 곳에서요."

- 화면: 미디엄샷, 손가락으로 포인트
- 자막: "네이버 말고" 빨간 X 표시 그래픽
- 편집: "네이버 말고"에서 점프컷 + 효과음

---

**[4] 증거 — 16~20초**

> "이미 이 방법으로 결과 만든 매장,
> 100개 넘습니다."

- 화면: 클로즈업, 자신감 표정
- 자막: "100개+" 큰 숫자 팝업
- 편집: 숫자 팝업 모션그래픽 (에센셜 그래픽스)

---

**[5] CTA — 20~23초**

> "매출 올리고 싶은 원장님.
> 아래 버튼 눌러서 무료 진단 받아보십시오. 실시!"

- 화면: 클로즈업 → 아래 가리킴
- 자막: "무료 진단" 빨간 박스 강조 + 아래 화살표
- 편집: "실시!" 시그니처 + 하단 CTA 바 등장
- 아웃트로: "아-잇!" 시그니처 사운드

## 대본 B — 손실 회피형 (훅 #4)

> **구조:** 충격(3초) → 원인(5초) → 해결(7초) → 사례(3초) → CTA(3초) = 약 21초
> **톤:** 긴급, 직설, 약간의 공포

---

**[1] 훅 — 0~3초**

> "지금 이 순간에도 외국인 손님이
> 원장님 말고 옆 가게 예약하고 있습니다."

- 화면: 정면 클로즈업, 심각한 표정
- 자막: "옆 가게" 빨간색 강조
- 편집: 긴장감 있는 BGM 시작

---

**[2] 원인 — 3~8초**

> "외국인은 네이버 안 씁니다.
> 원장님 가게가 안 보이니까
> 존재 자체를 모르는 겁니다."

- 화면: 미디엄샷, 점프컷
- 자막: "안 보이니까" → "모르는 겁니다" 강조
- 편집: "안 씁니다"에서 컷 전환 + 타격음

---

**[3] 해결 — 8~15초**

> "저희가 외국인이 실제로 검색하는 곳에
> 원장님 가게를 올려드립니다.
> 예약까지 바로 들어오게."

- 화면: 미디엄샷, 자신감 전환
- 자막: "예약까지 바로" 강조
- 편집: 톤 전환 — BGM 밝아짐

---

**[4] 사례 — 15~18초**

> "실제로 예약 0건에서 하루 5건까지
> 만든 매장 있습니다."

- 화면: 클로즈업
- 자막: "0건 → 하루 5건" 숫자 모션
- 편집: 숫자 카운트업 애니메이션

---

**[5] CTA — 18~21초**

> "무료 매장 진단, 지금 신청하십시오. 실시!"

- 화면: 아래 가리킴 + 미소
- 자막: "무료 매장 진단" 빨간 박스
- 아웃트로: "아-잇!"

## 대본 C — 희소성형 (훅 #7)

> **구조:** 충격 수치(4초) → 기회 설명(6초) → 해결(6초) → CTA(4초) = 약 20초
> **톤:** 정보 제공 + 기회 강조

---

**[1] 훅 — 0~4초**

> "이거 하고 있는 뷰티샵, 전체의 5%도 안 됩니다.

- 화면: 정면, 진지한 표정
- 자막: "5%" 작게 → "95%" 크게, 빨간색
- 편집: "돈을 바닥에 흘리고"에서 동전 효과음

---

**[2] 기회 — 4~10초**

> "한국 와서 반영구 받는 일본인,
> 작년보다 2배 늘었습니다.
> 이 손님들 객단가가 국내 손님의 3배에서 5배."

- 화면: 미디엄샷
- 자막: "2배" / "3~5배" 숫자 강조
- 편집: 점프컷으로 리듬감

---

**[3] 해결 — 10~16초**

> "근데 대부분 뷰티샵이
> 이 손님들을 못 잡고 있습니다.
> 저희가 잡아드리겠습니다."

- 화면: 클로즈업, 확신 표정
- 자막: "잡아드리겠습니다" 군대 폰트 강조
- 편집: "잡아드리겠습니다"에서 비트 드롭

---

**[4] CTA — 16~20초**

> "외국인 매출 올리고 싶은 원장님만.
> 아래 버튼, 지금 누르십시오. 실시!"

- 화면: 아래 가리킴

- 아웃트로: "아-잇!"

## 촬영 큐시트 (공통)

### 촬영 원칙 (해병듀오 스타일)

- **컷 전환: 1~2초마다** (느리면 집중력 깨짐)
- **3테이크 이상**: 각 구간별 최소 3번 촬영 (편집 선택지)
- **표정 전환 중요**: 훅(심각) → 해결(자신감) → CTA(밝음)
- **제스처 크게**: 군인 느낌의 확실한 손동작
- **"실시!" / "아-잇!"**: 마지막에 반드시 녹음

### 컷 구성

| 컷 # | 구간 | 화면 | 출연자 | 비고 |
|-------|------|------|--------|------|
| 1 | 훅 (0~3초) | 클로즈업 | 카메라 정면, 강한 눈빛 | 0.3초 무음 후 첫 마디 |
| 2 | 상황/원인 (3~8초) | 미디엄샷 | 점프컷용 같은 앵글 여러 테이크 | 뜸 제거를 위해 끊어서 촬영 |
| 3 | 해결 (8~16초) | 미디엄샷 | 포인트 짚는 손가락, 자신감 | "실제로" "만든" 단어에서 힘 |
| 4 | 증거 (16~20초) | 클로즈업 | 확신 표정 | 숫자 말할 때 강조 |
| 5 | CTA (20~23초) | 클로즈업 | 아래 가리킴 + "실시!" | 끝에 "아-잇!" 별도 촬영 |

### 촬영 체크리스트

- [ ] 조명: 자연광 or 링라이트 (그림자 없이)
- [ ] 배경: 깔끔한 단색 벽 or 사무실
- [ ] 오디오: 핀마이크 필수
- [ ] 의상: 검정 or 진한 색 (빨간 자막과 대비)
- [ ] 세로 9:16 확인
- [ ] 각 컷 3테이크 이상
- [ ] A/B 테스트용 훅 2~3개 추가 촬영

### 자막 스타일 (해병듀오 전용)

| 항목 | 설정 |
|------|------|
| 폰트 | **각진 고딕** (예: 에스코어드림 Bold, 배달의민족 한나체, 고딕 A1) |
| 크기 | 65~75pt (모바일 가독성, v1보다 약간 크게) |
| 위치 | 화면 중앙~하단 1/3 |
| 배경 | 반투명 검정 (opacity 70%) or 빨간 박스 |
| 기본 색상 | 흰색 |
| 강조 색상 | **빨간색 (#FF0000)** — 핵심 숫자, 키워드 |
| 보조 강조 | 노란색 (#FFD700) — 수치 대비용 |
| 방식 | 2~3단어씩 끊어서, 명령문 스타일 |

| **해결 파트** | 리듬 유지 점프컷 2~3회 | 뜸 0.3초 이상 전부 제거 |
| **사례 수치** | 카운트업 애니메이션 | 에센셜 그래픽스 or 별도 모션 |
| **CTA** | 하단 바 + 화살표 | 깜빡이는 화살표 (1초 루프), CTA 바 슬라이드업 |

## A/B 테스트 계획

### 촬영 시 훅만 바꿔서 3개 버전 촬영

| 변형 | 훅 | 테스트 목적 |
|------|-----|-----------|
| **v2-A** | #1 "월 매출 800이던 반영구샵~" | 결과 제시형 성과 |
| **v2-B** | #4 "지금 이 순간에도~" | 손실 회피형 성과 |
| **v2-C** | #7 "이거 하고 있는 뷰티샵 5%~" | 희소성형 성과 |

- 훅 이후 본문은 대본 A 기준으로 통일
- 3일간 CPL 비교 → 승자 훅 스타일로 다음 소재 방향

### 승자 훅 판단 기준

| 지표 | 의미 | 우선순위 |
|------|------|---------|
| CPL (결과당 비용) | 리드 1건 뽑는 데 얼마? | 1순위 |
| CTR (클릭률) | 훅이 스크롤 멈추게 했나? | 2순위 |
| 영상 시청률 (75% 이상) | 끝까지 봤나? | 3순위 |
