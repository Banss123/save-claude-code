# 태연남tv 후킹 패턴 분석

> 분석일: 2026-04-19
> 분석 대상: 29개 영상 / 조회수 1,345 ~ 625,012
> 기준: 각 영상 처음 10초 자막 (VTT 타임스탬프 기준)

## 후킹 공식 TOP 5

### 1. 손해·후회 예고형 ("이거 모르면 손해")
- 특징: 시청자가 이미 저지르고 있거나 저지를 가능성이 높은 실수를 먼저 콕 집는다. 공포 감정을 건드려 이탈을 막는다.
- 예시 A: "태양광 설치하고 후회하는 집들 생각보다 많습니다. 괜히 했네. 왜 이런 말이 나올까요?" (76,169회)
- 예시 B: "주택을 지어놓고 살아보니 후회하는 일들이 생각보다 훨씬 많습니다." (625,012회)
- 예시 C: "이 3가지 모르면 100% 후회합니다" (14,726회, 제목 직접 활용)
- 사용 영상: 7개 이상 (전체의 약 25%)

### 2. 피해사례·실화 투입형 ("실제로 있었던 일입니다")
- 특징: 첫 3-7초 안에 실존 인물(60대 부부, 70대 어르신, 경기도 용인 거주자 등)의 구체적 상황을 던진다. 픽션이 아님을 각인시켜 신뢰를 확보한다.
- 예시 A: "얼마 전 저에게 한 통의 전화가 걸려왔는데요. 70대 어르신이셨는데 태양광에 투자를 했는데 이제 와서 보니까 모든 게 거짓말이었다라는 내용이었습니다." (69,145회)
- 예시 B: "어제 AS 시골에 한 집을 방문했었습니다. 거기서 60대 부부를 만났는데 이분들의 이야기를 듣고 마음이 무거워졌습니다. 전기요금이 매달 8만 원 정도 나오는데 이것만 좀 줄이면 생활이 편해지지 않겠냐?" (119,485회)
- 예시 C: "실제로 있었던 일입니다. 경기도 용인에 사시는 분이..." (46,618회)
- 사용 영상: 5개

### 3. 내부자 폭로형 ("업체들은 절대 말 안 해주는")
- 특징: 화자가 업계 내부 사정을 아는 사람임을 자처하며, 시청자 편에서 감춰진 정보를 공개한다는 구도를 만든다. "적"(업체·사기꾼)과 "동지"(시청자) 구도가 핵심.
- 예시 A: "여러분이 정말 궁금해하시는 주택 태양광 3kw 설치 원가를 싹 다 공개하려고 합니다. 어깨에서 용목을 각오로 소비자 여러분의 알권리를 위해 솔직하게 말씀드릴 텐데요." (149,442회)
- 예시 B: "태양광 업자가 이 말을 했다면 사기꾼입니다" (69,145회, 제목)
- 예시 C: "설치 업자들이 감추는 3가지 진실" (6,945회, 제목)
- 사용 영상: 4개

### 4. 반박 질문형 ("이런 생각 해보셨나요? 그런데...")
- 특징: 시청자가 당연하다고 여기는 생각(통념)을 먼저 소환한 뒤 "그런데 사실은 다릅니다"로 뒤집는다. 인지 불협화음을 유발해 클릭 유지를 강요한다.
- 예시 A: "태양광도 중고로 사면 훨씬 싸지 않을까? 어차피 발전만 되면 되는데 중고나 새 거나 뭐가 다르겠어? 혹시 이런 생각 해 보셨나요?" (6,588회)
- 예시 B: "태양광 설치하면 전기료 줄어든다는 말은 많이 들으셨죠? 근데 막상 설치를 알아보려고 하면 실제로 전기료가 얼마나 줄어드는 거지?" (40,348회)
- 예시 C: "보조금 나오면 그때 설치해야지. 태양광 알아보시는 분들을 보면 대부분 이렇게 생각하십니다. 그런데 이거 하나 모르고 기다리다 아예 설치를 못 하게 되는 경우도 많습니다." (24,797회)
- 사용 영상: 5개

### 5. 즉각 혜택 약속형 ("이것만 누르면 24시간 내내 틀어도 전기료 폭탄 없음")
- 특징: 첫 3초 안에 구체적 행동 + 즉각 결과를 동시에 제시한다. 시청자가 영상을 끝까지 보면 실질적 이득이 있다는 계약을 맺는 구도다.
- 예시 A: "에어컨 리모컨에서 이것만 누르면 24시간 내내 틀어도 전기료 폭탄 받지 않습니다. 여러분은 지금까지 이 사용법을 몰라서 매달 10만 원 이상 손해 보고 계셨습니다." (29,785회)
- 예시 B: "불법 건축물에 단속이 되면 보통은 철거 대상입니다. 이행 강제금의 과태료까지 정말 골치 아픈 문제가 아닐 수 없는데요. 철거를 하지 않고도 합법적으로 해결할 수 있는 방법이 있다면 어떨까요?" (281,423회)
- 사용 영상: 4개

## 터진 영상 vs 안 터진 영상 후킹 차이

| 구분 | 터진 영상 (10만+) | 안 터진 영상 (1만 이하) |
|------|-------------------|------------------------|
| 첫 3초 | 페인·후회·폭로 감정 직격 | 주제 소개 또는 상황 설명으로 시작 |
| 화자 포지션 | 피해를 직접 목격한 내부자 | 정보 제공자 (중립적) |
| 구체성 | 실존 인물, 지역명, 금액 명시 | 일반적 상황 묘사 |
| 갈등 구조 | 피해자 vs 사기꾼 / 모르는 나 vs 아는 채널 | 정보 전달 위주, 갈등 없음 |
| 긴급성 | "지금 당장 모르면 손해" | "알아두면 좋은 정보" |

### 조회수 상위 3개 공통 패턴

1. 625,012회 — "주택을 지어놓고 살아보니 후회하는 일들이 생각보다 훨씬 많습니다." → 후회 예고 + 경험자 증언
2. 281,423회 — "불법 건축물에 단속이 되면 보통은 철거 대상입니다." → 최악 상황 선제 제시 + 탈출구 약속
3. 245,884회 — "여러분이 정말 궁금해하시는 주택 태양광 3kw 설치 원가를 싹 다 공개하려고 합니다." → 내부자 폭로 + 알권리 프레임

### 안 터진 영상 패턴

- 2,696회: "우리 집에 3kW 태양광을 설치하려고 하는데 왜 다들 전기료 절감 방식으로만 하고..." → 질문형이나 시청자 페인 없음, 정보 탐색자 톤
- 1,345회: 파일 없음 (분석 불가)
- 5,478회: "집을 지을 때 많은 분들이 고민하시는 부분 중 하나가 바로 2층 테라스를 설치해야 할까 하는 건데요." → 도입이 지나치게 일반적, 긴급성 없음
- 5,592회: "자연이 좋으면 경치 좋은 펜션 가서 며칠 푹 쉬다와라. 전원주택은 꿈도 꾸지 마라." → 후킹은 강렬하나 타겟이 '전원주택 꿈꾸는 사람'이라 모수가 작음

## 첫 3초 / 7초 / 10초 역할 구분

| 구간 | 역할 | 태연남 패턴 |
|------|------|------------|
| 첫 3초 | 이 영상을 봐야 하는 이유 | 페인 직격 or 충격 상황 제시 (후회, 폭탄, 단속, 사기) |
| 3~7초 | 신뢰 근거 + 갈등 심화 | 실존 인물/사례로 신빙성 추가, 손해 금액 명시 |
| 7~10초 | 해결 약속 or 본론 예고 | "오늘 제가 알려드리겠습니다" / "이 방법이 있다면?" |

## 반복 키워드 분석

### 고조회수 영상에서 반복되는 단어

- **후회**: 625,012 / 213,194 / 83,660 / 76,169회 영상에서 첫 5초 안에 등장
- **폭탄**: 29,785 / 11,968 / 전기료 관련 영상 다수에서 첫 7초 안에 등장
- **모르고**: "모르고 있었습니다" / "몰라서 손해" 패턴 반복
- **실제로**: 피해사례 영상의 신빙성 장치로 사용
- **업체**: 불신·의심 프레임 구축에 핵심 단어
- **절대 말 안 해주는 / 감추는**: 내부자 폭로 프레임

### 저조회수 영상에서 자주 나오는 단어

- "오늘은 ~ 대해 알아보겠습니다"
- "분들이 고민하시는"
- "이야기를 함께 나눠 보려고"
→ 정보 전달자 톤, 긴급성·감정 없음

## 해병듀오 적용 아이디어

> 해병듀오 = 구글맵 마케팅 대행사 / 타겟: 요식업·병원 사장

### 공식 1: 손해 예고형 적용
- 원본 패턴: "이거 모르고 하면 무조건 손해입니다"
- 적용: "구글맵 이거 안 세팅하고 광고 돌리면 그냥 버리는 돈입니다. 저 작년에 이 실수로 광고비 500만 원 날린 치킨집 사장님 봤습니다."
- 포인트: 구체적 업종 + 금액 + 직접 목격 경험 결합

### 공식 2: 내부자 폭로형 적용
- 원본 패턴: "업체들은 절대 말 안 해주는 영업비밀 폭로"
- 적용: "마케팅 대행사들이 절대 먼저 알려주지 않는 구글맵 세팅 3가지 — 이거 모르면 대행사만 좋은 일 시킵니다"
- 포인트: 대행사를 '적'으로, 사장을 '소비자 피해자'로 구도 설정

### 공식 3: 실화 투입형 적용
- 원본 패턴: "얼마 전 AS차 방문한 60대 부부"
- 적용: "지난달에 강남에서 일식집 하시는 사장님한테 연락이 왔습니다. 리뷰 200개인데 신규 손님이 안 온다는 거예요. 실제로 가서 보니까 문제는 전혀 다른 데 있었습니다."
- 포인트: 지역명 + 업종 + 구체적 수치 + 반전 예고

### 공식 4: 즉각 혜택 약속형 적용
- 원본 패턴: "이것만 누르면 24시간 내내 틀어도 전기료 폭탄 없음"
- 적용: "구글 비즈니스 프로필에서 이것 하나만 바꾸면 한 달 안에 신규 전화 2배 옵니다. 5분도 안 걸립니다."
- 포인트: 행동 1개 + 기간 + 결과 수치를 첫 3초에 압축

### 첫 3초 원칙 (해병듀오 버전)
- 반드시 포함: 페인(손해/후회/낭비) or 실화 인물(업종+지역)
- 금지: "오늘은 ~에 대해 알아보겠습니다" 형태의 중립 도입
- 권장 마무리(7~10초): "지금부터 그 방법을 알려드리겠습니다" or "이걸 보고나면 더 이상 대행사한테 속지 않을 겁니다"
