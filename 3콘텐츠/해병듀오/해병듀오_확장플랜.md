# 해병듀오 확장 플랜

> 작성일: 2026-04-19
> 기반: 『마케팅 설계자』(Russell Brunson) × 기존 해병듀오 마케팅 메모 × 2026-04-19 온라인 퍼널 전략

---

## 현재 파악

**해병듀오 정체:** 외국인 대상 구글맵(GBP) 마케팅 대행사
- 타겟: 요식업·병원·미용실·뷰티샵 사장
- 서비스: GBP 최적화 + 다국어(한/영/중/일) 세팅 + 리뷰·이미지 관리 + 24시간 소통방 + 월 레포팅
- 슬로건: "안되면 될 때까지 하는 마케팅" / "귀신 말고 외국인을 잡는 것"
- 목표: 월 계약 30개

**기존 마케팅 원칙 (해병듀오 메모에서):**
- 업주는 마케팅 피로도 높음 → 뻔한 메시지 금물
- 감성 > 이성 (구매는 감정이 결정)
- 동경 : 페인 = 1 : 3 비율 (페인 강조)
- 후킹 3-7초 — 결과부터 ("망해가던 매장이…")
- 레퍼런스 공식 따라하기
- 초반은 메타광고 + 브랜딩
- 권위 + 사회적 증거 확보 필수

## Phase 1 — 가치 사다리 설계

현재 월 구독만 팔고 있어 진입장벽 높음. 계단식으로 설계.

```
가치 사다리
━━━━━━━━━━━━━━━━━━━━━━━━━━
무료 미끼           : "외국인 매출 진단 체크리스트 PDF"
                     or "GBP 30초 무료 진단"
저가 (약 2만원)     : 전자책 "외국인 고객 100명 모으는 GBP 세팅법"
중가 (50만원)       : 원클릭 GBP 세팅 패키지 (1회성)
메인 (월 30~80만원) : 월 구독 운영 대행 ← 현재 주력
하이엔드 (500만원+) : 상권 독점 + 번외 상권 확장
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**왜 중요한가:**
처음부터 월 80만원은 못 팖. 무료 → 전자책 → 패키지 계단을 밟고 구독으로 넘어감. 전자책/패키지 팔리는 동안 **후기와 사회적 증거가 자동 누적**.

---

## Phase 2 — 퍼널 자동화 (매니챗)

기존 메모에 "매니챗 → 자동 프로그램" 이미 명시됨. 실제로 돌려야 함.

```
[릴스 페인 후킹]
   ↓
[댓글에 "신청" 치면]
   ↓
[매니챗 DM 자동 발송 → PDF + 상담 링크]
   ↓
[링크 클릭 → 캘린들리 30분 무료 진단 예약]
   ↓
[진단 → 패키지/구독 제안]
```

**채널별 역할** (2026-04-19 퍼널 전략 적용):

| 채널 | 역할 |
|---|---|
| 인스타 | 페인 릴스 대량 생산 → 매니챗 트리거 |
| 유튜브 미들폼 | 7-10분 Case Study → 권위·신뢰 구축 |
| 당근 | 지역 상권 밀착 직공략 |
| Threads | 사장님 커뮤니티 소통 |

**주의:** 유튜브 미들폼을 인스타 숏폼으로 잘라 올리지 말 것 (플랫폼 성격 다름)

---

## Phase 3 — 콘텐츠 공식 (후크-스토리-오퍼)

러셀 브런슨의 "후크-스토리-오퍼" + 해병듀오 기존 원칙 결합.

**매 콘텐츠 구조:**
```
후크 (3초)   : 결과부터 — "망해가던 매장이 이렇게 됐습니다"
스토리(페인) : 사장님이 공감할 고통 구체화
오퍼(가치)   : 우리가 해결한 방법
CTA         : "신청" 댓글 → 매니챗 자동
```

**주당 생산량:**

| 포맷 | 주 | 목적 |
|---|---|---|
| 페인 릴스 | 2-3개 | Before/After, 감정 자극 |
| 꿀팁 카드뉴스 | 1개 | 권위·신뢰 축적 |
| Case Study 미들폼 | 1개 | 진지한 후보군 전환 |

---

## Phase 4 — 후기 시스템

기존 메모에도 있지만 자동화 필요.

1. 계약 종료/성과 나온 시점 → 자동 후기 요청 DM
2. 비디오 후기(사장님 얼굴 나오는 것) = 할인 대가로 확보
3. 인스타 스토리 캡처 → 피드 재활용 → 매니챗 "후기 더 보기" 메뉴
4. 랜딩페이지/전자책/광고 소재에 전부 재사용

**후기 30개 확보 전까지는 광고 ROAS 제한적. 1순위 업무.**
