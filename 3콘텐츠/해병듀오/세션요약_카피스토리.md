## S109 — 어떤 스토리로 전술을 펼칠 것인가

### 핵심 원리
- 스토리는 '나의 경험'이 아니라 '우리의 정체성'에서 나온다 — 커뮤니티·형제애가 스토리의 뿌리
- '돈이 많고 싶다'(막연한 욕망)와 '돈을 벌고 싶다'(구체적 행동 의지)를 구분해라 — 후자만 움직인다
- 자기 한계를 스스로 설정하는 순간 비즈니스가 멈춘다 — "어차피 안 돼"가 가장 위험한 카피
- 소질보다 행동이 먼저다 — 잘하는 사람이 하는 게 아니라 하는 사람이 잘하게 된다

### 해병듀오 적용
- **계정 정체성 카피**: "외국인 손님 한 명도 잡지 못하던 식당이, 지금은 구글맵 1위입니다" — '우리(해병듀오+사장)가 함께 바꿨다'는 커뮤니티 서사
- **DM 후킹**: "관광객이 어떤 경로로 식당을 고르는지 아시나요?" — 막연한 광고 말고 구체적 행동 유도
- **극복 서사 구조**: [한계 상황] → [해병듀오 도입] → [수치 결과] — "어차피 외국인은 우리 식당 안 와"가 사장의 한계 설정이었음을 보여줘라

### 체크리스트
- [ ] 콘텐츠에 '우리(we)' 관점이 있는가 — 나 혼자가 아닌 사장과 함께하는 서사인가
- [ ] 행동 의지를 자극하는 구체적 다음 단계가 카피에 있는가
- [ ] 사장이 스스로 설정한 한계(오해/포기)를 콘텐츠가 깨주고 있는가

## S83 — 카피라이팅의 핵심

### 핵심 원리
- 카피 6대 원소: 명확한 메시지 / 주목 / 고객 중심 / 행동 유도 복선 / 신뢰 구축 / 감정적 연결선
- **고객 중심 전환**: "남성향수"→"남자친구 선물 향수" — 상품 설명이 아닌 고객의 상황·욕구로 프레임을 바꿔라
- **단순성의 힘**: "빨래 끝" 3글자로 세탁세제 시장 90% 장악 — 길게 설명할수록 카피가 죽는다
- 행동 유도 복선: 구매를 직접 요청하기 전에 이미 "다음에 또 오고 싶다"는 심리를 심어둬라
- 신뢰 구축은 리뷰·숫자·타인의 증언으로만 가능 — 업체 스스로의 주장은 신뢰가 안 된다

### 해병듀오 적용
- **GBP 카피 전환**: "GBP 최적화"→"외국인 관광객이 직접 찾아오는 식당으로 만들어드립니다" — 서비스 설명 대신 고객 결과로 프레이밍
- **행동 복선 카피**: "지금 당장 계약하세요" 대신 → "우선 우리 식당 구글맵 현황을 한번 확인해보세요" — 낮은 허들로 다음 행동 유도
- **신뢰 카피**: "저희가 잘합니다" 대신 → "이 식당, 3개월 전 구글맵 리뷰 0개였습니다 (지금은 147개)" — 수치+시간축이 증언
- **단순 후킹 실험**: "외국인 손님, 매달 늘고 있나요?" — 복잡한 설명 전 단 한 줄로 핵심 페인 찌르기

### 체크리스트
- [ ] 카피가 상품/서비스 설명인가, 고객 상황·욕구 설명인가 — 후자여야 한다
- [ ] 행동 유도가 직접적 구매 요청인가, 낮은 허들의 다음 단계인가 — 후자여야 한다
- [ ] 신뢰 근거가 업체 주장인가, 숫자+타인 증언인가 — 후자여야 한다
- [ ] 핵심 메시지를 10글자 이내로 압축할 수 있는가

## S82 — 카피라이팅의 기본기

### 핵심 원리
- 카피 9대 기본기: 숫자 / 직관 / 워딩>상품 / 간결 / 타겟 명확화 / 기능→결과로 전환 / 컨셉 / 규격 없이 / 대명사화
- **스토리 유무 = 대형 비즈니스 여부**: 스토리 없는 브랜드는 중소 규모를 벗어나지 못한다
- SNS 계정에 상품+기능만 나열하는 것은 치명적 오류 — 고객은 기능을 사는 게 아니라 변화를 산다
- "사람들은 좋아 보이면 사고, 사기 싫으면 합리적 이유를 만든다" — 감성이 먼저, 논리는 나중
- 대명사화: 브랜드가 특정 상황의 '고유명사'가 돼야 한다 ("엘리베이터 타면 오티스", "GBP 최적화하면 해병듀오")

### 해병듀오 적용
- **숫자 카피**: "관광객 방문 월 0명 → 47명" / "구글 리뷰 0개 → 200개 / 3개월" — 구체적 숫자가 카피다
- **기능→결과 전환**: "GBP 관리해드립니다" → "외국인이 찾아오는 식당으로 만들어드립니다"
- **대명사화 목표 카피**: "GBP 마케팅은 해병듀오" — 이 한 줄이 장기 브랜드 카피 방향
- **감성 우선 구조**: 결과 사진(만석 식당, 외국인 손님 줄) → 숫자 → 원리 설명 순서로 콘텐츠 배열
- **SNS 계정 점검**: 피드가 서비스 설명인가, 사장 변화 스토리인가 — 후자로 전환

### 체크리스트
- [ ] 각 콘텐츠에 구체적 숫자(기간+변화량)가 들어가 있는가
- [ ] 기능 설명이 결과(고객의 삶 변화)로 번역됐는가
- [ ] 피드 전체가 서비스 설명 나열이 아닌 스토리 흐름인가
- [ ] 감성(이미지·감동)이 논리(설명)보다 먼저 오는 구조인가

## S72 — 스토리는 어떻게 완성되는가

### 핵심 원리
- 스토리 완성의 핵심은 **진정성 있는 pain** — 꾸며낸 고난이 아닌 구체적·계산된 절박함
- Pain은 감정이 아닌 **숫자로 말할 때** 더 강하다 — "힘들었다"가 아닌 "5억 투자에 월 300만 순이익 계산이 나왔다"
- 개인 서사의 구조: [출발점의 결핍] → [계산된 선택] → [구체적 변화 수치] → [다음 세대 차단 욕구]
- 가난의 대물림 차단은 강력한 보편적 pain — 창업가의 서사에서 가장 공감 폭이 넓다
- 스토리에서 '자기 합리화'와 '진정성'의 차이: 실패도 숫자로 기록한 사람만이 신뢰받는다

### 해병듀오 적용
- **대표 서사 구조 제안**: [외국인 한 명도 못 잡던 식당] → [구글맵 최적화 도입] → [3개월 후 외국인 월 47명, 매출 +18%] → [이제 이 사장은 직원을 뽑을 수 있게 됐다]
- **숫자 pain 카피**: "리뷰 0개, 외국인 방문 0명, 매달 임대료 걱정" — 감정 서술 없이 숫자만으로 pain 전달
- **다음 세대 욕구 연결**: "이 식당 사장님은 아이에게 가난을 물려주고 싶지 않다고 했습니다" — 보편적 공감 트리거
- **실패 포함 서사**: 이전 SNS 마케팅 실패 경험을 가진 사장의 스토리 → 해병듀오 도입 → 첫 외국인 손님 → 이제는 월 정기 방문 — 실패가 있어야 신뢰가 생긴다

### 체크리스트
- [ ] 클라이언트 사례에 시작 시점의 구체적 수치(pain)가 있는가
- [ ] 스토리가 감정 묘사 없이 숫자로만 말해도 읽히는가
- [ ] '변화 전-후' 비교 수치가 콘텐츠에 명시됐는가
- [ ] 사장의 서사에 '왜 이 사업을 해야만 했는가'의 개인 동기가 있는가

## S163 — 아이템을 팔리게 하는 마술 STORY

### 핵심 원리
- "스토리 없이 돈 많이 번 비즈니스는 단 한 개도 없다" — 스토리는 선택이 아닌 필수 인프라
- 타임머신을 창고에 넣어두면 아무도 모른다 — 아무리 좋은 서비스도 알리지 않으면 없는 것
- 질/가격 홍보는 효과 없다 — "우리 맛집입니다", "가격 착합니다"는 판매자 오류
- 애플·인스타그램 = IT 기업이 아닌 라이프스타일 스토리 — 사람들이 사는 건 제품이 아니라 그 브랜드가 만드는 세계관
- "사람은 좋아 보이면 사고, 사기 싫으면 합리적 이유를 만든다" — 감성 설득이 먼저, 논리는 사후 정당화

### 해병듀오 적용
- **광고/프로모션 정당화 카피**: "홍보가 부끄러운 게 아닙니다. 좋은 식당인데 외국인이 모르는 게 진짜 문제입니다" — 영업/마케팅에 죄책감 있는 사장 설득 카피
- **세계관 카피 방향**: 해병듀오 = "외국인이 찾아오는 식당을 만드는 사람들" — GBP 기술 회사가 아닌 사장의 꿈을 실현하는 파트너 정체성
- **질/가격 대신 스토리**: "맛있는 식당 많습니다. 그런데 외국인이 구글맵으로 검색했을 때 나오는 식당은 따로 있습니다" — 품질 경쟁에서 가시성 경쟁으로 프레임 전환
- **라이프스타일 스토리 카피**: "이 식당 사장님, 이제 외국어를 배우기 시작했습니다. 손님이 바뀌면 사람이 바뀝니다" — 서비스 결과가 아닌 삶의 변화로 연결

### 체크리스트
- [ ] 콘텐츠가 서비스 설명인가, 라이프스타일·세계관 제시인가 — 후자여야 한다
- [ ] "우리 잘합니다" 류의 판매자 관점 카피가 없는가
- [ ] 광고/프로모션을 부끄러워하는 잠재 클라이언트의 저항을 카피가 해결하고 있는가
- [ ] 브랜드 정체성 카피(해병듀오 = ?)가 명확하게 정의됐는가
