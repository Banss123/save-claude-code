> 작성일: 2026-04-19
> 대상: 해병듀오 (외국인 관광객 대상 Google Business Profile 최적화 대행사)

## S1 — Session 73: 나는 어떤 마케팅을 하고 있는가?

**핵심 원리:**
- 마케팅에는 6단계 위계가 있다. 전단지(1단계) → 방문영업(2단계) → 전화영업(3단계) → SNS(4단계) → 콘텐츠(5단계) → 팬덤/문화(6단계)
- 단계가 높을수록 스토리·권위·관련성의 가치가 높아지고, 계약 전환율이 올라간다
- 높은 단계만이 좋은 것이 아니다. 사업 초기에는 낮은 단계(방문영업, 전화)도 반드시 병행해야 한다
- 본인이 어느 단계의 마케팅을 쓰고 있는지 먼저 파악해야 다음 단계로 올라갈 수 있다
- 팬슈머 마케팅(인플루언서 연계) 단계에서는 광고에 대한 고객의 거부감이 크게 희석된다

**해병듀오 적용:**
- 현재 단계 진단: 방문영업·전화영업(2~3단계) 수준 — 다음 단계인 SNS·콘텐츠(4~5단계)로 올라가는 로드맵 수립
- 인스타 계정에서 "외국인이 많이 오는 가게 사장 인터뷰" 포맷으로 팬덤 마케팅 씨앗 심기 — 사장이 콘텐츠 주인공이 되면 광고 거부감 제거
- 방문영업을 포기하지 말고, 인스타 콘텐츠와 병행해서 신뢰 누적 후 계약 전환율 높이기

**체크리스트:**
- [ ] 현재 주력 영업 채널을 6단계 중 어디에 위치시킬지 명시
- [ ] 다음 단계로 올라가기 위한 구체적 채널 1개 선정 (예: 인스타 릴스)
- [ ] 인플루언서 또는 실제 클라이언트를 활용한 팬슈머 마케팅 실험 1회 기획

## S2 — Session 62: 사람들은 언제 돈을 쓰는가?

**핵심 원리:**
- 미팅이 깨지는 이유는 딱 하나: 상대방의 구매 저항이 존재하기 때문이다
- 사람들이 지갑을 여는 근본 이유는 투자 — 지불하면 삶이나 사업이 편하고 윤택해질 것이라는 전제가 있어야 한다
- 구매 결정은 감성적 요소에서 일어난다. 이성(기능 설명)보다 감성(이득 욕구)을 먼저 건드려야 한다
- 아이템의 본질을 관통하는 질문 하나를 준비하라. 즉각 호기심이 없으면 시간을 더 쓰지 말고 워크어웨이
- 매출 상승이라는 이득뿐 아니라 "사장이 본업에 집중할 수 있는 시간 절약"도 강력한 감성 관통 포인트다

**해병듀오 적용:**
- 영업 첫 문장을 기능 설명이 아닌 결과 질문으로 시작: "지금 구글 지도에 외국인 후기 몇 개나 있으세요? 그게 0개면 외국인 손님은 사실상 못 받고 계신 거예요"
- 감성 관통 질문 완성: "외국인 관광객이 내 가게를 지나쳤을 때 선택받고 싶지 않으세요?" — 이득 욕구 자극
- 즉각 반응 없는 클라이언트는 그 자리에서 시간 소비하지 말고 다음 타겟으로 이동 (워크어웨이 원칙 적용)

**체크리스트:**
- [ ] 업종별 감성 관통 질문 3가지 작성 (요식업 / 병원 / 미용실)
- [ ] 미팅 시 "이득 어필" 스크립트를 기능 설명 없이 30초 버전으로 작성
- [ ] 즉각 반응 없는 경우 자연스럽게 자리 뜨는 퇴장 멘트 준비

## S3 — Session 33: 마케팅, 꽌시 전략 (중급)

**핵심 원리:**
- 현대 사업이 어려운 이유는 꽌시(관계)의 부재다 — 단순 홍보가 아닌 관계망이 클라이언트 DB를 만든다
- 꽌시 개발의 4요소: 관계 → 관련성 → 반응 → 보상 (4R 구조)
- 1차 생산자가 아닌 1차 유통자가 되어라 — 내가 직접 못해도 남의 능력으로 연결만 해도 돈이 된다
- 새로운 마케팅 채널 발굴이 사업의 핵심 난제다 (맥킨지 원칙) — 새 채널은 마케팅 비용을 10~15% 절감한다
- 소통(리뷰·커뮤니케이션)과 편의성이 현대 마케팅의 핵심 축이다. 리뷰가 없으면 광고·상세페이지도 소용없다

**해병듀오 적용:**
- 꽌시 구조로 신규 채널 발굴: 관광 가이드 / 숙박업소 직원 / 여행 블로거와 협업 구조 설계 — 이들을 1차 유통자로 활용
- 기존 클라이언트에게 월 레포팅 제공 → 레포팅 자체가 관계 유지 도구이자 추가 계약 물꼬 역할
- GBP 리뷰 관리를 "소통 채널"로 포지셔닝: 외국인 리뷰에 영어·일어로 답글 달아주는 것이 곧 사장 대신 하는 편의성 서비스

**체크리스트:**
- [ ] 꽌시 4R 기준으로 현재 클라이언트 관계 점검 (관계 / 관련성 / 반응 / 보상 각 항목 채우기)
- [ ] 새로운 마케팅 채널 후보 3곳 리스트업 (예: 관광 안내소, 여행사, 에어비앤비 호스트)
- [ ] 기존 클라이언트 1명에게 월 레포팅 이외 "추가 소통 접점" 1개 만들기

## S4 — 2023/05/07: 마케팅, 어떻게 팔 것인가?

**핵심 원리:**
- 물건을 판다는 생각 버려라. 상대방 사업이 더 잘 되게 컨설팅한다는 마인드로 접근해야 계약이 성사된다
- 상대가 생각하지도 못한 리스크와 기회를 찾아줘라 — 그게 영업의 시작점이다
- 컨설팅과 마케팅은 크게 다르지 않다. 여러분의 서비스 = 사장의 사업을 개선시켜주는 컨설팅
- 남의 능력으로 사업하라 — 내가 모르는 영역은 아는 사람과 연결하고 수익을 나눠라
- 본질을 관통하는 질문을 준비하라 — 상대의 감성적 니즈를 건드리는 단 하나의 질문

**해병듀오 적용:**
- 영업 프레임 전환: "GBP 관리 서비스를 팝니다" → "사장님 가게에 외국인이 안 오는 이유를 진단해드리고 해결해드립니다" (컨설팅 프레임)
- 미팅 전 해당 가게의 GBP 현황 미리 분석 → "지금 이 가게의 리뷰 상태에서 외국인 10명 중 몇 명이 선택할까요?"라는 문제 제기로 시작
- 리뷰 번역·답글 등 내가 못 하는 언어 영역은 외주 네트워크 활용 — 1차 유통자 포지션 유지

**체크리스트:**
- [ ] "컨설팅 프레임" 영업 스크립트 작성 (2분 버전)
- [ ] 미팅 전 타겟 가게 GBP 현황 체크리스트 완성 (리뷰 수 / 언어 구성 / 사진 수 / 카테고리 설정)
- [ ] 미팅 자리에서 즉석 GBP 진단 보여주는 루틴 설계

## S5 — 세션 18: 문화를 만들면 돈을 정복한다

**핵심 원리:**
- 일반인은 유행을 따르고, 사업가는 유행을 예측하고, 시인은 유행을 만든다 — 문화를 만드는 자가 시장을 지배한다
- 기업이 마케팅에 천문학적 돈을 쓰는 이유는 아이템 홍보가 아니라 소비 문화를 형성하기 위해서다
- 빼빼로데이, 삼겹살데이 등은 모두 대기업이 만든 인위적 문화 — 문화가 만들어지면 소비는 자동이다
- 유행이 끝나면 사업이 망하는 이유: 처음부터 유행 추종형 사업을 시작했기 때문이다
- 확신이 있어야 행동력이 나오고, 행동력이 있어야 문화를 만드는 데 집중할 수 있다

**해병듀오 적용:**
- 해병듀오가 만들 문화: "외국인한테 선택받는 가게 = 특별한 가게"라는 인식 심기 — 사장들이 먼저 찾게 만드는 문화
- 인스타에서 "외국인 후기 있는 가게 vs 없는 가게 매출 차이" 콘텐츠 시리즈 → 사회적 증거 + 문화 형성 동시 공략
- 빼빼로데이처럼 "외국인 시즌 마케팅 캘린더" 제안 — 벚꽃 시즌 / 여름 관광 피크 / 연말 방한 시즌에 맞춘 GBP 최적화 캠페인

**체크리스트:**
- [ ] 해병듀오가 만들고자 하는 문화를 한 문장으로 정의
- [ ] 인스타 콘텐츠 시리즈 기획 (비교 포맷 3편 이상)
- [ ] 외국인 관광 시즌 캘린더와 연계한 월별 클라이언트 제안 패키지 설계

## S6 — Session 3: 공포를 이용하는 방법

**핵심 원리:**
- 걱정(니즈)과 공포는 다르다. 걱정은 이미 인식된 니즈지만, 공포는 내가 만들어서 심어주는 니즈다
- 공포가 더 돈이 된다 — 대처 방안이 없기 때문에 즉각 해결책을 사려 한다
- 큰 리스크를 짊어지고 있는 사람(사업 리스크가 큰 자영업자)에게 공포를 만들면 더 강하게 작동한다
- 컨설팅 마인드: "내 물건을 어떻게 팔까"가 아니라 "이 사람이 생각하지 못한 리스크와 기회를 찾아주겠다"
- 동경:페인 = 1:3 원칙 — 동경(잘 됐을 때)보다 페인(안 됐을 때의 손실)을 3배 더 자극해야 구매로 이어진다

**해병듀오 적용:**
- 공포 포인트 설계: "지금 구글 지도에 한국어 리뷰만 있는 가게는 외국인 관광객에게 존재하지 않는 가게입니다" — 사장의 손실 공포 자극
- 동경:페인 = 1:3 콘텐츠 구성: 동경 1개(외국인 손님 가득한 가게 인터뷰) + 페인 3개(외국인 관광객이 지나치는 가게들 공통점 분석)
- 미팅 첫 3초 후킹: 사장 가게의 GBP 리뷰 현황 보여주며 "경쟁 가게는 지금 이렇습니다" — 비교 공포 즉시 생성

**체크리스트:**
- [ ] 업종별 공포 포인트 3개 작성 (요식업 / 병원 / 미용실)
- [ ] 인스타·블로그에서 "이 가게는 왜 외국인 손님이 없을까" 시리즈 기획 (페인 3배 원칙 적용)
- [ ] 미팅 첫 3초 후킹 스크립트 완성 — GBP 현황 캡처를 들고 들어가는 루틴 고정

## 통합 인사이트 (해병듀오 핵심 적용 방향)

1. **영업 단계 전략**: 방문영업(지금) + 콘텐츠/SNS(다음 단계)를 병행 — 어느 단계인지 인식하고 올라가는 로드맵 필요
2. **감성 우선 세일즈**: 기능 설명 금지, 첫 문장은 반드시 "사장님이 놓치고 있는 것" 공포 또는 이득 욕구 자극
3. **컨설팅 프레임**: "GBP 대행사"가 아니라 "외국인 손님 유입 컨설턴트"로 포지셔닝
4. **문화 형성**: 인스타 콘텐츠로 "외국인한테 선택받는 가게" 문화 만들기 — 사장이 먼저 찾아오게 만드는 구조
5. **꽌시 채널 발굴**: 관광 생태계(가이드, 숙박, 여행사)를 새로운 유통 채널로 확보
