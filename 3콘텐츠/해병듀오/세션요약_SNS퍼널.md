## S77 — SNS 마케팅에서 놓치는 것들

**핵심 원리:**
- SNS를 버리고 가는 건 홍보를 하지 않겠다는 것과 같다. 현대인은 TV가 아닌 핸드폰과 SNS를 본다
- "마케팅이 의미 없다"는 착각은 마케팅이 아닌 걸 마케팅이라 착각하는 것에서 비롯된다
- 브랜딩은 마케팅을 해야 완성된다. 브랜드의 힘은 가성비를 무력화한다 (실라호텔 망고빙수 10만원, 디올 원가 8만원 → 판매가 500만원)
- 마케팅에서 놓치는 핵심은 "유비패스의 관점" — 노출이 아닌 전환 흐름 설계
- 싼 것이 능사가 아니다. 브랜딩이 된 제품은 원가와 무관하게 가격이 결정된다

**해병듀오 적용:**
- GBP 대행 영업 시 "마케팅 의미 없다"는 사장님 반응 대비: "마케팅이 아닌 걸 마케팅이라 착각한 것입니다. 진짜 마케팅은 외국인이 검색했을 때 내 가게가 나오도록 흐름을 설계하는 것입니다"로 반박
- 해병듀오 자체 브랜딩 투자 필요: SNS(인스타) 꾸준히 운영해야 서비스 가격 방어 가능
- GBP 최적화를 단순 등록이 아닌 "외국인 트래픽 → 방문 전환 설계"로 포지셔닝

**체크리스트:**
- [ ] 영업 현장에서 "마케팅 의심" 반박 스크립트 1개 작성
- [ ] 해병듀오 인스타 계정 운영 주기 설정 (주 2회 이상)
- [ ] GBP 서비스 소개 자료를 "노출 수" 지표가 아닌 "전환 설계" 중심으로 리프레임

## S76 — SNS 마케팅 어떤 플랫폼을 이용할 것인가

**핵심 원리:**
- 정보의 단차가 낮아진 세상에서 단순 정보 차익 사업은 지속 불가능해졌다
- 단차에 의존한 사업(중고시장 등)은 인터넷 발달로 점점 붕괴 중
- 답이 정해진 플랫폼은 없다. 이커머스, 오프라인 영업, SNS 모두 도구일 뿐
- 결과로 입증해야 한다. 1년이 넘는 시간 투자 후 배수로 수익이 늘어나는 구조
- 커뮤니티, 바이럴 채널, 오프라인 영업 — 각자에 맞는 조합을 찾아야 한다

**해병듀오 적용:**
- 해병듀오의 경쟁 우위: GBP 다국어 최적화는 정보 단차가 여전히 높은 영역 (사장님들이 직접 하기 어려운 전문성)
- 단일 플랫폼에 의존하지 말고 인스타(브랜딩) + 직접 영업(오프라인) + GBP 레퍼런스(증거) 3축 병행
- 1년 안에 30개 계약 달성 목표 — 지금은 씨앗 시기, 6개월 후 배수 성장 예상

**체크리스트:**
- [ ] 해병듀오 SNS 플랫폼 우선순위 결정 (인스타 vs 유튜브 쇼츠 vs 네이버 블로그)
- [ ] 월 계약 건수 추적 대시보드 만들기 (언제부터 배수 성장 시작되는지 모니터링)
- [ ] 경쟁사 GBP 대행사 정보 수집 (정보 단차 현황 파악)

## S111 — 트래픽과 구매전환율

**핵심 원리:**
- 트래픽은 마케팅의 실험 샘플일 뿐이다. 트래픽 = 매출이 아님
- 마케팅의 핵심은 "트래픽을 구매 전환으로 이어지게 하는 구조 설계"다
- 트래픽과 구매 전환 사이에는 심리적 허들이 존재한다 (감정, 비용, 시간이 투자되어야 구매 발생)
- 행동 경제학을 알아야 이 허들을 깨는 요건을 이해할 수 있다
- 마케팅을 단순 홍보나 노출로 착각하는 사업가가 대부분이다

```
트래픽 흐름도:
노출 → [심리적 허들 존재] → 유입 동기 파악 → CTA 설계 → 구매 전환

허들 요인: 감정 투자 / 비용 부담 / 시간 소비
허들을 낮추는 방법: 사회적 증거 / 레퍼런스 / 리뷰 / 무료 체험
```

**해병듀오 적용:**
- GBP 대행 영업 시 트래픽(조회수) 보여주는 것보다 "전환 설계"를 보여줘야 계약 성사
- 외국인이 GBP 보고 전화/방문으로 이어지는 심리적 허들: 언어 장벽 + 신뢰 부족 → 다국어 리뷰 + 사진 최적화로 허들 제거
- 영업 시 "조회수 몇 명"이 아닌 "방문 문의 몇 건 발생했다"는 CTA 데이터로 설득

**체크리스트:**
- [ ] GBP 성과 지표를 "노출 수"에서 "전화/길찾기/예약 클릭"으로 전환
- [ ] 계약 사장님 가게별 심리적 허들 3가지 정의 후 제거 전략 수립
- [ ] 신규 영업 제안서에 "트래픽 → 전환 구조도" 1페이지 추가

## S110 — 소비는 어떻게 만들어지는가

**핵심 원리:**
- 시장에 합리적인 소비자는 없다. 인간은 충동적이며 자신이 보는 대로, 남들이 하는 대로 따라 결정한다 (대니얼 카너먼)
- 수요는 어떻게 발생하나: 심리적 만족감의 효율로 결정된다. 기능이 아니라 감정
- 합리적 소비를 이끌어낼 것인가가 아니라, 비합리적 소비를 어떻게 이끌어낼 것인가를 고민해야 한다
- 소비자 스스로는 합리적이라 믿지만 실상은 브랜드, 유행, 타인의 시선에 따라 결정한다
- 내 상품이 왜 팔리는지 이해하는 사람이 사업을 지속할 수 있다

```
소비 발생 공식:
합리성(기능/가격) → 실제로 작동 안 함
감정 + 사회적 증거 + 브랜드 이미지 → 실제 구매 발생
```

**해병듀오 적용:**
- GBP 대행 계약 영업 시 사장님에게 "외국인이 맛집 고를 때 합리적으로 비교하지 않는다. 사진 예쁘고, 리뷰 많고, 검색에 나오면 바로 간다"로 설명
- 감성 우선 영업 전략: 이성(ROI 계산) 먼저 꺼내지 말고, 동경(다른 가게 성공 사례) + 페인(외국인이 내 가게를 못 찾는 손해) 순서로 접근
- 리뷰 콘텐츠를 기능 설명이 아닌 감정 표현으로 채우기 (음식 맛이 아닌 경험, 분위기, 기억)

**체크리스트:**
- [ ] 영업 스크립트를 "이성 설득" → "감성 자극" 순으로 재구성
- [ ] 동경:페인=1:3 비율로 영업 멘트 점검
- [ ] 클라이언트 가게의 "비합리적 구매 요소" 3가지 파악 후 GBP에 반영

## S58 — 커뮤니티 퍼널

**핵심 원리:**
- 커뮤니티는 무에서 유로 갈 수 있는 가장 대표적인 사업 구조 중 하나다
- 정보의 단차가 낮아진 현대에서 "정보 판매" 방식의 커뮤니티는 한계가 있다
- 현대 커뮤니티는 누구든 쉽게 만들 수 있다 → 희소성 하락, 신뢰 기반 커뮤니티가 진짜 자산
- 커뮤니티 퍼널 구조: 커뮤니티 형성 → 신뢰 축적 → 상품/서비스 자연스럽게 판매
- 먼저 커뮤니티를 형성하고, 그 안에서 사업 기회를 만들어야 한다

```
커뮤니티 퍼널 흐름:
콘텐츠로 팔로워 모으기
     ↓
관심사 기반 소규모 커뮤니티 형성
     ↓
신뢰 축적 (무료 가치 제공)
     ↓
자연스러운 서비스 권유 → 계약
```

**해병듀오 적용:**
- 외국인 관광객을 타겟으로 한 B2C 커뮤니티보다 B2B(사장님) 커뮤니티가 더 현실적: 요식업 사장님 오픈카톡방 또는 뷰티샵 사장님 모임 침투
- 해병듀오 인스타를 단순 포트폴리오가 아닌 "외국인 마케팅 사장님 커뮤니티" 진입 채널로 활용
- 월 30개 계약 목표의 핵심 경로: 기존 계약 사장님 → 소개 네트워크 → 커뮤니티 퍼널

**체크리스트:**
- [ ] 요식업/뷰티 사장님 네트워크 1개 (오픈카톡방 또는 협회) 참여 또는 생성
- [ ] 기존 계약 사장님에게 소개 인센티브 설계 (1건 소개 시 1개월 무료 또는 할인)
- [ ] 인스타 콘텐츠 방향을 "서비스 광고"에서 "사장님에게 유용한 외국인 마케팅 정보"로 전환

## 통합 인사이트 — 해병듀오 실전 적용

### 핵심 메시지 정렬

```
[S110 소비 원리] 사장님은 감정으로 계약 결정
       ↓
[S77 브랜딩 원리] 해병듀오 자체 브랜딩이 가격 방어
       ↓
[S111 전환 구조] 영업 멘트가 심리적 허들을 제거해야 계약
       ↓
[S58 커뮤니티] 네트워크에서 소개가 주요 성장 엔진
       ↓
[S76 플랫폼] 인스타+오프라인+GBP 레퍼런스 3축 병행
```

### 즉시 실행 3가지

1. 영업 스크립트 재구성: 이성(ROI) → 감성(동경+페인) 순서, 트래픽이 아닌 전환 데이터로 설득
2. GBP 성과 지표 변경: 노출 수 → 전화/방문/예약 클릭 수 (심리적 허들 제거 증거)
3. 사장님 커뮤니티 1개 진입: 소개 네트워크가 월 30개 계약의 핵심 경로
