## ✅ 해병듀오 실행 체크리스트

### 1. 제목 공식 (비가역성 + 페인)

- [ ] "외국인 놓치면 평생 후회하는 매장 특징 10가지" 같은 **비가역성 단어** 탑재
- [ ] 숫자 붙이기: "3가지 / 5가지 / 10가지" (리스트형이 클릭 잘 됨)
- [ ] "사기" / "폭로" / "진짜 이유" / "절대 하지 마세요" 키워드 포함 테스트
- [ ] 기능 설명형 제목 금지 ("GBP 관리 방법" ❌)
- [ ] 결과·손실·공포 중심 ("외국인 손님 뺏기는 진짜 이유" ✅)

### 2. 첫 3초·7초 후킹 공식

- [ ] **실존 인물 + 구체 금액 + 갈등 구도**를 첫 7초 안에
- [ ] 템플릿: "{업종} {지역} 사장님이 {수치}를 놓쳤습니다"
  - 예: "강남 일식집 사장님, 리뷰 200개인데 외국인 손님 0명"
  - 예: "홍대 고깃집, 매출 5천만 놓치고 폐업 직전"
- [ ] "오늘은 ~에 대해 알아보겠습니다" 식 중립 도입 **금지**
- [ ] 첫 문장에 **즉각적 손해 감정** (후회 / 폭탄 / 사기 / 철거)
- [ ] "더 화나는 게 뭔지 아세요?" 같은 감정 고조 전환 멘트 활용

### 3. 오프닝 3문장 공식

- [ ] **문장 1** — 공감 ("사장님, 이런 고민 한 적 있죠?")
- [ ] **문장 2** — 고통 확대 ("근데 이게 끝이 아닙니다")
- [ ] **문장 3** — 수치 포함 이득 약속 ("해병듀오가 도와드린 매장은 3개월 만에 매출 3억 달성")
- [ ] 순서 절대 바꾸지 말 것 (이탈률 상승)

### 4. 본문 구조 (5단 고정 비율)

- [ ] **후킹 10%** — 첫 7초 페인·수치
- [ ] **문제 제기 15%** — "왜 이게 문제냐면…"
- [ ] **번호 리스트 본론 55%** — "사기꾼 업자 특징 3가지", "GBP 꼭 해야 할 5가지"
- [ ] **결론 정리 10%** — 핵심 메시지 반복
- [ ] **CTA 10%** — 하나만, 짧게

### 5. 내부 고발자 포지셔닝

- [ ] "대행사들이 절대 말 안 해주는 ~~"
- [ ] "영업사원한테 속지 마세요"
- [ ] "업계 실명 폭로" 류 멘트
- [ ] "저희는 직접 해본 사람이기 때문에" 전문성 주장
- [ ] 경쟁사가 고객을 속이는 구조를 폭로하는 콘텐츠 시리즈 기획

### 6. 사례 활용 원칙

- [ ] "많은 분들이" / "여러 매장이" **금지** (뇌가 위험으로 인식 X)
- [ ] 반드시 **지역 + 업종 + 수치**:
  - "강남 S한식집: 월 매출 900만 → 1,700만"
  - "명동 J뷰티샵: 외국인 예약 0 → 월 6건"
- [ ] Before/After 수치 필수
- [ ] 실명에 준하는 구체성 (머리글자 + 지역)

### 7. CTA 규칙

- [ ] **CTA는 하나만** (터진 영상은 예외 없이 1개)
- [ ] 구독·좋아요·댓글·링크 다 붙이지 말 것
- [ ] CTA **직전에 감정/공감 문장** 삽입
  - "사장님 매출에 도움이 되었으면 좋겠습니다. 궁금하시면 프로필 링크로 문의 실시!"
- [ ] 길이: 마지막 15초 이내
- [ ] 구체 행동 1개만 ("댓글에 '진단' 남기세요")

### 8. 시그니처 개발

- [ ] 해병듀오 고정 클로징 만들기 (태연남tv의 "태양광을 연구한 남자 태연남이었습니다" 급)
- [ ] **후보 시안:**
  - "외국인 잡아주는 해병듀오였습니다. 아-잇!"
  - "안되면 될 때까지 하는 해병듀오. 이상 보고 끝!"
- [ ] 모든 영상 엔딩에 예외 없이 사용
- [ ] 인트로 시그널도 동일 원칙 ("아-잇!" 이미 있음)

### 9. 감정 강도 설계

- [ ] **공포 / 분노 / 불안** 중 하나는 반드시 포함
- [ ] 태연남tv는 "사기 당한 부부", "철거 위기" 같은 강한 감정 사용
- [ ] 해병듀오는 "옆집에 손님 뺏기는 중", "외국인 매출 월 1천 날림" 식으로 불안·손실 공포 사용
- [ ] 뜨거운 감정 없는 중립 콘텐츠는 터지지 않음

### 10. 제작 원칙 (대량·빠른 테스트)

- [ ] 한 달에 15개 찍어내는 공장 마인드 (이미 해병듀오 원칙)
- [ ] 영상 1개당 완벽 추구 ❌ → 구조·공식 충실
- [ ] 터진 영상 분석 → 구조만 복제해서 해병듀오 맥락으로 재촬영
- [ ] 안 터진 영상도 보존 (구조 개선 학습용)

## 📊 벤치마크 지표 (태연남tv 기준)

### 터진 영상 (10만+) 공통 수치
- 영상 길이: **6~8분** (360~480초)
- 첫 감정 단어 등장: **3초 이내**
- 구체 수치 등장: **7초 이내**
- CTA 개수: **1개**
- 번호 리스트 사용: **거의 100%**

### 안 터진 영상 (1만 이하) 공통 실수
- 중립 도입 ("안녕하세요, 오늘은…")
- 추상적 표현 ("많은 분들이")
- CTA 여러 개 (구독+좋아요+댓글+링크)
- 업체 자랑 중심
- 숫자 없는 주장
