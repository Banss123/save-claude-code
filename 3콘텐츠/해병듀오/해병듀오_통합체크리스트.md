## 1. 영업 스크립트·미팅 (페인 중심)

- [ ] 영업 스크립트를 "판매자 모드" → "진단·파트너 모드"로 리라이팅
- [ ] 감정 흐름 5단계(호기심→이해→욕망→저항→해소)로 미팅 순서 스크립트화
- [ ] 동경:페인 = 1:3 비율로 모든 멘트 점검
- [ ] 업종별(요식업/병원/미용실) 감성 관통 질문 3가지씩 작성
- [ ] 미팅 첫 3초 후킹 스크립트 완성 (타겟 가게 GBP 현황 캡처 지참)
- [ ] 미팅용 "이득 어필 30초 버전" 스크립트 (기능 설명 없이)
- [ ] "컨설팅 프레임" 2분 버전 스크립트
- [ ] 미팅 자리 즉석 GBP 진단 루틴 설계
- [ ] 즉각 반응 없을 때 자연스러운 퇴장 멘트 준비
- [ ] 마케팅 의심·반박 대응 스크립트
- [ ] 최저가 경쟁사 대응 스크립트 ("우리는 외국인 특화라 다릅니다")
- [ ] 업종별 공포 포인트 3개 (요식업/병원/미용실)

## 2. 카피·후킹 공식

- [ ] 카피를 서비스 설명 → 고객 결과 중심으로 재작성
- [ ] 핵심 메시지 10글자 이내 압축
- [ ] 기능 설명 → 결과(삶의 변화)로 번역
- [ ] 구체 숫자(기간+변화량) 삽입
- [ ] Before/After 비교 수치 명시
- [ ] "우리 잘합니다" 류 판매자 관점 카피 전부 제거
- [ ] 후킹 문구를 공포·손실 유발형으로 전환
- [ ] 인스타 프로필 한 줄 소개 3안 A/B 테스트
- [ ] DM 영업 카피 A/B 테스트 (기능형 vs 결과형) 2주 비교
- [ ] "인지도 올려드려요" → "외국인 손님 오게 만들어드려요" 언어 전환
- [ ] 카피에 '우리(사장과 함께)' 관점 확인
- [ ] 낮은 허들의 다음 단계 CTA (직접 구매 요청 X)
- [ ] 신뢰 근거는 업체 주장이 아닌 숫자+타인 증언

## 3. 브랜딩·포지셔닝

- [ ] 해병듀오 브랜드 정체성 한 문장 정의
- [ ] STP 한 장짜리 정의 문서 작성
- [ ] "외국인 관광객 특화"를 모든 소개 자료 첫 줄에 배치
- [ ] 업종별 포지셔닝 메시지 3종 세트 초안
- [ ] "경험의 일관성"이 로고·색상보다 우선인지 점검
- [ ] "우리 서비스를 써야 할 사장님" 한 줄 기준 정의
- [ ] 브랜드 터치포인트 지도 (첫 DM → 계약 → 레포팅 → 리뷰)
- [ ] 해병듀오가 만들고자 하는 문화 한 문장 정의
- [ ] 대표 철학·스토리를 인스타 피드 1개로 제작
- [ ] 해병듀오 식별 한 줄 + 구별 포인트 3가지 원페이지

## 4. 가치 사다리·상품 설계

- [ ] 무료 미끼: GBP 진단 리포트 템플릿 제작
- [ ] 저가 미끼: 원타임 무료 세팅 or 전자책 설계
- [ ] 패키지 가격표 앵커링 재설계 (고가 → 중가 → 기본 순 제시)
- [ ] 월 관리 패키지 수익 구조 명시 (캐시카우 명확화)
- [ ] 외국인 관광 시즌 캘린더 연계 월별 패키지 제안

## 5. 퍼널·자동화

- [ ] 리드 유입 → 계약 퍼널 단계별 전환율 측정 시작
- [ ] 월 계약 건수 추적 대시보드 구축
- [ ] 문의 → 계약 전환율 월별 추적 시트
- [ ] 광고 크리에이티브 교체 주기 + A/B 테스트 기준 수립
- [ ] 무료 진단 → 유료 계약 넛지 구조 영업 단계별 문서화
- [ ] AI 활용 GBP 다국어 최적화 워크플로우 표준화 (작업 시간 측정)

## 6. 측정·KPI

- [ ] GBP 성과 지표: "노출 수" → "전화/길찾기/예약 클릭" 전환
- [ ] 월 레포팅에 "인식 강화 지표" 추가 (외국인 리뷰 수, 외국어 검색 노출)
- [ ] 월 레포팅을 "숫자만" → "체감 만족 프레임"으로 리디자인
- [ ] 인식 KPI 3단계(인지 → 인식 변화 → 행동 변화) 측정 루틴

## 7. 후기·사회적 증거

- [ ] Before/After 수치 명확한 레퍼런스 케이스 1개 제작 (실제 계약 고객)
- [ ] 계약 후 리뷰/방문자 증가 수치 사례 2건 이상 확보
- [ ] 레퍼런스 고객 사례를 영업 자료에 포함
- [ ] 영업 시 "주변 ○○도 하고 있어요" 레퍼런스 리스트 준비
- [ ] 부정적 리뷰·비교 질문 대응 FAQ 5개 준비

## 8. 네트워크·레퍼럴

- [ ] 요식업/뷰티 사장님 커뮤니티 진입 (오픈카톡방/협회)
- [ ] 기존 계약 사장님 소개 인센티브 설계 (1건 소개 → 1개월 무료 or 할인)
- [ ] 레퍼럴 구조 문서화 (조건·혜택·실행 방법)
- [ ] 기존 계약 고객에게 소개 프로그램 제안 문자 초안
- [ ] 꽌시 4R(관계/관련성/반응/보상) 기준 현재 클라이언트 관계 점검
- [ ] 관광 가이드·숙박·여행사 꽌시 채널 확보 리스트
- [ ] 신규 마케팅 채널 후보 3곳 리스트 (관광 안내소, 여행사, 에어비앤비 호스트)

## 9. 콘텐츠 제작

- [ ] SNS 플랫폼 우선순위 결정 (인스타 vs 유튜브 쇼츠 vs 네이버)
- [ ] 인스타 계정 운영 주기 설정 (주 2회 이상)
- [ ] 인스타 콘텐츠 방향: "서비스 광고" → "유용한 외국인 마케팅 정보"
- [ ] "이 가게는 왜 외국인 손님이 없을까" 시리즈 기획 (페인 3배 원칙)
- [ ] 비교 포맷 콘텐츠 시리즈 3편 이상 기획
- [ ] 손실 회피·사회적 증거·앵커링 중 하나를 다음 콘텐츠에 적용
- [ ] 공유 유도형 인스타 포맷 1개 개발 (데이터 카드, 사례 비교)
- [ ] 영업 제안서에 "트래픽 → 전환 구조도" 1페이지 추가
- [ ] 기존 계약 사장님 성공 사례 1건 콘텐츠화 (사회적 증거 + 팬덤 유인)

## 10. 경쟁·차별화

- [ ] 외국인 관광객 증가 통계 데이터 수집 (2024~2025 기준)
