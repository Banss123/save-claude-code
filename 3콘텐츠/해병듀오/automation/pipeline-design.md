# 메타 리드 자동화 파이프라인 설계서

## 환경

- n8n: v2.9.4, 셀프호스팅 (미니PC)
- 접속: https://genius-n8n-cloudflared.shop
- 외부 웹훅: Cloudflare Tunnel (활성)

---

## 전체 흐름

```
[고객이 메타 광고 클릭]
       ↓
[인스턴트 양식 제출]
       ↓
[Meta Lead Ads] ──Webhook──→ [n8n]
                                │
                                ├─→ [데이터 가공] 필드 정리 + 포맷팅
                                │
                                ├─→ [디스코드 웹훅] 실시간 알림
                                │
                                └─→ [구글 시트] 자동 기입
```

---

## n8n 워크플로우 노드 구성 (5개 노드)

### 노드 1: Meta Lead Ads Trigger
- 유형: Meta Lead Ads Trigger (n8n 내장 노드)
- 역할: 메타 인스턴트 양식에 새 리드가 들어오면 자동 트리거
- 설정 필요:
  - Meta 비즈니스 계정 OAuth 연결
  - 페이지 선택
  - 양식(Form) 선택

### 노드 2: Set (데이터 가공)
- 유형: Set 노드
- 역할: 메타에서 들어온 raw 데이터를 정리된 필드로 매핑

매핑 테이블:

| 메타 양식 필드 | 정리된 필드명 | 비고 |
|---------------|-------------|------|
| ad_name | 담당자 | 광고소재명 = 담당자명 (재원/민재). 해당 광고를 보고 들어온 리드를 그 담당자가 컨택 |
| full_name | 이름 | |
| phone_number | 전화번호 | |
| company_name | 업장명 | 커스텀 질문으로 수집 |
| (커스텀) | 업종 | |
| (커스텀) | 마케팅 이력 | |
| (커스텀) | 통화 가능 시간 | |
| created_time | 접수 시간 | 한국 시간(KST)으로 변환 |

### 노드 3: Date/Time (시간 변환)
- 유형: Date & Time 노드
- 역할: UTC → KST (Asia/Seoul) 변환
- 포맷: `YYYY-MM-DD HH:mm`

### 노드 4: Discord 웹훅
- 유형: HTTP Request 노드 (Discord Webhook URL로 POST)
- 역할: 새 리드 알림 전송

### 노드 5: Google Sheets
- 유형: Google Sheets 노드 (Append Row)
- 역할: 정리된 데이터를 시트에 자동 추가

## 구글 시트 포맷

시트명: `리드 관리`

| A | B | C | D | E | F | G | H | I |
|---|---|---|---|---|---|---|---|---|
| 접수일시 | 업장명 | 담당자 | 업종 | 이름 | 전화번호 | 마케팅 이력 | 통화 가능 시간 | 상태 |

- A~H열: n8n에서 자동 기입
- I열 (상태): 수동 관리 — 미연락 / 통화완료 / 상담중 / 계약완료 / 불발
- 시트 상단에 헤더 행 고정 (freeze)
- 날짜 기준 내림차순 (최신이 위)

## 구현 순서

### Step 1: 디스코드 준비
1. 해병듀오 디스코드 서버에 `#리드알림` 채널 생성
2. 채널 설정 → 연동 → 웹훅 생성 → URL 복사

### Step 2: 구글 시트 준비
1. 새 구글 시트 생성 → 시트명 `리드 관리`
2. 헤더 행 입력: 접수일시 | 업장명 | 담당자 | 업종 | 이름 | 전화번호 | 마케팅 이력 | 통화 가능 시간 | 상태
3. 헤더 행 고정 (보기 → 고정 → 1행)

### Step 3: n8n 워크플로우 구축
1. Meta Lead Ads Trigger 노드 추가 → Meta 계정 연결
2. Set 노드 → 데이터 필드 매핑
3. Date/Time 노드 → KST 변환
4. HTTP Request 노드 → 디스코드 웹훅 URL + JSON 포맷
5. Google Sheets 노드 → 시트 연결 + 열 매핑

### Step 4: 테스트
1. 메타 광고 관리자에서 테스트 리드 생성
2. 디스코드 알림 수신 확인
3. 구글 시트 기입 확인
4. 필드 매핑 정확도 검증
