# 캠페인 성과 분석

## Overview

메타 광고 캠페인의 핵심 지표를 분석하고, 예산 배분/타겟팅/크리에이티브 최적화 방향을 제안한다.
해병듀오와 모두마케팅 모두 적용.

## 입력

- 캠페인 데이터 (사용자 제공: 스크린샷, 텍스트, 스프레드시트 링크)
- 또는 기존 데이터: `해병듀오/해병듀오_캠페인데이터.md`

## 분석 항목

### 핵심 지표
| 지표 | 정의 | 벤치마크 (우리 기준) |
|------|------|---------------------|
| CPL | 리드 1건 획득 비용 | 6,752원 (캠페인1 기준) |
| CPA (CAC) | 계약 1건 획득 비용 | 109,716원 |
| 전환율 | DB → 계약 | 6% |
| ROI | 3개월 기준 투자 대비 수익 | 474% |
| CTR | 광고 클릭률 | 업종별 상이 |

### 분석 프레임
1. 퍼널 단계별 이탈률 (노출 → 클릭 → 양식 시작 → 양식 완료 → 영업 전환)
2. 크리에이티브별 성과 비교 (담당자별 = 소재별)
3. 타겟 세그먼트별 CPL 비교
4. 일별/주별 트렌드
5. 스케일업 시뮬레이션 (일 예산 변화에 따른 예상 DB/계약/수익)

## 출력 포맷

```
캠페인 성과 요약
- 기간: / 총 지출: / DB: / 계약:
- CPL: / CAC: / 전환율: / ROI:

핵심 발견
1. ...
2. ...

최적화 제안 (우선순위)
1. [높음] ...
2. [중간] ...

다음 액션
- ...
```

## 규칙

- 감이 아닌 수치로 판단. "잘 되고 있다"가 아니라 "CPL이 기준 대비 X% 하락"
- 벤치마크(캠페인1 실적)와 항상 비교
- 업종 미스매치 리스크 언급 (캠페인1 교훈: 요식업 병목)
- 스케일업 시뮬레이션 시 CPL 체증 가능성 반영
