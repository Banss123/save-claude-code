# 리드 퍼널 분석

## Overview

메타 광고 인스턴트 양식 → 영업 전환 퍼널을 분석하고 병목을 진단한다.
리드 품질, 영업 전환율, 이탈 패턴, 담당자별 성과를 포함.

## 입력

- 리드 데이터 (구글 시트 또는 사용자 제공)
- 영업 결과 데이터 (계약/미계약/부재/거절 등)
- 분석 기간

## 분석 프레임

### 1. 퍼널 단계별 전환율
```
광고 노출 → 클릭 (CTR)
    → 양식 시작 (양식 진입률)
    → 양식 완료 (양식 완료율)
    → 영업 연결 (연락 성공률)
    → 계약 (최종 전환율)
```

### 2. 리드 품질 분석
- 업종별 전환율 차이 (병의원 vs 뷰티 vs F&B)
- 마케팅 이력별 전환율 (경험 有 vs 無)
- 시간대별 리드 품질 (통화 가능 시간 패턴)

### 3. 담당자별 영업 성과
- 민재 vs 재원 전환율 비교
- 소재별 리드 배분 균형

### 4. 병목 진단
- 가장 큰 이탈이 발생하는 단계 특정
- 이탈 사유 패턴 (비용, 타이밍, 니즈 불일치, 신뢰 부족)

## 출력 포맷

```
퍼널 분석 요약
- 분석 기간: / 총 리드: / 계약:
- 전체 전환율: X%

단계별 전환율
노출 → 클릭: X%
클릭 → 양식완료: X%
양식완료 → 연락성공: X%
연락성공 → 계약: X%

핵심 병목: [단계] — [사유]

개선 제안 (우선순위)
1. [높음] ...
2. [중간] ...
```

## 규칙

- 캠페인1 벤치마크(전환율 6%, CPL 6,752원)와 비교
- 업종 미스매치 패턴 반복 확인 (요식업 병목 교훈)
- 데이터 부족 시 "N이 부족하여 통계적 판단 어려움" 명시
- 개선 제안은 실행 가능한 수준으로 (추상적 조언 금지)
