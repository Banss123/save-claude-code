# 광고 카피 작성

## Overview

메타 광고(해병듀오/모두마케팅) 크리에이티브의 텍스트 요소를 작성하거나 개선한다.
광고 헤드라인, 본문, CTA, 인스턴트 양식 질문, 릴스 캡션/훅 포함.

## 입력

- 광고 목적 (리드 수집 / 브랜드 인지 / 트래픽)
- 타겟 업종 (병의원, 뷰티, F&B 등)
- 기존 카피 (개선 요청 시)
- 프로젝트 (해병듀오 / 모두마케팅)

## 카피 유형별 규칙

### 메타 광고 헤드라인
- 40자 이내
- 타겟의 핵심 페인 포인트 또는 결과 약속
- 숫자/구체적 수치 포함 시 CTR 상승

### 메타 광고 본문
- 125자 이내 (접기 전 노출 영역)
- 구조: 페인 포인트 → 솔루션 → CTA
- 감정 트리거 우선 (손실 회피, 사회적 증거)

### CTA
- 행동 유도 명확: "무료 상담 신청하기", "지금 확인하기"
- 인스턴트 양식 연결 시 양식 내용과 일관성

### 인스턴트 양식 질문
- 3-5개 이하 (이탈률 최소화)
- 영업에 필요한 최소 정보만 수집
- 해병듀오 필드: 담당자, 업종, 마케팅 이력, 업장명, 통화 가능 시간, 이름, 전화번호
- 모두마케팅 필드: 병원/클리닉명, 직함/직책, 이름, 전화번호

### 릴스 캡션/훅 (해병듀오 전용)
- 첫 3초 후킹 — 시청 유지율의 80%가 여기서 결정
- 캡션: 해시태그 포함, 질문형 마무리

## 출력 포맷

```
[카피 유형]

A안:
헤드라인: ...
본문: ...
CTA: ...

B안:
헤드라인: ...
본문: ...
CTA: ...

추천: A/B안 중 [X]안 — 이유: ...
```

## A/B 테스트 원칙

- 한번에 1개 변수만 변경 (헤드라인 OR 이미지 OR 타겟, 동시 변경 금지)
- 최소 1,000 노출 후 판단
- CTR과 CPL 모두 확인 (CTR 높아도 CPL이 높으면 의미 없음)
