# 해병듀오 릴스 편집 기획

## Overview

대본을 받아 프리미어 프로 편집 큐시트를 생성한다.
컷 포인트, 줌인, 그래픽 팝업, 효과음을 구간별로 지정.
자막은 Vrew(브루)로 별도 생성 — 컷 확정 후 돌릴 것.

참조 대본: `해병듀오/content/릴스_뷰티샵_대본_v3.md`

---

## 고정 요소 (매 편집 동일, 묻지 않음)

### 시퀀스
- 해상도: 1080×1920 (9:16)
- 프레임레이트: 30fps

### 자막 스타일 (Vrew 스타일 기준)
- 폰트: 에스코어드림 Bold / 고딕 A1
- 크기: 65~75pt
- 위치: 화면 중앙~하단 1/3
- 기본: 흰색 + 반투명 검정 박스 (70%)
- 강조: 빨간색 #cc101f — 숫자, 핵심 단어

### BGM
- 분위기: 긴장감 트랩 비트 → 사례 구간에서 밝아짐
- 볼륨: 음성 대비 -12~-15dB
- 훅 시작: 서스펜스 / 비트 없이 시작 → "틀렸습니다" 또는 임팩트 단어에서 비트 인

### 브랜딩 컬러
- 강조 빨강: #cc101f
- 포인트 노랑: #ffcc00

---

## 편집 순서

```
1. 테이크 선택     원테이크 중 최적 1개 고르기
2. 러프 컷         뜸(0.3초+) 제거, 기본 타이밍 잡기
3. BGM 삽입        비트 먼저 깔고 컷을 비트에 맞춰 파인튜닝
4. ── 컷 확정 ──
5. Vrew 자막 생성  확정된 영상으로 Vrew 돌리기 → 프리미어로 가져오기
6. 그래픽 팝업     자막 위치 확인하면서 팝업 배치
7. 효과음 + 줌인   강조 포인트 기준으로 동시 작업
8. 오디오 믹싱     볼륨 레벨 정리
9. 내보내기
```

> 주의: Vrew는 반드시 컷 확정(4번) 이후에 돌릴 것.
> 컷이 바뀌면 자막 타이밍 전체가 틀어짐.

---

## 강조 판단 기준

대본에서 아래 패턴을 감지하면 효과 적용:

| 패턴 | 비디오 효과 | 효과음 |
|------|------------|--------|
| 전복 단어 ("틀렸습니다", "안 됩니다" 등) | 줌인 Scale 108% + 0.2초 홀드 | 임팩트 히트 |
| 매출/숫자 강조 ("1,000만원", "3배" 등) | 빨간 대형 텍스트 팝업 | 캐시 사운드 |
| 3요소 열거 ("노출, 리뷰, 예약" 등) | 하나씩 순차 팝업 | 팝 × 3 |
| 권위 수치 ("300개" 등) | 숫자 팝업 | 챠칭 |
| 경쟁 언급 ("옆 가게" 등) | 빨간 자막 강조 | — |
| 첫 마디 (훅 시작) | 0.3초 무음 후 줌인 펀치 | — |
| "실시!" | 줌인 + 군대 폰트 풀스크린 | 드럼 히트 |
| 아웃트로 | Freeze Frame 0.5초 + 로고 | "아-잇!" 시그니처 |

## 출력 포맷 (큐시트)

대본을 받으면 아래 형식으로 출력:

```
[구간명 — 예상 타임코드]
컷:    편집 지시 (줌인 여부, 홀드 등)
그래픽: 팝업 텍스트 내용 + 색상
효과음: 효과음 종류 + 타이밍
메모:  특이사항 (없으면 생략)
```

구간 순서: 훅 → 시장 기회 → 경쟁 공포 → 권위+사례 → CTA → 아웃트로

마지막에 **전체 예상 길이**와 **강조 포인트 요약** 추가.

---

## 입력 처리

사용자가 대본을 주면:
1. 대본 구간(훅/시장기회/경쟁공포/사례/CTA) 파악
2. 각 구간에서 강조 판단 기준에 해당하는 단어/패턴 추출
3. 구간별 큐시트 생성
4. 전체 예상 길이 계산 (말하기 속도 기준 25~35초 목표)

기존 대본(v3) 편집 요청 시: `해병듀오/content/릴스_뷰티샵_대본_v3.md` 직접 읽어서 처리.
