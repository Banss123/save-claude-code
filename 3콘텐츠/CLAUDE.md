## 4-way 리뷰 페르소나 (프로젝트 오버라이드)

Step 4(계획 수립)에서 복잡한 작업은 아래 4명으로 압박 테스트.

| 페르소나 | 시각 | 핵심 질문 |
|---|---|---|
| **시청자대표** | 시청 동기, 후킹, 유지율 | 첫 3초에 멈출까? 끝까지 볼 이유 있나? |
| **카피라이터** | 카피, 헤드라인, 훅 | 이 문구로 클릭/저장이 일어나나? 더 강한 단어 없나? |
| **알고리즘분석가** | 메타 CTR, 시청시간, CPL/ROAS | 데이터상 이게 다음 우선순위 맞나? |
| **썸네일디자이너** | 비주얼 임팩트, 대비, 가독성 | 모바일 작은 화면에서 읽히나? 경쟁 영상 사이에서 눈에 띄나? |

라우팅:
- 단순 카피 다듬기 → 카피라이터 1명
- 새 릴스 기획 → 시청자대표 + 카피라이터 + 썸네일디자이너 3명
- 캠페인 전략 변경 → 4명 다

## 에러 처리

- **n8n 장애 시**: 디스코드 직접 메시지로 리드 알림, 구글 시트 수동 입력으로 전환. 즉시 보고.
- **GAS 실행 실패 시**: 재실행 1회. 실패하면 정리 탭 수동 업데이트 후 보고.
- **스킬 실행 에러 시** (ffmpeg 미설치, 파일 경로 오류 등): 멈추고 에러 보고. 수동 대체 가능하면 대안 제시.
- **리드 데이터 필드 오류 시**: 원본 탭 확인 후 수정. 해병듀오/모두마케팅 필드 혼동 금지.
