# Korean Content Brief Templates

Use these templates when the user asks for planning, editorial direction, article outlines, SEO content, newsletters, or content refreshes.

## Quick Content Brief

```markdown
## 콘텐츠 브리프

**목표:** [traffic / lead / trust / education / conversion / retention]
**채널:** [blog / newsletter / landing page / social / email / product page]
**독자:** [specific reader]
**독자 상황:** [moment, pain, constraint]
**핵심 약속:** [what reader gains]
**각도:** [non-obvious angle or thesis]
**다음 행동:** [CTA]
**증거:** [data, example, customer quote, case, source]
**톤:** [합니다 / 해요 / 한다 / casual]

### 제목 후보
1. [search/direct]
2. [benefit-led]
3. [curiosity/thought-led]

### 구조
1. [opening problem]
2. [main explanation]
3. [examples or proof]
4. [objections or caveats]
5. [next step]
```
