| 단축어 | 의미 |

## 2. 스킬 목록 & 호출법

스킬은 `/스킬명` 또는 관련 업무 말하면 자동 트리거.

### 자주 쓸 스킬

| 스킬 | 언제 | 호출 |
|------|------|------|
| `pptx` | PPT 생성/편집/읽기 | `/pptx` 또는 "ppt 만들어줘" |
| `xlsx` | 엑셀 생성/편집/정리 | `/xlsx` 또는 "엑셀 파일 ~" |
| `pdf` | PDF 생성/읽기/편집 | `/pdf` 또는 "pdf로 ~" |
| `docx` | Word 문서 작업 | `/docx` |
| `brand-guidelines` | 브랜드 가이드라인 작성 | `/brand-guidelines` |
| `canvas-design` | 캔버스 디자인 | `/canvas-design` |
| `internal-comms` | 내부 커뮤니케이션 문서 | `/internal-comms` |
| `frontend-design` | 프론트엔드 UI 작업 | `/frontend-design` |
| `update-config` | hooks/settings.json 설정 | `/update-config` |
| `skill-creator` | 새 스킬 직접 만들기 | `/skill-creator` |
| `loop` | 반복 실행 (예: `/loop 5m /명령`) | `/loop` |
| `schedule` | 예약 자동 실행 에이전트 | `/schedule` |

## 3. 에이전트 목록 & 호출법

에이전트는 대화에서 이름 언급하거나 Claude가 자동 판단.

| 에이전트 | 역할 | 사용 예시 |
|----------|------|-----------|
| `archive-curator` | 자료 보내면 템플릿 맞춰 아카이브 저장 | "archive-curator로 이 링크 저장해줘" |
| `learning-note-writer` | 학습 내용을 노트 형식으로 정리 저장 | "오늘 배운 내용 노트로 정리해줘" |

## 4. 자주 하는 업무별 조합

### 리더스피부과 (+ 기타 병원 클라이언트) SEO 업무

| 업무 | 방법 |
|------|------|
| 제안서 PPT 작성/수정 | `/pptx` 자동 → PPT 파일 언급하면 됨 |
| 프로필 진단 PPTX | `/pptx` + 기존 파일 @참조 |
| 키워드 리스트 정리 | `/xlsx` |
| 진단 결과 문서화 | `/docx` 또는 `/pdf` |

**예시 명령:**
```
리더스피부과_명동점_프로필_진단.pptx 열어서
구글 프로필 최적화 항목 분석하고
개선 제안 슬라이드 추가해줘
```

---

### 나미브로우 (뷰티 클라이언트) 콘텐츠 업무

| 업무 | 방법 |
|------|------|
| 소식글 / 업장 소개 작성 | 직접 대화 (스킬 불필요) |
| 브랜드 가이드라인 정리 | `/brand-guidelines` |
| 콘텐츠 캘린더 (엑셀) | `/xlsx` |

---

### 구매대행 업무

| 업무 | 방법 |
|------|------|
| 주문서/발주서 엑셀 | `/xlsx` + 파일 @참조 |
| 선적 리스트 정리 | `/xlsx` |

**예시 명령:**
```
ship_list_20260406.xlsx 열어서
운송비 컬럼 추가하고 합계 자동계산 수식 넣어줘
```

## 5. 설정 파일 위치

| 파일 | 경로 | 용도 |
|------|------|------|
| 행동 규칙 | `~/.claude/CLAUDE.md` | 워크플로우, 단축어, 규칙 |

## 6. 현재 자동화 설정

### 훅 (settings.json)
- **Write/Edit 시**: `.env`, `.lock` 파일 → 자동 차단

### 훅 실패 규칙 (CLAUDE.md)
- 훅 실패 시 stderr 읽고 즉시 교정 시도
- 같은 훅 3회 연속 실패 → 멈추고 원인 보고

## 7. 유용한 패턴

### 파일 @참조해서 작업
```
@Desktop/리더스피부과/도곡점.txt 이 내용 기반으로
구글 프로필 최적화 체크리스트 만들어줘
```

### 여러 파일 동시 처리
```
전부 맡겨 - 리더스피부과 3개 지점 txt 파일 분석해서
각각 프로필 진단 리포트 docx로 만들어줘
```

### 설정 변경
```
/update-config - Write 이후 자동으로 prettier 실행하는 훅 추가해줘
```
