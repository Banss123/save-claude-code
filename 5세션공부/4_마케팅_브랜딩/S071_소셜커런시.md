# S071 — 소셜 커런시를 만드는 법

## 핵심 요약 (3줄)

- 사람들은 자신을 **좋아 보이게 만들어 주는 것**을 소문낸다
- 바이럴은 우연이 아니다 — 소문낼 이유를 **설계**해야 일어난다
- 소셜 화폐 5요소: **계기 / 감성 / 대중성 / 실용가치 / 이야기성**

---

## 핵심 원리

| 요소 | 의미 | 핵심 질문 |
|------|------|----------|
| 계기 | 반복적으로 떠올릴 트리거 | "이 서비스를 언제 생각하게 만들까?" |
| 감성 | 강한 감정 유발 (경이, 분노, 유머) | "어떤 감정을 건드리나?" |
| 대중성 | 남들도 한다는 증거 | "이미 쓰는 사람이 보이는가?" |
| 실용가치 | 공유하면 상대방에게 이득 | "이걸 알려주면 내가 유능해 보이나?" |
| 이야기성 | 스토리에 자연스럽게 녹아드는가 | "대화 중 자연스럽게 언급될 수 있나?" |

---

## 레퍼런스 시나리오 10개

1. **프리미엄 미용실** — "예약 대기 3주"라는 정보 자체가 소셜 화폐. 예약했다는 사실을 말하는 것만으로 본인의 안목을 증명함.

2. **B2B SaaS** — "우리 팀은 ○○ 쓰면서 보고서 작성 시간 50% 줄었어요" → 실용가치+이야기성. 공유하면 팀장처럼 보임.

3. **식당** — 특이한 메뉴 이름 또는 인스타 찍을 수밖에 없는 비주얼 → 계기+감성. SNS에 올리는 것 자체가 입소문.

4. **컨설팅 서비스** — 클라이언트에게만 제공하는 월간 인사이트 레터 → 실용가치. 지인에게 전달하면 "나 좋은 사람 알아"가 됨.

5. **의류 브랜드** — "이 브랜드는 환경부담금 전액 공개함" → 감성(경이+지지). 공유하면 가치관을 드러낼 수 있음.

6. **코칭/강의** — 수강생 성과 사례를 구체적 숫자로 공개 → 대중성+이야기성. "나도 저렇게 됐어" 스토리가 만들어짐.

7. **부동산 중개** — "이 동네 숨겨진 정보" 뉴스레터 → 실용가치. 지인에게 보내면 정보통 이미지 얻음.

8. **IT 스타트업** — 초대장 시스템(베타 접근 제한) → 대중성+계기. 초대받은 사람이 특별함을 느끼고 공유함.

9. **오프라인 소품샵** — 직접 만든 스토리가 담긴 태그 → 이야기성. 선물할 때 스토리를 같이 전달함.

10. **영업 대행/광고 대행** — 성과 보고서에 경쟁사 대비 수치 포함 → 실용가치. 클라이언트가 내부 보고할 때 그대로 씀 = 자연스러운 추천.
