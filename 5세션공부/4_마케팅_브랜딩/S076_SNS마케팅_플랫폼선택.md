## 핵심 요약 (3줄)

- 팔로워가 많을수록 노출이 줄어든다 — 1천명 계정 여러 개가 10만 계정 하나보다 효율적
- 인스타 = DB 수집(CPL), 유튜브 = 판매 전환(CPS) — 목적에 따라 플랫폼이 달라진다
- 마케팅 최신 전략은 UPASS: 유저처럼 보이는 UCC를 여러 계정에서 동시 발행 → 자연스러운 바이럴

---

## 핵심 원리

### 1. 팔로워 노출 역설

| 팔로워 수 | 실제 노출률 |
|---------|----------|
| 1천명 | 21% |
| 1만명 | 14% |
| 10만명 | 8.8% |
| 50만명 이상 | 5% |

→ 바이럴 전문가들이 팔로워 1천명짜리 계정을 여러 개 운영하는 이유
→ 10만 계정 키우려고 시간 쏟지 말고, 1천명 계정 10개 만들기가 더 효율적

---

### 2. 플랫폼별 KPI 강점

| 플랫폼 | 강한 KPI | 쓸 때 |
|--------|---------|------|
| 인스타그램 | CPL (고객 정보 수집당 비용) | DB 모으고 싶을 때 |
| 유튜브 | CPS (판매당 비용) | 돈 안 들이고 판매 극대화할 때 |
| 둘 다 | CPL + CPS 동시 | 스케일업 단계 |

→ 인스타에서 DB 모아서 → 유튜브 or 카카오톡 채널로 빼는 게 공식

---

### 3. 플랫폼별 금전적 가치 로드맵

```
[낮음 → 높음]

인스타 포스팅만          → 금전적 가치 낮음
지속적 인스타 포스팅     → 금전적 가치 중간
전자책 출판             → 금전적 가치 중상
전자책 → 유료강연       → 금전적 가치 높음
유튜브 채널             → 금전적 가치 높음
유명 출판사 책 출판     → 금전적 가치 최고
```

→ 인스타 혼자로는 돈이 안 됨. 반드시 다음 단계로 연결해야 함

---

### 4. 마케팅 모델 진화

| 모델 | 흐름 | 시대 | 장기 효과 |
|------|------|------|---------|
| AIDMA | 주의→흥미→욕구→기억→행동 | TV 시대 | 단기 효과적 |
| AISAS | 주의→흥미→검색→행동→공유 | 인터넷 시대 | 단기 효과적 |
| UPASS | UCC→흥미→브라우징→구매→확산 | 현재 최신 | 장기 극대화 |

---

### 5. UPASS 전략 (핵심)

```
여러 계정에서 "유저처럼 보이는" UCC 게시물 동시 발행
        ↓
사람들이 흥미 → 검색 → 비슷한 글 여러 개 발견
        ↓
신뢰 형성 → 구매
        ↓
구매자가 다시 공유 (자연스러운 바이럴 완성)
```

실제 사례: LG 그램 UCC 마케팅 — "내가 직접 홍보해 준다"는 유저들이 SNS를 뒤덮음. 실제로는 바이럴 마케팅 팀이 만든 것.

## 레퍼런스 시나리오 10개

1. **인테리어 디자이너** — 팔로워 10만 계정 하나 대신, 지역별 1천명 계정 10개 운영. 각 계정에서 "이 디자이너한테 맡겼더니" 형식의 UCC 스타일 후기 발행 → 검색하면 비슷한 글 여러 개 나옴 → 신뢰 형성.

2. **B2B 영업 서비스** — 인스타에서 잠재 고객 DB 수집 → 카카오톡 채널로 이동 → 정기 콘텐츠로 관계 유지 → 적절한 시점에 제안. 인스타는 입구 역할만.

3. **퍼스널 트레이너** — 인스타 릴스로 DB 모음 → 카카오톡 오픈채팅 → 무료 1회 PT 제안 → 유료 전환. 팔로워 수보다 DB 수가 핵심 KPI.

4. **의류 셀러** — 팔로워 5만 계정 하나보다, 1천명 패션 계정 5개 운영. 각 계정에서 같은 옷 다른 스타일링으로 발행 → 노출 총량이 훨씬 높음.

5. **컨설팅 서비스** — 유튜브에서 무료 강의 콘텐츠 발행(CPS 최적화) → 댓글/링크로 인스타 연결(CPL) → 인스타 팔로워에게 유료 컨설팅 제안.

6. **식품 브랜드** — 실제 소비자처럼 보이는 UCC 계정 여러 개에서 "내돈내산 후기" 동시 발행 → 네이버/인스타 검색 시 같은 제품 후기 다수 노출 → 구매 결정 유도.

7. **학원/교육** — 인스타에서 입시 팁 콘텐츠 지속 발행 → DB 수집 → 전자책 무료 제공 → 유료 강의 연결. 팔로워보다 이메일/전화 DB가 실제 매출과 직결.

8. **부동산 에이전트** — 지역 커뮤니티 느낌의 UCC 계정 운영 ("실거주자가 알려주는 ○○동 정보") → 실제 매물 문의로 자연스럽게 연결.

9. **뷰티 브랜드 신규 런칭** — AISAS 활용: 인스타 광고로 노출 → 사람들이 네이버/인스타에서 검색 → UCC 스타일 후기 미리 배포해놓기 → 검색 결과에 긍정 콘텐츠 가득.

10. **코칭/멘토링** — 인스타를 "증명의 용도"로 활용. 포트폴리오/성과 사례를 꾸준히 아카이빙 → 처음 만나는 사람이 인스타 보고 신뢰 형성 → 영업 클로징 속도 향상.

## 내 상황 적용

**현재 구조:**
비즈하이 마케팅 서비스 판매 / 인스타 릴스 + 메타광고 / 팔로워 30명
이전 캠페인(요식업&뷰티 해외마케팅): DB 66개 → 계약 4개 (전환율 6%)
DB → TM 직접 영업

- [x] 목적: DB 수집(CPL) + 판매 전환 → 메타광고가 맞는 선택. 팔로워 30명은 문제 없음. 메타광고는 팔로워 관계없이 노출됨.
- [x] DB 빠지는 곳: TM → 직접 영업 (인스타 → 폼/DM → 전화)
- [ ] UPASS 적용 포인트: "계약한 원장님/사장님이 직접 후기 올리는 것처럼" 보이는 UCC 콘텐츠 → 요식업/뷰티 타겟 피드에 배포 가능한가?: ___________

**세션 기준 현재 구조 진단:**
- 인스타(CPL) → TM(CPS) 구조 정석. 팔로워 30명은 메타광고에서 무관.
- 전환율 6% = 돌방(2~3%) 대비 2배 이상. TM 영업 자체는 문제 없음.
- **진짜 병목: 업종-예산 미스매치**
  - 요식업 → 해외 마케팅 비용 감당 불가 → 비용 듣고 이탈
  - 병원&뷰티 → 객단가 높고 해외 마케팅 ROI 설득 가능
- **CPL 개선 방향**: 타겟 업종 선택 = DB 질의 90%. 광고 소재보다 업종 필터링이 우선.
- 현재 병원&뷰티 전환으로 DB 질 자체가 올라갈 것. 전환율 10% 목표 가능.
