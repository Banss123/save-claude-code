## 세션 1개 처리 흐름

```
1. YouTube 영상 ··· → "스크립트 열기" → 전체 복사
2. 자막 텍스트를 Claude에 붙여넣기
   → "이 세션 지식카드 만들어줘" 한 마디면 끝
3. 생성된 카드에서 "내 상황 적용" 3칸 채우기
4. MASTER_INDEX 상태 ⬜ → ✅ 업데이트
5. SITUATION_MAP 해당 상황에 링크 추가
```

## 자막 추출 방법

### 기본 (YouTube 내장)
```
1. 영상 재생 화면 하단 ··· 클릭
2. "스크립트 열기" 선택
3. 우측에 열리는 스크립트 패널 전체 선택 → 복사
4. Claude에 붙여넣기
```
타임스탬프 포함/제외 토글 가능. 설치 불필요, 가장 빠름.

### 대안 (yt-dlp, 파일로 저장할 때)
```bash
yt-dlp --write-auto-sub --sub-lang ko --skip-download --convert-subs srt -o "%(title)s" [URL]
```

### 대량 (플레이리스트 전체)
```bash
yt-dlp --write-auto-sub --sub-lang ko --skip-download -o "%(playlist_index)03d_%(id)s" [PLAYLIST_URL]
```

## Claude에게 요청하는 표준 프롬프트

### 지식카드 생성
```
아래는 [세션 제목] 자막이야.
지식카드 형식으로 만들어줘:
- 핵심 요약 3줄
- 핵심 원리 (표 형식)
- 레퍼런스 시나리오 10개 (비즈니스 상황, 다양한 업종, 3줄 이내)
- 내 상황 적용 필드 (빈칸으로)
- 연결 세션 추천

[자막 붙여넣기]
```

### 핵심 구간 추출 (시청 전)
```
이 자막에서 가장 중요한 구간 5-7곳을 타임스탬프로 알려줘.
각 구간이 왜 중요한지 한 줄로.

[자막 붙여넣기]
```

## 폴더 구조

```
3 세션공부/
├── _INBOX/
│   └── 1_시청중/       ← 지금 보고 있는 영상 자막 (임시)
├── _INDEX/
│   ├── MASTER_INDEX.md ← 전체 현황 (여기서 시작)
│   ├── SITUATION_MAP.md ← 상황별 탐색 (실전 사용)
│   └── HOW_TO_USE.md   ← 이 파일
├── _transcripts/       ← YouTube 자동 자막 VTT + 분류 스크립트
├── _체크리스트/        ← 세션 쌓이면 자연히 생성
├── skills/             ← 커스텀 스킬
├── 1_사업구상/
├── 2_사업구조/
├── 3_영업_협상/
├── 4_마케팅_브랜딩/
├── 5_조직_경영/
└── 6_번외_프로젝트/
```

## 주간 루틴

```
[공부 시작 전]
→ SITUATION_MAP 열기
→ "지금 내 상황"에 맞는 세션 선택

[주 1회 리뷰]
→ 처리한 세션들 "내 상황 적용" 칸 확인
→ 실제로 써먹었는지 체크
→ 써먹었으면 _체크리스트에 반영
```
