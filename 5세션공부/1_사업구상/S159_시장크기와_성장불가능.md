## 핵심 요약 (3줄)
- TAM이 크다고 확장 가능한 사업이 자동으로 만들어지는 건 착각이다 (시장 착시)
- 실제 사업은 TAM이 아니라 RPM(복제 가능하고 실현 가능한 시장)에서 이루어진다 — 고객 획득 → 가치 제공 → 수익 반복 창출이 가능한 구조인가?
- 성장이 안 되면 외부 환경 탓이 아니라 내부 프로세스의 5가지 병목(메시지/세그먼트/채널/전환/운영) 중 어디서 막혔는지를 찾아야 한다

## 핵심 원리

### 1. 시장 착시 (Market Size Illusion)

TAM이 클 때 사람들이 빠지는 3대 착각:

| 착각 | 현실 |
|------|------|
| 일부만 먹어도 크게 벌 수 있다 | RPM(실현 가능한 시장)이 없으면 0원 |
| 진입만 하면 성장하기 쉽다 | 시장 크기와 확장 가능성은 별개 |
| 브랜드만 알려지면 대박 난다 | 구조 없이는 월 2천만원이 한계 |

### 2. RPM — Replicable & Playable Market

TAM 대신 봐야 할 진짜 시장 판단 기준

```
RPM 검증 질문:
  고객을 획득할 수 있는가? (접근성)
  → 가치를 제공할 수 있는가? (전환성)
    → 수익을 반복해서 창출할 수 있는가? (재현성)

이 3가지가 반복 구조 안에서 일어나지 않으면
전체 시장 크기는 아무 의미 없다
```

| 구분 | TAM | RPM |
|------|-----|-----|
| 관점 | 외부 가능성 (인구/구매력/문화) | 내부 실행 가능성 |
| 속도 | 느리게 변함 | 빠르게 설계/수정 가능 |
| 활용 | 투자 유치, 장기 비전 | 고객 획득 전략, GTM 설계 |
| 측정 | 고객수 × 예상 소비액 | 전환율 × 반복성 × 고객 세그먼트 크기 |
| 관계 | RPM은 TAM의 부분집합. TAM이 커도 RPM은 0일 수 있음 |

### 3. TAM → RPM 전환 3단계

```
[1] 세그먼트 분할
  전체 시장을 하위 세그먼트로 분류
  (지역/연령/행동/양식/대체제)
  → 유사한 행동을 가진 소규모 고객군 도출
         ↓
[2] 반복 가능성 검증
  해당 세그먼트에서 획득→전환→유지가
  일정 수준 이상 반복되는지 확인
  → 안 되면 [1]로 돌아가서 타겟/메시지 재설계
         ↓
[3] 확장 가능성 검증
  해당 세그먼트를 넓게 확장하거나
  유사 세그먼트로 이동 가능한지 분석
  → 확장 후에도 일관된 가치 전달이 가능한지 확인
```

**락토핏 사례**: 다이어트(살 빼고 싶은 사람) vs 슬림(라인 정리하고 싶은 사람) — 제품은 동일하지만 세그먼트를 키워드로 확장한 케이스

### 4. 확장 가능성의 3요소

| 요소 | 정의 | 핵심 질문 |
|------|------|----------|
| 재현성 | 같은 품질의 상품/서비스를 새 고객에게 계속 제공 가능한가 | 획득 경로 동일? 전환율 유지? 가치 전달 동일? |
| 단위 확장성 | 고객 1명 획득·유지 비용 < 이익인가 | CAC < LTV? 규모의 경제 작동? |
| 성장 방해 요소 | 확장을 막는 치명적 요소 존재 여부 | 조직/비용/기술/인적 제약은? |

**열심히의 한계**: 월 2천만원까지는 열심히 살면 도달 가능. 그 이상은 구조와 전략이 필요

### 5. 재현성 5대 병목

| 병목 | 증상 | 대응 |
|------|------|------|
| 메시지 확장 | 전환율 하락 — 세그먼트 확장 시 기존 메시지와 충돌 | 세그먼트별 맞춤형 메시지 재설계 |
| 세그먼트 포화 | CAC 증가, ROAS 급감 | 미세 세그먼트를 새롭게 찾아내기 |
| 채널 스케일 | 신규 유입 정체, 특정 채널 의존도 70%↑ | 제휴/파트너십 or 새 유입 채널 구축 |
| 전환 | 고객 이탈 상승 | 여정 내 이탈 지점 분석 → 온보딩 재구축 |
| 운영 | 고객 증가 속도 > 내부 대응 속도 | 업무 맵핑 → 자동화 가능 부분 전환 |

**BJI 사례**: 구글 SEO 판매 시 "국내 시장은 돈 안 된다"고 어필했다가 네이버 상품 확장하면 메시지 충돌 발생

### 6. 단위 확장성 검증 4요소

| 요소 | 정의 |
|------|------|
| CAC | 고객 1명 획득 비용 |
| LTV | 고객 1명이 평생 가져다 주는 총 수익 |
| 재구매율 | 이탈 없이 지속 결제하는 비율 |
| 투자금 회수 기간 | CAC 회수까지 걸리는 시간 |

→ 4가지 모두 유기적 연결. 성장할수록 고객당 이익이 보장되지 않으면 **규모의 저주**에 빠짐

### 7. 일관된 가치 전달 (핵심 원칙)

- 확장 전에 반드시 시즈 모드로 박아놔야 할 것: **일관된 가치 전달**
- 확장하면서 "이번에 이거 하느라 이걸 못 했어요"가 가장 흔한 실패 패턴
- 서비스/상품을 늘리기 전에 기존 가치 전달이 흔들리지 않는지 먼저 확인

### 8. 취미 → 능력 → 재화 공식

```
취미 (재미있다)
  → 딥하게 공부 (끝장을 본다)
    → 능력이 된다
      → 재화의 가치가 생긴다

* 대부분은 취미에서 끝남. 능력이 될 때까지 파야 함
```
