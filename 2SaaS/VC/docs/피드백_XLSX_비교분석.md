## 수정 우선순위

### P0: 구조 자체가 다름 (먼저 수정)

#### 1. J~N열 옵션 영역 추가
- **원본**: 메뉴(C~H) | I열 구분선 | 옵션(J~N) 병렬 구조
  - J: 옵션그룹 번호 (#)
  - K: 옵션그룹명 + 조건 (필수/선택)
  - L: 옵션 목록 (개별 행)
  - M: 속성 (필수선택/선택)
  - N: 추가가격
- **생성물**: G~H열에 텍스트로 압축 (구조 완전 다름)
- **수정**: J~N 5개 열 추가, I열 구분선(너비 2.0), 옵션을 행 단위로 풀어서 배치

#### 2. 옵션 행 단위 배치
- **원본**: 옵션그룹 헤더 행 + 그 아래 옵션 항목 행들 (독립 흐름)
- **생성물**: 메뉴와 같은 행에 콤마 구분 텍스트
- **수정**: 메뉴 영역과 옵션 영역이 독립적으로 행을 사용하는 구조로 변경

### P1: 스타일 보정

#### 3. 색상 체계 전면 교체
| 요소 | 원본 | 생성물 | 수정 |
|------|------|--------|------|
| 타이틀(1행) bg | `#D9E1F2` (연파랑) | 없음 | `#D9E1F2` |
| 타이틀 폰트 | 맑은 고딕 11pt bold, center | 13pt bold, 녹색 | 11pt bold, center |
| 헤더(2행) bg | `#FBE4D5` (연주황) | `#1E3A32` (어두운 녹) | `#FBE4D5` |
| 헤더 글자색 | 기본(검정) | 흰색 | 기본으로 변경 |
| 메뉴그룹(C열) bg | `#BDD7EE` (연파랑) | `#FBE4D5` (연주황) | `#BDD7EE` |
| 구성(F열) bg | `#EAF4FB` (아주 연한 파랑) | 없음 | `#EAF4FB` 추가 |
| 설명(G열) bg | `#FEF9E7` (아주 연한 노랑) | 없음 | `#FEF9E7` 추가 |
| 옵션# 헤더(J열) bg | `#1F4E79` (남색) | 없음 | 추가 |
| 변경 표시 | bold만 (색상 없음) | bold + `#E2EFDA` 배경 | 배경색 제거, bold만 |

#### 4. C열 병합 제거
- **원본**: 병합 안 함 (첫 행에만 값, 이하 빈 셀)
- **생성물**: 세로 병합
- **수정**: 병합 제거, 첫 행만 값 입력

#### 5. 행 높이/열 너비
- **행 높이**: 35.1 고정 (현재 22)
- **열 너비**: 원본 값으로 교체
  ```
  A=1.0, C=16, D=26, E=10, F=28, G=?, H=14, I=2.0, J=4, K=22, L=38, M=7, N=9
  ```

### P2: 세부 보정

#### 6. 기타
- wrap_text: 전 셀 True
- 타이틀 병합 범위: C1:N1 (현재 C1:H1)
- 옵션그룹 간 빈 행 삽입
- 가안에서 G열(설명) 대부분 삭제 (None 처리)
