# 솔루션 판단 규칙 체계 (v1.0)

> 작성일: 2026-04-10
> 근거: 레퍼런스 6건 역추출 (갓꾸운닭, 강시떡, 빅스타피자, 부엉이돈까스, 스테이크존, 더피플버거)
> 상태: 초안 — 솔루션 담당자 피드백 대기

---

## 1. 진단 규칙 (현황 → 문제 식별)

| # | 조건 (IF) | 진단 (THEN) | 근거 사례 |
|---|-----------|-------------|-----------|
| D1 | CVR < 지역평균 × 0.6 | **전환 구조 실패** — 메뉴판 VMD + 사진 자산 최우선 | 부엉이(4.8% vs 10%+), 갓꾸운닭(15% vs 21%) |
| D2 | CTR < 2% | **클릭 유도력 부족** — 메뉴사진/설명 개선 우선 | 갓꾸운닭(1.9%) |
| D3 | CTR ≥ 4% AND 광고 예산 소규모 | **노출 효율 양호** — 예산 확대로 볼륨 스케일링 | 빅스타피자(CTR 4.1%, 예산 20만→200만) |
| D4 | 별점 ≤ 4.6 | **리뷰 리스크** — 리뷰이벤트 게시 중단 + 자연 리뷰 유도 | 스테이크존(4.6) |
| D5 | 카테고리 진입 < 3개 | **노출 채널 부족** — 도시락/한그릇 등 신규 카테고리 입점 | 스테이크존, 갓꾸운닭 |

## 2. 메뉴판 규칙

| # | 조건 (IF) | 액션 (THEN) | 근거 |
|---|-----------|-------------|------|
| M1 | 주요 키워드가 옵션으로 매몰 | 독립메뉴로 분리 (검색 노출 확보) | 부엉이(히레/로스카츠 옵션→독립) |
| M2 | 메뉴명에 카테고리 키워드 불통일 | 모든 메뉴명에 업종 키워드 추가 | 빅스타피자('피자' 통일) |
| M3 | 메뉴모음컷 미보유/미흡 | 단체 촬영 + 메뉴모음컷 1번 배치 | 강시떡, 갓꾸운닭 |
| M4 | 메뉴설명에 리뷰 키워드 미반영 | 리뷰 키워드 추출 → 설명문 삽입 | 갓꾸운닭("촉촉함","양많음"), 부엉이("바삭함") |
| M5 | 대표메뉴가 최상단 아님 | 대표메뉴 최상단 배치 (리뷰이벤트 하단으로) | 빅스타피자 |
| M6 | 객단가 < 25,000원 | 세트메뉴/밥류 추가 → 한그릇 카테고리 진입 | 갓꾸운닭(26K), 스테이크존 |

## 3. 광고 전략 규칙

| # | 조건 (IF) | 액션 (THEN) | 근거 |
|---|-----------|-------------|------|
| A1 | 배민 신규 진입 또는 구조 정비 전 | CPC 300~450원 범위, 예산 200만원 기본 | 6건 공통 (388~451원, 200만원) |
| A2 | 전환 구조 정비 완료 후 | CPC 단계적 인하 (451→388원) | 갓꾸운닭, 부엉이 |
| A3 | 쿠팡이츠 신규 | CMG 9~11% (신규), 4~5% (재주문) | 6건 공통 |
| A4 | 매장명이 자연 검색 유입 있음 | 매장명을 광고 금지어에 추가 (중복 지출 방지) | 빅스타피자, 스테이크존 |
| A5 | 쿠폰/할인이 분산 | 단일 할인으로 통합 (고객 명확성↑) | 부엉이(3종→즉시할인 2,000원 단일) |
| A6 | 신메뉴 런칭 시 | 할인 선행 필수 (할인 없이 노출만으로 전환 어려움) | 강시떡 |

## 4. 운영 규칙

| # | 조건 (IF) | 액션 (THEN) | 근거 |
|---|-----------|-------------|------|
| O1 | 항상 | 숨김/품절 절대 금지 (운영지수 + CTR 치명적) | 강시떡, 부엉이 |
| O2 | 항상 | 조리시간 20분 이하, 접수시간 9초 이하 | 빅스타피자, 갓꾸운닭 |
| O3 | 조리시간 준수율 < 95% | 운영지수 개선이 광고 확대보다 우선 | 갓꾸운닭 |
| O4 | 최소주문금액 > 지역 경쟁 평균 | 인하 검토 (진입장벽 제거) | 빅스타피자(4500→2000), 스테이크존(24900→11900) |
| O5 | 최소주문금액 플랫폼별 차이 > 5,000원 | 플랫폼 간 통일 | 스테이크존 |
| O6 | 광고 확대 시 | 피크타임 접수율 집중 관리 (노출↑ → 미접수 리스크) | 스테이크존, 빅스타피자 |

## 5. 수수료 규칙

| # | 규칙 | 근거 |
|---|------|------|
| F1 | 성과 연동 구조: 목표 매출 미달성 시 수수료 미발생 | 6건 공통 |
| F2 | 구간별 차등: 목표1 달성 = 3%, 목표2 달성 = 5% | 강시떡(1200만/3%, 1800만/5%), 스테이크존(2000만/3%, 2500만/5%) |
| F3 | 목표 매출은 현재 매출 × 2~2.5배 수준으로 설정 | 스테이크존(1000만→2000~2500만) |

## 6. 업종별 특수 규칙

| 업종 | 특수 고려사항 |
|------|-------------|
| **치킨** | 맛 단계(매운맛) 옵션 명시, "가성비/양많음" 키워드 강조, 후라이드 기본메뉴 반드시 활성화 |
| **분식** | 가성비 업종 — 할인 선행 필수, 세트구성(떡볶이+음료) 한그릇 입점 |
| **피자** | 사이즈별 분류가 메뉴 구조의 기축, 반반피자 한그릇 입점, 1인(퍼스널) 진입장벽 제거 |
| **돈까스** | 부위별 선호도(히레/로스) → 독립메뉴 분리, 시즈널 메뉴(냉모밀), 감각적 키워드("바삭함") |
| **양식/스테이크** | 프리미엄 가격대 유지 + 도시락으로 가격 민감층 확보, 숙성/HOT 등 특색 강조 |
| **버거** | 프리미엄 키워드(한우/1++) 상단 고정, 트렌드 분류(BBQ/트러플), 사이드 필수 구성 |

|------|------|-------------|
| 규칙 기반 (D1-D5, M1-M6, A1-A6, O1-O6, F1-F3) | **26개** | 시스템 프롬프트로 바로 적용 가능 |

## 미확정 사항 (담당자 피드백 필요)

- [ ] CPC 388~451원, 예산 200만원이 업종 불문 고정인지, 업종별로 달라지는지?
- [ ] CVR/CTR 임계값이 지역/업종별로 다른 기준이 있는지?
- [ ] 수수료 구간(3%/5%)과 목표 매출 배수(×2~2.5)가 고정 규칙인지?
- [ ] 한그릇/도시락 카테고리 진입 기준이 객단가 외 다른 조건이 있는지?
- [ ] 위 규칙 중 "이건 틀렸다" 또는 "이건 빠졌다" 항목?
