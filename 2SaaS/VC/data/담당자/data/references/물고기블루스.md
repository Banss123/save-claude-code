## 가격 구조

### 회 크기 3단 (모둠/단품 회)
- **소(1~2인): 48,000원** / **중(2~3인): 75,000원** / **대(3~4인): 98,000원**
- 대방어: 중 80,000 / 대 120,000
- 단품(광어/참돔/연어/농어/숭어): 중 42,000~45,000 / 대 65,000~69,000
- 대방어 단품: 중 79,000 / 대 95,000

### 1인 숙성사시미 (단일 가격)
- 광어/참돔/농어/연어: 각 30,000원

### 반반회 (중/대)
- 광어+연어/광어+참돔: 중 65,000 / 대 85,000
- 광어+방어: 중 78,000 / 대 98,000
- 광어+참치: 중 100,000 / 대 125,000

### 참치회
- 참치모둠 70g: 70,000 / 100g: 20,000(오류? — 참치모둠 100이 20,000은 아마 20,000 단위 100g 추가 가격으로 추정)
- 참치뱃살 5pcs: 20,000 / 10pcs: 35,000

### 기타
- 물회: 소 35,000 / 중 45,000
- 탕류: 20,000~28,000
- 식사류: 11,000~20,000
- 사이드: 2,000~12,000
- 음료: 1,500원 (355ml)

## 옵션 그룹 (가안 기준 5개)

1. 회&사이드 추가선택 — 회 추가(14점) 25,000 / 전복 추가(2~3마리) 10,000 / 와다 추가 10,000 / 우니 추가 20,000 / 안키모 추가 5,000 / 타코와사비 추가 4,000 / 묵은지 추가 2,000 / 무순 추가 500 / 생와사비 추가 300

## 변경 포인트

### 1. 메뉴명에 [용도/원산지/등급] 브래킷 삽입
- [혼술 안주] / [인기 혼술] / [오늘은 일식] / [인기 술안주] — 용도
- [노르웨이산] — 원산지
- [프리미엄] — 등급
- [기름기 꽉 찬] / [부드러운 식감] / [살얼음 동동] — 상태

### 2. 모둠 숙성회 1인 신설
- 현안 "오마카세 모둠숙성회 소(1~2인)" → 가안 **"[혼술 안주] 1인 모둠숙성회"** — 혼술 시그널

### 3. 반반회 네이밍 `&` → `+` 전환
- 광어&연어 → 광어+연어 숙성회 (가독성)

### 4. 가격 인원 표기 명시화
- 현안: "1~2인분:48,000" → 가안: "소(1~2인분):48,000" / 중(2~3인분) / 대(3~4인분)

### 5. 옵션 그룹 체계화
- 회&사이드 추가(1) / 음료(2) / 사이드(3) / 단품 추가(4) / 리뷰이벤트(5)
- 현안의 4개 옵션 그룹(회&사이드/음료/사이드/단품) + 리뷰이벤트 추가

### 6. 단품 숙성회에 옵션 풀 적용
- 현안은 단품회에 옵션 1번만 / 가안은 1,4,3,2,5 5개 적용 — 모든 회 메뉴에 동일 옵션셋 적용

### 7. "아주매운" → "[아주 매운]" 브래킷 스타일
- 강조 형용사를 브래킷으로

### 8. 사이드 카테고리 헤더
- "사이드메뉴" → "필수 추가 사이드메뉴" (업셀 시그널)

## 학습 인사이트

1. **회 업종 사이즈 3단 기본** — 소(1~2인) / 중(2~3인) / 대(3~4인) — 신만선횟집의 4단(소/중/대/특대)과 차이
2. **1인 진입 메뉴 2개 라인업** — 1인 숙성사시미(30,000) + 1인 모듬숙성회(48,000) — 혼술 객단가 구간 2개
3. **숙성회 카피 축**: 원산지([노르웨이산]), 등급([프리미엄]), 용도([혼술 안주]/[인기 혼술]/[오늘은 일식]), 상태([기름기 꽉 찬]/[부드러운 식감]/[살얼음 동동])
4. **반반회는 별도 카테고리** — [조합] 브래킷 불필요 (메뉴명 자체가 "광어+참치 숙성회")
5. **동일 옵션셋(1,4,3,2,5) 모든 회 메뉴 일괄 적용** — 회 + 사이드/음료/단품/리뷰 5종 조합 일관성
6. **리뷰이벤트 조건 "중복 불가"** — 다중 선택 시 랜덤 1개 지급 룰 명시
7. **탕류 카테고리 헤더 "숙성회 옆자리엔 뜨끈한 탕류"** — 회 + 탕류 크로스셀 시그널
8. **옵션명을 소비자 지향 카피로** — "사이드 추가선택" → "없어선 안될 사이드메뉴"
