# 동경카레&돈까스 — 메뉴판 가안 학습 노트

## 핵심 정체성 (가게 메시지)
- **"[일본풍 동경돈까스]"** — 카테고리 헤더에 일본풍 브랜딩 직접 노출
- **"돈까스와의 환상조합, 카레라이스"** — 돈까스+카레 결합 축
- **"빠지면 섭섭한 면류"** — 면 카테고리 필수성 강조
- **"소스듬뿍/고소한/달콤 가득/정통"** — 정통성·맛 키워드

## 가격 구조
- **돈까스**: 16,900~18,900원 (고구마돈까스 18,900이 최상위)
- **카레라이스 덮밥**: 12,000~14,500원 (가안에서 현안 대비 대폭 인상: 9,000→12,000, 10,500→13,500, 11,500→14,500)
- **면류**: 8,000~10,000원 (우동 8,000 / 마제소바 10,000 / 메밀소바 8,000)
- **음료**: 355ml 2,000원 / 뽀로로 2,000원

## 옵션 그룹 (가안 기준 4개)

1. [오늘은 더 든든하게, 카츠 추가] — 치즈카츠 2P 추가 +5,000 / 안심카츠 4P 추가 +6,000 / 등심카츠 추가 +8,000 / 카레라이스 추가 +9,000
2. [빠지면 섭섭한 면류] — 시원한 메밀소바 +8,000 / 느끼함 잡아주는 마제소바 +10,000 / 따끈한 우동 +8,000
3. [시원한 음료 추가] — 코카콜라 355ml +2,000 / 스프라이트 355ml +2,000 / 코카콜라 제로 355ml +2,000 / 뽀로로 밀크맛 +2,000 / 뽀로로 딸기맛 +2,000
4. [리뷰이벤트] — 리뷰는 꼭 작성해주세요 (중복 선택 시 랜덤으로 1개만 배송) / [el벤트] 함박카츠 1p\n카콜라 / [el벤트] 치킨카츠 1p 0원

→ **옵션명 자체도 `[헤더]` 형식 통일** — 시각 일관성

## 변경 포인트

### 1. 카테고리 헤더 `[헤더]` 대괄호 스타일 통일
- 돈까스 → [일본풍 동경돈까스]
- 카레라이스 → [돈까스와의 환상조합, 카레라이스]
- 면류 → [빠지면 섭섭한 면류]
- 시원한 음료메뉴 → [시원한 음료메뉴]
- 옵션명까지 동일 포맷 ([오늘은 더 든든하게, 카츠 추가])

### 2. 카레라이스 가격 대폭 인상 (현안 → 가안)
- 함박돈까스 카레라이스 덮밥: 10,500 → **13,500** (+3,000)
- 가라아게 카레라이스 덮밥: 10,500 → **13,500** (+3,000)
- 카레라이스 덮밥: 9,000 → **12,000** (+3,000)
- 새우카츠 카레라이스 덮밥: 11,500 → **14,500** (+3,000)
- → 전 라인업 동일 폭 인상 (가격 구조 재설계)

### 3. 메뉴명에 [] 브래킷 + 정통성/맛 키워드 삽입
- [소스듬뿍] / [고소한] / [달콤 가득] / [든든] / 정통 / 따끈한 / 시원한 / 느끼함 잡아주는

### 4. "일본식/일본풍" 정체성 카피 강화
- 모듬돈까스 → 일본식 모듬 돈까스세트
- 카테고리 헤더 [일본풍 동경돈까스]

### 5. 리뷰이벤트 "el벤트" 표기 (금칙어 회피)
- "이벤트" → "el벤트" 의도적 변형 (글로벌푸드 Z뷰 OI벤트와 동일 전략)

### 6. 옵션 [헤더] 포맷 통일
- 옵션명을 [카피 헤더] 형식으로 표기 → 카테고리 헤더와 일관성

### 7. 트리플 모듬돈까스 구성 명시
- 트리플 모듬돈까스 → 트리플 모듬 돈까스세트**(함박, 치즈, 안심)**

## 학습 인사이트

1. **가게명 기반 카테고리 브랜딩** — [일본풍 동경돈까스] — 카테고리 헤더 = 브랜드 재호출
2. **`[대괄호]` 스타일 전체 통일** — 카테고리 헤더, 옵션명 모두 대괄호로 장식. 시각적 일관성 패턴
3. **가격 일괄 인상 전략** — 카레라이스 전 라인업 +3,000 동일 폭 인상 (현안 → 가안)
4. **메뉴명 브래킷 축**: 정통성([일본풍]/정통), 양([소스듬뿍]/달콤 가득), 온도(따끈한/시원한), 용도([든든]/느끼함 잡아주는/[돈까스와의 환상조합])
5. **금칙어 회피 공통 패턴**: "이벤트" → "el벤트" (글로벌푸드 "OI벤트"와 유사)
6. **구성품 괄호 명시**: 트리플 모듬돈까스세트**(함박, 치즈, 안심)** — 소비자 선택 전 사전 인지
7. **뽀로로 음료**: 키즈/가족 타깃용 음료 옵션(밀크맛/딸기맛) 배치
8. **옵션 그룹 최소화(4개)** — 단순 돈까스/카레 전문점은 옵션 복잡도 최소화
