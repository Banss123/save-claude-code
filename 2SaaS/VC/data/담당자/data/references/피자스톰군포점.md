# 피자스톰 군포점 — 메뉴판 가안 학습 노트

## 핵심 정체성 (가게 메시지)
- **"[재료를 아끼지 않습니다] 폭풍 가성비 세트"** — 아낌없는 양 + 가성비
- **"[8년간의 노하우] 미국식 클래식 피자"** — 업력 수치 + 장르
- **"[최고급 프리미엄 토핑] 프리미엄 피자"** — 원재료 등급
- **"[내맘대로 선택] 인기 폭발 반반피자"** — 선택권
- **"[트렌디하게] 스페셜 피자"** — 트렌드

## 변경 포인트 (현안 → 가안)

### 1. 모든 카테고리에 수식 브래킷 부착
- [재료를 아끼지 않습니다] / [8년간의 노하우] / [최고급 프리미엄 토핑] / [트렌디하게] / [내맘대로 선택]

### 2. 업력 수치 브래킷
- "[8년간의 노하우] 미국식 클래식 피자" — 연륜 수치

### 3. 피자 메뉴명 접미사 "피자" 통일
- "아메리칸 치즈" → "아메리칸 치즈피자"
- "딥체다치즈 베이컨" → "딥 체다치즈 베이컨피자"
- 메뉴 종류 명확화

### 4. 세트 메뉴에 "세트" / "피자세트" 명시
- "스톰 더블 세트 R" → "스톰 더블피자 세트 R"
- "스톰 사이드 콤보 R" → "스톰 사이드콤보 세트 R"

### 5. 양/토핑 어필 브래킷
- [폭풍토핑] / [토핑폭탄] / [2가지 감자토핑] / [치즈폭발]

### 6. 스파게티 카테고리 헤더 재미
- "사이드" → "피자 옆에 꼭 붙어있는 스파게티"
- "사이드" → "피자와 조합이 좋은 사이드메뉴"
- 피자와의 **조합 유도** 메시지

### 7. 조리법 브래킷
- [직화] / [불향 가득] / [오동통] / [육즙 폭발]

### 8. 매운맛/조합 수식
- [맵.단.짠] — 맛 축 3가지 동시 어필 (점 구분 표기)

### 9. 크러스트 옵션 네이밍
- "치즈크러스트" → "[치즈폭발] 치즈 크러스트"
- "고구마크러스트" → "고구마 크러스트"

## 학습 인사이트 (이 사례에서 추출)

1. **피자 업종 사이즈 2단**: R / XL 표준
2. **카테고리 전체에 [수식 브래킷] 부착**: 가성비·연륜·트렌드·선택권·프리미엄 — 각 카테고리가 하나의 포지셔닝 슬로건
3. **업력 수치**: "[8년간의 노하우]" — 연수 노출 (한끼다 30년, 인생뼈해장국 100억매출과 동일 패턴)
4. **메뉴명 접미사 "피자" 통일**: 피자 업종에서는 모든 메뉴명에 "피자" 접미사
5. **세트 메뉴 명확화**: "세트 R/XL" — 사이즈까지 포함
6. **양 어필 5축**: [폭풍토핑] / [토핑폭탄] / [치즈폭발] / [육즙 폭발] / [2가지 감자토핑]
7. **맛 축 점 구분 표기**: [맵.단.짠] — 3가지 맛 동시 어필
8. **피자와의 조합 유도**: "피자 옆에 꼭 붙어있는 스파게티", "피자와 조합이 좋은 사이드메뉴" — 사이드 카테고리명에서 조합 유도
9. **크러스트 옵션에 수식**: [치즈폭발] 치즈 크러스트 — 업그레이드 옵션에도 브래킷 부착
