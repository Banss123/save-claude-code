## 핵심 정체성 (가게 메시지)
- **"14년 세월이 한그릇에, 소바의온도"** — 업력 14년 + 가게명 호출
- **"깊고 진한 육수, 우동"** — 우동 콘셉트

- **소바 단품**: 10,500~11,000원 (자루/비빔/냉메밀 11,000, 냉소바 10,500)
- **우동 단품**: 10,000~11,000원 (온기 10,000, 김치온기 11,000)
- **돈카츠 단품**: 12,000~14,000원
- **1인 세트(소바/우동+미니카츠)**: 12,900~15,500원

## 변경 포인트

### 1. 단품메뉴 → 소바/우동/돈카츠 3개 콘셉트 헤더로 분할
- 현안의 평면 "단품메뉴" → 가안은 "14년 세월이 한그릇에, 소바의온도" (소바) / "깊고 진한 육수, 우동" (우동) / "고기결을 정확히 살려 썰어낸 돈카츠" (돈카츠)

### 2. 메뉴명 브래킷 키워드
- [황금비율 메밀면] / [비빔국수 스타일] / [오뎅 듬뿍] / [일식 스타일] / [두툼] / [결을 살려 썰어낸] / [일식 튀김덮밥] / [색다른 매력]

### 3. 세트메뉴 네이밍 정돈
- "김치온기우동정식+미니카츠" → "김치 온기우동+미니카츠 1인세트"
- 정식 → 1인세트 (용도 명확)

### 4. 우동 → 메밀소바 명칭 정리
- 비빔소바 → [비빔국수 스타일] 비빔 메밀소바
- 냉소바 → 냉메밀소바

### 5. 리뷰이벤트 통합 (옵션 1)
- 현안은 별도 옵션 그룹 / 가안도 유지 (리뷰이벤트 = 필수 옵션)
- 리뷰이벤트 내 [리뷰] 미니규동 선택 시 +2,000 (리뷰로 미니규동 얻기)

### 6. 카테고리 헤더 전면 카피화
- "미니사이드" → "가볍게 추가하기 좋은 미니사이드"
- "추가메뉴" → "한숟갈 부족할 때, 추가메뉴"

### 7. 1인세트 가격 그대로 유지
- 현안·가안 모두 동일 가격 (네이밍만 정돈)

### 8. 흑돼지 소바국밥 포지셔닝 강화
- 단품 → [색다른 매력] 흑돼지 소바국밥 (한식 카테고리 이동)

## 학습 인사이트

1. **업력을 카테고리 헤더에 명시** — "14년 세월이 한그릇에, 소바의온도" — 가게명 재호출
2. **평면 "단품메뉴" → 장르별 카테고리 분할** — 소바/우동/돈카츠 3개 — 각 카테고리마다 콘셉트 헤더
3. **카피 축**: 제조법([황금비율 메밀면]/[결을 살려 썰어낸]/[비빔국수 스타일]), 상태([오뎅 듬뿍]/[두툼]), 스타일([일식 스타일]/[일식 튀김덮밥]/[한식메뉴])
4. **1인세트 = 단품+미니카츠 조합** — 모든 소바/우동/돈카츠 단품을 세트화 가능 → 세트 단가 업셀
5. **리뷰이벤트 내 유료 선택지** — [리뷰] 미니규동 +2,000 (리뷰 작성 조건으로 저렴하게 추가)
6. **옵션 그룹 최소 운영(4개)** — 리뷰/미니사이드/추가/음료
7. **미니사이드 카테고리 별도** — 4,000~7,000원대 소소한 추가 메뉴 풀 (미니소바/카레/카츠/규동/메밀전병/일식교자)
8. **옵션명 감성 카피** — "미니사이드" → "가볍게 추가하기 좋은 미니사이드" / "추가메뉴" → "한숟갈 부족할 때, 추가메뉴"
