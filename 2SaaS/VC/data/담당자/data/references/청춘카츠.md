## 변경 포인트 (현안 → 가안 전략)

1. **카테고리 헤더 = 내레이션 톤 카피** ("직접 손질한 고기만", "허전한 돈카츠 옆자리엔")
2. **시그니처 재정의** — "청춘 시그니쳐" → **"먹물빵가루"** 키워드로 구체화 (모호→구체)
3. **"( ) 키워드" → "[ ] 브래킷 키워드"** 표기 통일
4. **"도시락 → 1인 도시락" 명시** (혼밥 타겟)
5. **가격 유지** — 먹물빵가루 시그니처 프리미엄은 +500원으로 상징적
6. **옵션 그룹명을 친근하게** — "수제 소스 추가선택 ~", "꼭 선택부탁드립니당~" (~ 물결 추가)
7. **잘못된 메뉴 정리** — "100장 백돈 까스를 먹었습니다 스마일 합니다" 111,111원 같은 오류 제거
8. **1인세트/2인세트의 옵션 중복 배치** (13~15번과 18~20번 중복) — 세트별 전용 옵션 라인 관리

## 학습 인사이트

1. **시그니처 = 구체적 차별화 요소(먹물빵가루)로 단일화** — "시그니처"라는 추상 용어만으론 부족
2. **옵션 그룹 23개 대규모 운영** — 카츠 도시락처럼 변주 많은 업종은 옵션 매트릭스가 크다
3. **1인세트 vs 2인세트 옵션 구조가 완전히 다름** — 세트마다 옵션 체인 전용 관리
4. **노컷팅/컷팅 옵션은 필수 옵션으로 운영** — 배달 특성(눅눅함) 반영
5. **옵션 그룹명도 가게 톤(~물결/~당/~용)** — 카피 일관성
6. **"도시락" 장르 정체성 유지** — 일식 돈카츠인데 "도시락" 프레임이 더 강하게 작용
7. **리뷰 이벤트 옵션(2뷰/2벤트 = 리뷰/이벤트 우회 표기)** — 플랫폼 검열 우회 표기 패턴
