## 가격 구조 (사이즈 4단)
- **소(2인) / 중(3~4인) / 대(5인) / 특대(6~8인)** — 회 카테고리 표준
- 메뉴 1개당 가격 옵션 4개. 우리 `MenuPlan` 스키마는 메뉴 1개당 가격 1개이므로 **4행으로 분할** 또는 스키마 확장 필요 (구조 갭)

### 가격대 정리 (가안 기준)
- 1인 혼술회: 30,000~32,000원 단일
- 활어회 일반: 소 41,000~76,000 / 특대 76,000~156,000원
- 세꼬시: 소 46,000~61,000원

## 변경 포인트 (현안 → 가안)

### 1. 메뉴명 카피 보강
- 모든 회 메뉴에 [콘셉트 키워드] 접두사 추가 (식감/부위/계절/등급)

### 2. 카테고리 헤더에 가게 메시지 삽입
- 그룹별로 가게 정체성/특장점 1줄 카피

### 3. 가격 인하 (전략적)
- 광어회 중(3~4인) 66,000 → **56,000** (현안 대비 -10,000)
- 우럭회 중 71,000 → 66,000 (-5,000)
- 일부 메뉴 가격 인상 (참숭어회 소 46,000 동일, 중 61,000 → **66,000** +5,000)
- → "가성비" 정체성 위해 메인 회 가격 인하 + 일부 보조 메뉴 인상

### 4. 옵션 그룹 통합 운영
- 모든 회 메뉴에 동일 옵션 셋(1,3,4,5,7,6) 적용 → 일관성 + 객단가 부스터

### 5. 진입 메뉴 보존
- 1인 혼술 회(30~32,000원) 카테고리 유지 — 1인 객단가 진입 점

## 학습 인사이트 (이 사례에서 추출)

1. **회 업종은 "사이즈 4단(소/중/대/특대 = 2/3-4/5/6-8인)" 표준** — 인원 기반
2. **메뉴명 [브래킷 키워드]는 5축**: 식감 / 부위 / 계절 / 등급 / 보양
3. **카테고리 헤더 = 가게 정체성 노출 자리** (단순 분류 아님)
4. **옵션 그룹 7개 운영**: 야채무침·매운탕·해산물·매운탕추가·초밥·음료·한그릇 — 회 1개 주문 시 객단가 +20,000~50,000 가능
5. **가격 인하는 전략적** — 메인 가성비 어필 + 보조 메뉴 인상으로 마진 확보
