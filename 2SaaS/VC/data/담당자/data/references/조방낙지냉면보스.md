- 사이드: 1,000(메추리알)~4,000(떡갈비/튀김만두)

## 변경 포인트 (현안 → 가안)

### 1. 상호명 활용 "보스" 반복
- 냉면보스: 메뉴명·카테고리·이벤트까지 "보스" 반복 ("보스 물냉면", "보스냉면 세트")

### 2. 메뉴명에 내용물 명시
- 낙새볶음 → "낙새볶음 (낙지+새우)"
- 낙곱볶음 → "낙곱볶음 (낙지+곱창)"
- 약어 + 풀이 병기

### 3. "1인분/도시락" 표기
- 모든 볶음 단품에 "1인 ~" 접두사 추가, 복합 메뉴는 "도시락 1인분"

### 4. 전골에 "(소)곱창" 부위 명시
- "곱창 전골" → "[한우 대창] (소)곱창 전골"
- 부위/소 구분 명확화

### 5. 고명/수량 명시 수치화
- "튀김만두 4,000원" → "튀김만두 (7P: 4,000원)"
- "떡갈비 4,000원" → "떡갈비 (2P: 4,000원)"

### 6. 반반/반반 세트 라벨링
- "[반/반] 냉면" → "[냉면도 반반] 보스 반반냉면"

### 7. 현안 이벤트 옵션 분해 → 가안 이벤트 통합
- 조방낙지: 이벤트 6종(공기밥/음료/사리 3종 등)을 옵션 2번에 통합

### 8. 배달비 지역별 차등
- 6지역 (일광/대변/연화리/내리/시랑리(빌라쥬드,오시리아)/신평리)
- +3,000 ~ +10,000 범위

## 학습 인사이트 (이 사례에서 추출)

1. **상호명 활용 반복 노출**: 냉면보스 사례 — 메뉴명·카테고리·세트·이벤트에 "보스" 반복 → 브랜드 암기 효과
2. **약어 메뉴 풀이 병기**: "낙새(낙지+새우)", "낙곱(낙지+곱창)" — 배달 앱에서 이해도 상승
3. **비법/연륜 브래킷**: "40년 비법소스", "[한우 대창]" — 차별화 + 프리미엄
4. **수량 단위 명시**: "튀김만두 (7P: 4,000원)" — 양 감각 구체화
5. **지역별 배달비 차등 옵션**: 6단계 지역 + 배달비 3,000~10,000원 — 원거리 주문 조건 명확화
6. **이벤트 서비스 그룹화**: 공기밥/음료/사리 3종을 "[서비스]" 브래킷으로 통일
7. **냉면 맛 라인 5종**: 물/비빔/물비빔/불/마라 + 반반 — 맛 축 세분화
8. **세트 메뉴 조합 카피**: "맛있는 조합만, 보스냉면 세트메뉴" + "[냉만], [냉함], [냉돈], [냉불/냉제]" 약칭 카피 유지
