# 바운스무드 — 메뉴판 가안 학습 노트

## 핵심 정체성 (가게 메시지)
- **브랜드명을 영문 대문자로 카테고리에 그대로 노출** (BOUNCEMOOD, PREMIUM, SIGNATURE, SIDE, DRINKS)
- **"주문즉시 굽는 패티"** — 신선 조리 신뢰도
- **"바운스무드의 특별한 소스"** — 자체 시그니처 소스 라인
- **"수제 / 미국식 / 아메리칸"** 키워드 반복
- **"완벽한 후숙 아보카도"** — 식재료 품질 표현
- 메뉴명에 **장르(미국식)**, **맛(단짠단짠)**, **방식(수제)**, **타겟(건강식)** 다축 브래킷

## 변경 포인트 (현안 → 가안 전략)

1. **카테고리 전체를 영문 대문자로 변경** — 아메리칸 브랜드 아이덴티티 일관
2. **"(X)원 할인" 문구 카테고리 제거** — 노이즈 제거 (할인은 별도 프로모션)
3. **메뉴명 `( )`와 `[ ]` 브래킷 통일** — [ ] 표기
4. **"여심저격 → [건강식]"** 젠더 중립 + 기능성 어필로 전환
5. **"근본인 → [미국식]"** 추상 → 구체 장르
6. **옵션 메뉴명에 감각 키워드 추가** ("주문즉시 굽는" 패티, "짭짤한" 베이컨, "완벽한 후숙" 아보카도)
7. **[ 매콤 야식 ] 키워드로 시간대 타겟** 보강
8. **수제 / 미국식 / 아메리칸** 반복 노출
9. **가격 유지 + 구조 개선** (버거 업종 공통 패턴)
10. **리뷰이벤트 옵션 "RI뷰)"** 검열 우회 표기

## 학습 인사이트

1. **아메리칸 수제버거 매장은 영문 대문자 카테고리** (PREMIUM BURGER, SIDE MENU, DRINKS) — 브랜드 톤 일관 유지 수단
2. **버거 업종 옵션은 23개로 극대** (청춘카츠 23개와 유사)
3. **"세트는 버거 2개 + 사이드 + 음료 선택 체인"** — 버거 업종 표준
4. **프라이즈 가격 계단(오리지널 0 → 이탈리안 +2.7k)** — 업그레이드 설계 패턴
5. **"여심저격" 같은 성별 표현 → [건강식]으로 대체** (젠더 중립 트렌드 반영)
6. **"근본인" 같은 모호한 수식어 → "[미국식]" 구체 장르 명칭** 대체
7. **토핑 옵션에도 감각 형용사** ("주문즉시 굽는 패티", "짭짤한 베이컨") — 옵션명이 판매 카피
8. **소스 라인업 풍부** (하우스/트러플/와사비마요/바질마요/갈릭마요/더티) — 버거 업종 차별화 수단
9. **쉐이크 라인(밀크쉐이크/솔티드 카라멜)** — 버거 매장 음료 프리미엄
10. **커스터마이징 옵션 전용 그룹(11번)** — 토핑 교체 개별 메뉴로 분리
