# 호참치 — 메뉴판 가안 학습 노트

## 핵심 정체성 (가게 메시지)
- **"당일손질, 당일판매 원칙 [프리미엄 호참치]"** — 신선도 최우선
- **"[혼마구로] 참치뱃살 세트"** — 프리미엄 부위명 노출
- **"[노르웨이산] 프리미엄 연어회 세트"** — 원산지 프리미엄
- **"혼자 즐겨요 [1인 참치회세트]"** — 1인 진입 카테고리
- **"든든한 한끼 [호참치 식사메뉴]"** — 식사 수요 흡수

## 카테고리 구조

| 그룹 | 콘셉트 헤더 (가안) | 현안 |
|------|-------------------|------|
| 세트 | 당일손질, 당일판매 원칙 [프리미엄 호참치] | 세트메뉴 |
| 1인세트 | 혼자 즐겨요 [1인 참치회세트] | (신설) |
| 100g 추가 | 오늘은 더 푸짐하게 [회 100g 추가] | 100g 추가 |
| 식사류 | 든든한 한끼 [호참치 식사메뉴] | 식사류 |
| 사이드 | 필수 추가 사이드메뉴 | 사이드 메뉴 |
| 음료 | 시원한 음료메뉴 | 음료류 |
| 주류 | 주류메뉴 (유지) | 주류메뉴 |

→ **세트 4단계 구조**:
  - 기본셋트(혼마/황새치) → 둘다좋아(참치+연어) → 참치는 뱃살(혼마구로) → 연어는 맛있다 → **[신설] 1인 참치회세트**

## 메뉴명 카피 패턴 (현안 → 가안 [브래킷 키워드])

### 세트
| 현안 | 가안 |
|------|------|
| 기본셋트 | **[야식/술안주]** 기본 참치회세트 (눈다랑어+황새치) |
| 둘다좋아 | **[반반]** 참치회+연어회 세트 (눈다랑어+황새치+연어) |
| 참치는 뱃살(혼마구로) | **[혼마구로]** 참치뱃살 세트 (참다랑어 포함) |
| 연어는 맛있다 | **[노르웨이산]** 프리미엄 연어회 세트 |
| — | **[오늘의 혼술]** 1인 참치회 (단품 20k / 세트+어묵우동 30k) — 신설 |

### 식사류
| 현안 | 가안 |
|------|------|
| 참치덮밥 | **[일식 혼밥]** 참치회덮밥(비빔밥) |
| 연어덮밥 | (생)연어회덮밥 |
| 참연 덮밥 | 참치+연어 덮밥 |
| 간장새우 덮밥 | 간장새우장 덮밥 |
| 장어 덮밥 | **[원기회복]** 장어덮밥 |
| 모듬 덮밥 | 모듬덮밥 |
| 참치 초밥 | **[혼술 맞춤]** 참치초밥 |
| 연어 초밥 | 연어초밥 |
| 오미카세 초밥 | 오미카세 초밥 |

### 사이드
| 현안 | 가안 |
|------|------|
| 간장새우장 | 비법소스로 만든 간장새우장 |

→ **브래킷 축**:
  - **시간대/용도** (야식/술안주, 오늘의 혼술, 일식 혼밥, 혼술 맞춤)
  - **부위 프리미엄** (혼마구로 = 참다랑어)
  - **원산지** (노르웨이산)
  - **구성** (반반)
  - **기능** (원기회복)

→ **"참치회" 장르명 명시** — 단순 "세트"에서 "참치회세트"로 본질 노출
→ **재료 포함내역을 메뉴명에 직접 삽입** ("눈다랑어+황새치", "+연어", "+참다랑어 포함")

## 가격 구조 (사이즈 3단)
- **세트: 소/중/대** 3단 인원 기준
  - 기본셋트(2종): 소 29k / 중 39k / 대 49k
  - 둘다좋아(3종): 소 35k / 중 45k / 대 55k
  - 혼마구로(4종): 소 45k / 중 65k / 대 85k (프리미엄)
  - 연어(노르웨이산): 소 27k / 중 37k / 대 47k
  - **1인 세트(신설): 단품 20k / 세트(+어묵우동) 30k**
- 100g 추가: 눈대절 12k / 눈다랑어뱃살 15k / 황새치등살 10k / 황새치뱃살 17k / 참다랑어뱃살 40k / 연어 15k
- 식사류: 12,900~20,000원
- 사이드: 7,900 (5마리) / 14,900 (10마리)
- 음료 1,500원 / 주류 4,000~80,000원 (마루·준마이 등 프리미엄 사케 라인업)

## 옵션 그룹 (가안 5개)

| # | 그룹명 | 핵심 |
|---|--------|------|
| 1 | 오늘은 더 푸짐하게 [회 100g 추가] (선택) | 눈대절(적신) 12k / 눈다랑어 뱃살 15k / 황새치등살 10k / 황새치뱃살 17k / 참다랑어 뱃살 40k / 연어 15k |
| 2 | 든든한 한끼 [호참치 식사메뉴] (선택) | 참치회덮밥 12.9k / 연어덮밥 13.9k / 참치+연어 덮밥 13.9k / 간장새우 덮밥 14.9k / 장어덮밥 15.9k / 모듬덮밥 16.9k / 참치초밥 13.9k / 연어초밥 14.9k / 오미카세 20k |
| 3 | [인기] 비법소스로 만든 간장새우장 (선택) | 소(5마리) 7.9k / 대(10마리) 14.9k |
| 4 | 시원한 음료메뉴 (선택) | 콜라 1.5k / 사이다 1.5k |
| 5 | 리뷰이벤트 (중복 시 랜덤발송) (선택) | 랜덤 초밥 1P / 간장새우장 1P / 콜라 / 사이다 |

### 메뉴별 옵션 매핑
- 세트(기본/둘다/혼마/연어/1인): `1,2,3,4,5` (전체 적용)
- 식사류(덮밥/초밥): `3,4` (일부만)
- 사이드/음료/주류: 옵션 없음

→ **세트 메뉴에만 5개 옵션 풀 세트** (객단가 부스팅)
→ **옵션 2번 = 식사류를 옵션화** (세트 주문 시 덮밥/초밥 추가 쉽게)

## 변경 포인트 (현안 → 가안 전략)

1. **카테고리 헤더에 "당일손질, 당일판매 원칙"** — 신선도 정체성
2. **1인 참치회 카테고리 신설** — 혼술 진입점 (20,000원대 단일)
3. **메뉴명에 부위/원산지 노출** — [혼마구로], [노르웨이산]
4. **메뉴명에 구성 내역 명시** — "(눈다랑어+황새치)", "(참다랑어 포함)" → 실체 노출
5. **식사류를 옵션으로도 제공** — 카테고리 + 옵션 양쪽 등록 (객단가 추가)
6. **사이드/덮밥에도 "[인기]" 표시** — 인기지표 노출
7. **가격은 현안과 동일** — 메뉴명 리브랜딩 + 구조 개선
8. **사이사이즈 3단(소/중/대)은 회 업종 표준** — 인원 기반(명시 없지만 통상적)

## 학습 인사이트

1. **회 업종은 사이즈 3단(소/중/대) 기본** (일부 4단: 신만선횟집 참고)
2. **"참치 vs 연어 vs 혼합" 3축 + 프리미엄 부위(혼마구로)** 구조
3. **1인 메뉴 신설이 가안 작업의 빈번한 패턴** — 거의 모든 매장에서 1인세트 신설
4. **세트 메뉴명에 "(눈다랑어+황새치)" 실체 표기** — 애매함 제거
5. **[원산지] 브래킷은 회/연어 업종 필수** — 노르웨이산 연어 등
6. **옵션 그룹 5개는 회 업종 적정** — 100g 추가 + 식사류 + 간장새우장 + 음료 + 리뷰
7. **식사류를 카테고리 + 옵션 양쪽 배치** — 세트 주문 시 추가 구매 유도 (전통 업장 기법)
8. **주류 라인업 풍부 (사케 35k~80k, 화요 30k~50k)** — 횟집의 주류 객단가 의존
