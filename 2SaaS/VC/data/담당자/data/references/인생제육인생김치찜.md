# 인생제육 & 인생김치찜 — 메뉴판 가안 학습 노트

## 핵심 정체성 (가게 메시지)
- **"매일 먹어도 안질리는 인생제육"** — 브랜드명 = 콘셉트("인생"=최고)
- **"호불호 없는 '인생김치찜' 1인분 세트"** — 보편성 강조
- **"감칠맛도 2배, 인생김치찜 2인분 세트"** — 양 강조
- **"국내산 돼지고기"** 반복 (원산지)
- **1인용/2인용 분리 전략** — 혼밥 + 커플/둘이 먹기 각각 카테고리
- 시트 2개 병렬 운영 (인생김치찜 + 인생제육)

## 가격 구조
- 1인세트: 11,900~15,900원
- 2배(2인)세트: 24,000~29,000원 (2배 양에 2배+α 가격)
- 고기만: 2인 19,000 / 3인 26,000
- 세트(제육+김치찜): 단품 24,000 / 2~3인 30,000 / 제육+찌개 22~28k
- 사이드: 500~5,000원
- **가격은 현안과 동일** (메뉴명 리브랜딩만)

## 옵션 그룹 (가안 10개, 인생김치찜 기준)

| # | 그룹명 | 핵심 |
|---|--------|------|
| 1 | 맵기 선택(찌개) 필수 | 살짝매콤/중간매콤/매우매콤 |
| 2 | 후기를 들려주세용% | 고기 100g +500 / 두부 +300 / 랜덤 뚱깬 / 계란후라이 / 파김치 / 공기밥 |
| 3 | 시원한 음료선택 | 펩시/제로/사이다/환타/식혜 1,500 |
| 4 | 직화제육 추가로 한식세트 완성~ | 살짝/중간/매우매콤/단짠갈비/새콤매콤김치 제육 12,000 |
| 5 | 김치찜 토핑추가 | 돼지고기 100g / 우삼겹 100g / 스팸 / 파김치 / 김치 / 두부 / 순두부 / 밀떡사리 |
| 6 | 제육 토핑추가 | 고기 100g~200g / 마늘쫑 / 양배추 / 두부 / 밀떡사리 |
| 7 | 김치찜 종류선택 필수 | 인생 김치찜 / 우삼겹 / 스팸 / 고등어(+1,000) |
| 8 | 찜 토핑 추가 (중복) | 5번과 유사 |
| 9 | 김치찜 필수추가 사이드 | 버터간장계란밥 4k / 두부김치 5k / 계란후라이 / 계란찜 / 파김치 / 볶음김치 / 쌈무 / 김 / 미소장국 |
| 10 | 밥 추가 | 공기밥 1~3개 1k/2k/3k |

### 메뉴별 옵션 매핑
- 김치찜 1인: `1,10,8,4,9,3,2`
- 김치찜 2배: `1,10,8,4,9,3,2`
- 세트(제육+김치찜): `7,1,10,8,6,9,3,2`
- 사이드는 옵션 없음

## 변경 포인트 (현안 → 가안 전략)

1. **카테고리에 [브래킷 카피] 헤더** — 제목 자체가 광고
2. **"혼밥세트" → "1인세트" 통일** — 용어 표준화
3. **( ) → [ ] 브래킷 표기 통일** — 시인성 상승
4. **원산지(국내산 돼지고기) 메뉴명 삽입** — 신뢰도
5. **신메뉴 [국물맵떡 스타일] 떡볶이제육** — 트렌드 흡수
6. **사이드에도 [브래킷] 부여** — [매콤함에 딱] 버터간장계란밥, [국민 술안주] 두부김치 → 사이드도 콘셉트 카피
7. **옵션 그룹명에도 감정 톤** — "후기를 들려주세용%", "시원한 음료선택" 등
8. **옵션 번호 확장** — 현안 8개 → 가안 10개 (밥 추가 별도 분리)

## 학습 인사이트

1. **"인생"이라는 단어를 브랜드명 + 메뉴명 + 카테고리 전체에 반복** — 브랜드 일관성 극대화
2. **1인/2인 카테고리 분리** — "혼밥"에서 "1인세트"로 네이밍 정비 (혼밥의 부정 뉘앙스 제거)
3. **메뉴 종류가 많아도 옵션 그룹으로 체계화** — 10개 옵션 그룹, 메뉴별 3~7개씩 조합
4. **맛 카테고리(단짠/매콤/고소)와 용도 카테고리(야식/혼밥/가성비)를 동시 활용**
5. **사이드까지 콘셉트 카피 부여** — 기본 사이드를 "필수 추가"로 격상
6. **토핑 추가 옵션 2회 중복 배치** (5번 + 8번) — 찜 메뉴는 찜 토핑 전용 옵션 따로
7. **제육과 김치찜의 크로스 세트** — "환상궁합 제육+찌개 세트"로 교차 판매
