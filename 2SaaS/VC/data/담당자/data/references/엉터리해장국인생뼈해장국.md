## 변경 포인트 (현안 → 가안)

### 1. 매출 실적을 카테고리 헤더에 유지/강화
- **[전국 100억 매출]** — 실적 숫자 + 브랜드 카테고리 헤더에 그대로 노출

### 2. 산지 공급 정보 브래킷
- **[양평 직접공수 소고기]**, **[해남] 우거지**, **[제주산]** 등 산지 명시
- 신뢰도 + 차별화

### 3. "특/왕/특대" 수량 표기 표준화
- "[1.5 특]" → "1.5인분" (분수 표기로 통일)
- "[특]" / "[특대]" — 사이즈 표기 일관화

### 4. 상시 할인/무료배달 표기 정리
- 현안: "(무료 배달)" 직접 노출
- 가안: 메뉴명에서 제거 (카테고리 또는 배너로 이동)

### 5. "사장님이 미쳤어요" 브랜딩 옵션
- 리뷰이벤트 + 라면/당면사리 무료 제공 옵션 그룹을 **"X제발 서비스 그만주세요X"** 문구로 의인화
- 엉뚱 유머 브랜딩

### 6. O/X 체크 패턴
- 들깨가루, 고추기름, 와사비소스 모두 O/X 필수 선택으로 통일
- "X 빼주세요 X" / "O 넣어주세요O" 형식 — 명확한 주문 의사전달

### 7. 카테고리 헤더 맥락화
- "사이드 메뉴" → "완벽한 한끼를 위해, 필수 사이드메뉴"
- "엉터리 감자탕" → "남녀노소 모두 좋아하는 엉터리 감자탕"

## 학습 인사이트 (이 사례에서 추출)

1. **매출 실적 숫자를 카테고리 헤더에 노출** — [전국 100억 매출] 같은 수치 배지는 가안에서도 그대로 유지
2. **산지 브래킷 = 신뢰 자산**: [양평], [해남] — 지역명 산지 공개
3. **수량 표기 표준화**: "특/왕/특대" → 1인분/1.5인분 분수 표기로 통일
4. **브랜딩 옵션 그룹**: "사장님이 미쳤어요" 같은 유머 문구로 리뷰이벤트/무료서비스 옵션을 통합
5. **O/X 필수 옵션**: 빠짐없이 "X 빼주세요 X" / "O 넣어주세요 O" 포맷 — 주문 실수 방지
6. **TPO 브래킷**: [든든하게 혼밥], [마성의 야식] — 시간/상황 기반 카피
7. **포지셔닝 브래킷**: [백반 스타일], [한식밥상], [원조 한식] — 메뉴를 특정 이미지로 포지셔닝
8. **제조/수량 브래킷**: [수제], [맛보기], [특 등급 쌀] — 품질·단위 정보 노출
