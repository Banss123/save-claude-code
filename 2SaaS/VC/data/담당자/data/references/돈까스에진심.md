## 변경 포인트

### 1. 메뉴명 [NO밀가루소스]/[육즙이츄릅]/[부드러움끝판왕] → 가안의 [육즙 가득]/[자연치즈 듬뿍]/[100% 한돈]
- 과도한 감탄형 브래킷을 차분한 카피로 교체
- 원산지·품질 키워드 강화

### 2. 메뉴명 `돈카츠` → `돈까스` 통일
- 현안은 돈카츠/돈까스 혼용 → 가안은 "돈까스"로 일관

### 3. 세트 네이밍
- "카츠세트 + 우동" → "돈까스 + 우동세트" (어순 정돈)
- "카츠세트 + 떡볶이" → "[분식집 감성] 돈까스 + 떡볶이세트"

### 4. 반반돈까스 조합 명시
- "등심카츠 + 안심카츠" → **[등심+안심]** 반반돈까스
- 브래킷으로 조합 즉시 인지 가능

### 5. 카테고리 헤더에 정체성 카피 삽입
- "좋은 고기만 씁니다, '돈까스에 진심'" — 가게명 재호출
- "가성비 훌륭한 '1.5인 세트'" / "완벽한 한끼를 위한 '반반돈까스'" / "돈까스와 카레의 환상궁합"

### 6. 옵션 합산/재조정
- 소스·반찬·공기밥을 한 그룹(4번 "다채로운 맛, 소스 추가")으로 통합
- 기존 옵션 그룹 9개 → 가안 10개로 재조합

### 7. 메뉴 "김치나베" 제외
- 현안에 있던 `[얼큰] 김치나베돈까스 13,900`이 가안에서 누락 (변경/통합 가능성)

### 8. 냉쫄면/메밀소바 → 콩국수로 단순화
- 1.5인 세트 구성 축소

## 학습 인사이트

1. **가게명을 카테고리 헤더에 직접 호출** ("좋은 고기만 씁니다, '돈까스에 진심'") — 브랜드 리마인더
2. **반반돈까스 = 옵션 아닌 별도 메뉴** — [재료1+재료2] 브래킷으로 조합 즉시 표현
3. **돈까스 업종의 필수 옵션**: 후라이 완숙/반숙, 컷팅 O/X — 일식 돈까스 맞춤 품질 옵션
4. **돈까스 카피 축**: 원산지(100% 한돈), 용도(일식 혼밥/매콤 야식/분식집 감성), 재료(자연치즈 듬뿍/등심+안심), 식감(육즙 가득/바삭바삭/겉바속촉/느끼함 잡아주는)
5. **1.5인 세트** — "1인+α" 객단가 업셀 전략 (돈까스+면/분식)
6. **소스·반찬 통합 옵션** — "다채로운 맛" 네이밍으로 업셀 친화
7. **떡볶이/우동 맵기 선택 필수 옵션** — 매운맛 소비자 분기 처리
8. **과장형 브래킷([육즙이츄릅]/[부드러움끝판왕])은 가안에서 차분하게 정돈** — 배달앱 가독성/신뢰 우선
