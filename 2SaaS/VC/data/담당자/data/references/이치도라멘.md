# 이치도라멘 — 메뉴판 가안 학습 노트

## 핵심 정체성 (가게 메시지)
- **"이치도 정통 일본식라멘"** — 정통성/일식 프레임
- **"깔끔한 일본식 소바"**, **"빠질수없는 일본풍 사이드"** — 일본 로컬리티 일관
- **"맛있는 조합으로만, 세트메뉴"** — 세트 번들링 명시
- **"한국인 입맛으로 재해석한 규동&카레"** — 로컬라이제이션 포인트

## 가격 구조 (변동)
- 대부분 현안 = 가안 동일
- **한정 세트(14,500) 제거** → 기본 돈코츠 11,000 시작
- **신규 1인 조합세트 4종 추가** (돈코츠+가라아게 14k / +교자 13.5k / 매운+가라아게 15k / 매운+교자 14.5k)
- → **단일 신메뉴 대신 조합형 세트 신설**로 객단가 상승 (주 기법)

## 옵션 그룹 (가안 7개)

| # | 그룹명 | 핵심 항목 |
|---|--------|-----------|
| 1 | 리뷰이벤트 | 후렌치 후라이/멘치카츠/교자만두/음료 랜덤/가라아게 (중복 시 1개만) |
| 2 | 카레 맵기 선택 (필수) | 기본/약간매콤/매운맛 |
| 3 | 규동 추가 (선택) | 밥 1,500 / 차슈 2,000 / 아지타마고 2,000 |
| 4 | 마제소바 추가 (선택) | 마제소바 밥/식초/차슈/아지타마고 |
| 5 | 옵션 추가 (선택) | 차슈/아지타마고/밥/돈코츠 면 추가 |
| 6 | 빠질수없는 일본풍 사이드 (선택) | 고로케 5k/교자 5k/새우튀김 5k/멘치 4k/가라아게 6k/돈까스 6k/하프박스 11k |
| 7 | 시원한 음료 추가 (선택) | 코카콜라/스프라이트/제로콜라 2,000 |

### 메뉴별 옵션 매핑
- 라멘: `1,5,6,7`
- 소바: `1,4,6,7`
- 규동/카레: `1,3,6,7` or `2,1,3,6,7`
- → **리뷰(1) + 장르별 추가(3/4/5) + 사이드(6) + 음료(7)** 4축 조합

## 변경 포인트 (현안 → 가안 전략)

1. **단품 중심에서 세트 중심으로** — 1인 조합세트 4종 신설 (라멘+사이드 번들)
2. **카테고리 재편** — 메인메뉴 단일 뭉치 → 라멘/세트/소바/규동카레 4개
3. **카테고리에 가게 메시지** — "이치도 정통 일본식라멘" 등 브랜드명+정체성 노출
4. **메뉴명에 "1인분/1인세트" 명시** — 혼밥 수요 흡수
5. **사이드메뉴 카테고리 옵션화** — 단품 진열과 옵션 중복 배치로 객단가 부스터
6. **정통성 키워드 반복** — [정통 일본식], 나고야식, 일본식 등 일관
7. **일부 가격 유지 + 한정 메뉴 정리** — 구조 정리

## 학습 인사이트

1. **"단품 → 조합세트 신설"이 객단가 핵심 전략** (가격 손대지 않고 상위 세트 유도)
2. **카테고리는 "정통 + 지역(나고야식)" 프레임으로 일식 아이덴티티 강화**
3. **리뷰 이벤트를 옵션 1번에 배치** (주문 즉시 노출)
4. **맵기 선택은 필수 옵션(카레 전용)** — 맵기 선호 파편화 흡수
5. **같은 사이드가 카테고리 메뉴 + 옵션 양쪽에 배치됨** — 단품/추가 모두 수익화
6. **[재해석한]** 키워드로 "일본식인데 한국인이 맛있게" 양립
