## 변경 포인트 (현안 → 가안)

### 1. 상호명 → 카테고리 헤더에 스토리
- "일품요리" → **"인생은 혼자서 먹는 한끼다 [1인 세트]"** — 상호와 메시지 일치

### 2. "30년간의 노하우" 연륜 어필
- 도시락 카테고리에 연륜 브래킷

### 3. 도시락 → "고기도시락/1인분" 표기
- 현안 "한끼 제육 도시락" → 가안 "[한식의 정석] 직화 제육볶음 고기도시락(1인분)"
- "1인분" 명시 + 포지셔닝 브래킷

### 4. 1인 세트에 "혼밥세트/한정식세트/1인세트" 혼합 라벨링
- 김치찜 = 혼밥세트 / 쭈꾸미 = 한정식세트 / 제육·목살 = 1인세트
- 한식 계열이면 한정식 라벨

### 5. 맵기 의성어 표기
- "쓰읍 / 쓰으읍 / 쓰으으으으읍" — 언어적 차별화 + 친근감

### 6. 리뷰이벤트 "100원의 행복" 브랜딩
- 100원 추가로 소형 사이드/음료 제공 — 리뷰이벤트를 "유료처럼 보이되 체감 가치" 연출

### 7. 전 메뉴 카피
- 모든 전 메뉴에 식감/향 수식어 ([살살녹는], 오동통, 솔솔, [술안주로 제격])

### 8. 김치찜 토핑 그램 단위
- 삼겹/목살 × (120g/250g) + 스팸/동원참치 추가 — 6종 그램 토핑

## 학습 인사이트 (이 사례에서 추출)

1. **상호와 카테고리 헤더 일치**: "인생은 한끼다" → "인생은 혼자서 먹는 한끼다 [1인 세트]" — 상호명의 의미를 카테고리 헤더에서 완성
2. **연륜 브래킷**: "30년간의 노하우" — 업력 수치를 카테고리 헤더에 직접 노출
3. **"(1인분)" 표기 정책**: 도시락/세트 메뉴에 "(1인분)" 괄호 표기로 1인 객단가 명확화
4. **1인 세트 라벨링 다각화**: 혼밥세트 / 한정식세트 / 1인세트 — 메뉴 성격별로 라벨 분리
5. **맵기 의성어 표기**: "쓰읍 / 쓰으읍 / 쓰으으으으읍 [3단계 매움 주의]" — 시각+청각 재미
6. **"100원의 행복" 리뷰이벤트**: 100원이라는 상징적 가격으로 유료 옵션 + 리뷰 유도 동시 달성
7. **전 메뉴 시청각 수식어**: 오동통(시각), 솔솔(후각), 살살녹는(식감), 바삭바삭(청각) — 감각 단어 활용
8. **김치찜 그램 단위 고기 토핑**: 삼겹/목살 × 120g/250g 2단 + 스팸/동원참치 — 객단가 부스터
