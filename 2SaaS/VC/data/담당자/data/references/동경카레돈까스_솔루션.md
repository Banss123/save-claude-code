## 학습 인사이트 (이 사례에서 추출)

1. **CPC 단가 381원** — 돈까스·카레 업종 중간 가격대
2. **CPC 월 한도 180만원** — 중형 광고 예산
3. **쿠팡 CMG 9%** — 표준 수수료율 (제한 없음)
4. **즉시할인 3,000원 + 최소주문 10,900원** 조합 — 진입 장벽 낮추고 할인 강조
5. **한그릇/하나만 라인업 분리**: 1인 세트(덮밥) vs 단품(돈까스) 구분 전략
6. **자매점 설정 재활용**: "동양카츠와 동일히 설정" — 같은 운영사 내 설정 복사 패턴
7. **상권 분석 섹션 없음** — 일부 케이스는 3장 구조 (섹션 2 생략)
8. **CPC 할인·배달팁 항목 생략** — 업장별로 해당 항목만 선별 기재
