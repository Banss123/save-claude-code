# 이세계반점 & 훈연실 — 메뉴판 가안 학습 노트

## 핵심 정체성 (가게 메시지)
- **이세계반점**: "차원을 넘어온 색다른 퓨전중화요리" — 서브컬쳐/판타지 콘셉트
- **훈연실**: "국내산 참나무로 훈연한 바베큐" — 훈제/원산지 강조
- 브랜드별 시트 분리 운영 (같은 사업장의 듀얼 콘셉트)
- 공통 옵션 운영으로 교차 판매 (훈연실에서 이세계 탕수육 추가 가능)

## 카테고리 구조 (그룹별 콘셉트 헤더 패턴)

### 이세계반점
| 그룹 | 콘셉트 헤더 | 비고 |
|------|-------------|------|
| 세트 | 이세계 소녀의 추천메뉴 | 캐릭터 화자 |
| 퓨전중화 | "차원을 넘어온 색다른 '퓨전중화요리'" | 세계관 |
| 중화요리 | [소환 필수] 이세계 탕수육추가 | 판타지 키워드 |
| 사이드 | 이야기의 끝, 곁들임 메뉴 | 스토리텔링 |
| 음료 | 시원한 음료추가 | 기능 |

### 훈연실
| 그룹 | 콘셉트 헤더 |
|------|-------------|
| 바베큐 | 국내산 참나무로 훈연한 바베큐 |
| 파스타 | 양식당 스타일 파스타 & 리조토 |
| 샐러드 | 신선한 샐러드 |
| 퓨전중화 | 훈연실식 퓨전 중화요리 |
| 중화요리 | 클래식 중화요리 |
| 사이드 | 바베큐와 함께하는 사이드 |

→ **세계관 언어 + 원산지/조리법** 두 축을 카테고리에 직접 노출

## 메뉴명 카피 패턴 (현안 → 가안 [브래킷 키워드])

### 이세계반점 (판타지·게임 콘셉트)
| 현안 | 가안 |
|------|------|
| 소녀 픽 세트 | **[소녀픽 혼밥]** 짜장면 1인분세트 |
| 이세계 시그니처 돈마호크 짬뽕 | **[이세계 중국집]** 돈마호크 짬뽕 |
| HP 풀회복 삼선 짬뽕 | **[HP 풀회복]** 삼선짬뽕 |
| 이세계 풀드포크 짜장면 | 이세계 풀드포크 1인 짜장면 |
| 차돌 짬뽕 | **[맛있는 야식]** 차돌짬뽕 |

→ **브래킷 축**: 타겟(소녀픽/혼밥), 업종재규정(이세계 중국집), 게임 스테이터스(HP 풀회복), 시간대(야식)

### 훈연실 (프리미엄·야식·1인 맞춤)
| 현안 | 가안 |
|------|------|
| 포크플래터+풀드포크.치킨 | **[고급 양식]** 바베큐 포크플래터+풀드포크+치킨 |
| 해물마라볶음 파스타 | **[매콤 야식]** 해물 마라 볶음파스타 |
| 쉬림프투움바 파스타 | 쉬림프 투움바 **1인** 파스타 |
| 꺳잎차돌크림 숏파스타 | 깻잎차돌 **1인** 크림파스타 |
| 훈제연어샐러드 | **[촉촉]** 훈제 연어샐러드 |
| 부라타치즈샐러드 | **[고소한]** 부라타 치즈 샐러드 |
| 차돌 짬뽕 | **[1인 중식]** 차돌짬뽕 |
| 시그니쳐 돈마호크 짬뽕 | **[고기듬뿍]** 시그니처 돈마호크 짬뽕 |
| 브리스킷 볶음짜장 | **[고기폭탄]** 브리스킷 볶음 짜장면 |

→ **브래킷 축**: 등급(고급 양식), 시간대(야식), 인분(1인), 식감(촉촉/고소한), 양/재료(고기듬뿍/고기폭탄)

## 가격 구조
- 이세계반점: 9,000~28,000원 (1인 9~14k, 메인 18~24k, 탕수육 28k)
- 훈연실: 14,000~350,000원 (파스타 17~20k, 바베큐 39~350k — 사이즈 풍부)
- 가격은 현안과 가안 동일 (메뉴명만 리프레쉬)

## 옵션 그룹 (가안 기준)

### 이세계반점 4그룹
1. **시원한 음료추가** — 펩시/제로/사이다/닥터페퍼 2,000원
2. **이야기의 끝, 곁들임 메뉴** — 짜장소스/짬뽕국물 3,000 / 군만두 6,000
3. **[소환 필수] 이세계 탕수육추가** — 이세계 달콤/픽오브 탕수육 28,000
4. **리뷰 이벤트(중복 랜덤)** — 군만두 2P/볶음짜장소스/짬뽕국물/공기밥/랜덤음료

### 훈연실 4그룹
1. 시원한 음료추가 (2,000)
2. **바베큐와 함께하는 사이드** — 디너롤 2k/또띠아 1k/야채구이 5k/훈연실 감바스 22k/모찌 리브륄레 8k/바게트 3k/코울슬로 2k/어니언링 6k/콘립 8k
3. **클래식 중화요리 추가** — 이세계 탕수육 28,000 (크로스 브랜드 옵션!)
4. 리뷰 이벤트

### 메뉴별 옵션 매핑
- 이세계 메인: `3,2,1,4` (탕수육→사이드→음료→리뷰)
- 이세계 탕수육: `2,1,4`
- 훈연실 바베큐: `3,2,1,4`
- 훈연실 파스타/샐러드: `1,4` (경량 조합)
- 훈연실 중화: `3,1,4`

## 변경 포인트 (현안 → 가안 전략)

1. **카테고리에 가게 메시지 주입** — 단순 분류("세트")를 "이세계 소녀의 추천메뉴"로
2. **세계관 언어 보강** — 이세계반점은 판타지/게임 용어 ([소환 필수], [HP 풀회복])로 재밌는 톤
3. **훈연실은 프리미엄 + 1인 맞춤** — "[고급 양식]", "1인" 명시로 가격 정당화 + 혼밥 타겟
4. **시간대 타겟** — [매콤 야식], [1인 중식], [맛있는 야식] 등 시간대 카피
5. **양 강조** — [고기듬뿍], [고기폭탄]으로 물리적 만족감
6. **크로스 브랜드 옵션** — 훈연실에서 이세계 탕수육 추가 가능 (브랜드 시너지)
7. **가격은 현안 유지** — 메뉴명 리브랜딩만으로 인지 가치 상승 의도

## 학습 인사이트

1. **1사업장 2브랜드 병행 운영** — 각 시트 독립 메뉴판, 옵션은 상호 참조 가능
2. **판타지/게임 IP 톤은 카테고리 레벨에서도 일관되게 노출** (카테고리 헤더 = 세계관)
3. **[브래킷] 사용 패턴이 브랜드별로 다름**
   - 이세계: 세계관/캐릭터/게임 용어
   - 훈연실: 등급/재료/시간대/인분
4. **1인 메뉴 강조** — 파스타·리조또에 "1인" 명시로 혼밥 수요 흡수
5. **세트 번들 우선** — 각 브랜드 맨 첫 카테고리는 세트/추천 구성
6. **훈연실 바베큐 가격 스펙트럼 광대(39k~350k)** — 1인~6인 가격대별 진입
