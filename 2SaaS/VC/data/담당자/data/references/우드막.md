## 변경 포인트 (현안 → 가안)

### 1. 브랜드 [우드막] 접두사 반복
- 카테고리마다 "[우드막 전골요리]", "[우드막 우삼겹 야채전골]", "[우드막 덮밥 시리즈]" — 브랜드 자산화

### 2. 육수 자가 제조 문구
- "매일 직접 우려내는 육수" / "깊은 맛의 육수" — 신뢰 메시지

### 3. 1인 전골을 "세트"로 포지셔닝
- "1인 인기 전골 (밥 포함)" → "[혼밥/혼술] 우드막 전골 1인분세트"

### 4. 전골에 "버섯" 추가 어필
- "한우(모츠나베) 대창전골" → "한우(모츠나베) 대창 버섯전골"
- 야채 포함 뉘앙스 강화

### 5. 맛 특성 브래킷
- "칼칼한 →[칼칼한 야식]", "담백한 →[담백한 한식밥상]"
- 현안은 맛 수식어를 메뉴명 앞에, 가안은 **대괄호 브래킷 + 포지셔닝 통합**

### 6. 식감/조리 브래킷 추가
- "[겉바속촉]" 가라아게, "[육즙 가득]" 떡갈비
- "스노우 옥수수알" — 시각적 수식어

### 7. 비조리 옵션 상세화
- "비조리" → "비조리 (일회용 직화냄비 제공)" — 배달 컨디션 명확화

### 8. 옵션 그룹 동일 구성 이동 운영
- 1번 곁들임 사이드 / 2번 토핑 — 전골 전 메뉴에 동일 적용

## 학습 인사이트 (이 사례에서 추출)

1. **전골 업종 2단 사이즈**: 중(2~3인) / 대(3~4인) 표준
2. **브랜드 접두사 반복**: [우드막 전골요리], [우드막 덮밥 시리즈] — 카테고리마다 브랜드 노출
3. **육수 자가 제조 어필**: "매일 직접 우려내는 육수" — 차별화 메시지를 카테고리 헤더에
4. **전골 종류 옵션화**: 1인 전골에서는 4종(고추장/된장/간장/마라) + 3종 모츠나베 업그레이드를 옵션 그룹 3으로 통합 → 단일 메뉴로 압축
5. **비조리 직화냄비 제공**: 배달 시 조리 경험 전달 = "일회용 직화냄비 제공" 문구 명시
6. **식감 브래킷**: [겉바속촉], [육즙 가득] — 튀김/육류 메뉴 표준 키워드
7. **맛 비유 브래킷**: [마라탕 맛] — "이 맛이 친숙한 그 맛" 연상 유도
8. **"야채/버섯" 추가 어필**: 전골 메뉴에 "버섯" 추가로 야채감 증가
