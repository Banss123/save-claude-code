## 핵심 정체성 (가게 메시지)
- **"내 맘대로 골라먹는 '도홍' 세트메뉴"** — 세트 자유도 카피
- **"도홍 프리미엄 중식 면요리"** — 프리미엄 포지셔닝
- **"국내산 등심/국내산 최고급 등심"** — 원재료 프리미엄 신뢰
- **"겉바속촉" / "중국집 인기메뉴" / "중국집에 빠질 수 없는, 스페셜요리 라인업"** — 중식 정체성 + 식감/위상

## 가격 구조
- **세트메뉴**: 22,000~28,000원 (1~2인)
- **면류(면 단품)**: 8,000~15,000원 (등심짜장 8,000 / 차돌짬뽕 15,000)
- **식사류**: 10,000~15,000원
- **요리류 단품**: 14,000원(1인)~36,000원(대)
- **사이즈 3단(요리류)**: 소/중/대 (예: 탕수육 22/28/34 → 가안도 동일)
- **사이드**: 2,000~5,000원
- **음료**: 355ml 2,000원

## 변경 포인트

### 1. 카테고리 헤더 `[]면류[]` 장식 → 카피형 헤더로
- `[]면류[]` → "도홍 프리미엄 중식 면요리"
- `[]식사류[]` → "가성비 가득, 프리미엄 식사메뉴"
- `[]요리류[]` → "중국집에 빠질 수 없는, 스페셜요리 라인업"
- `[]사장님 .강.력.추.천/ 세트 메뉴[]` → "내 맘대로 골라먹는 '도홍' 세트메뉴"

### 2. 메뉴명에 "중식당 시그니처/중국집 인기메뉴/광둥식/별미" 등 위상 키워드
- 마파두부덥밥 → **[중식당 시그니처]** 마파두부덥밥 1인분
- 잡채밥 → **[중국집 인기메뉴]** 잡채밥 1인분
- 등심탕수육 → **[광둥식]** 1인 미니 등심탕수육

### 3. 용도/콘셉트 브래킷
- 혼밥 맛집 / 무료배달 / 매운 야식(중독성 강한 야식)

### 4. 상태 카피
- 뽀얀 국물, 고기 듬뿍, 고기폭탄, 바삭바삭, 겉바속촉(그대로 유지)

### 5. 메뉴 재조합
- 차돌짬뽕 → "차돌+소고기 짬뽕" / 차돌짬뽕(차돌+해물) → "차돌+해물 짬뽕" — 원재료 명시 분화

### 6. 옵션 곱빼기 변경 체계
- 메뉴 × 곱빼기 조합을 옵션화 → 짜장/짬뽕/볶음밥 1개/2개, 매운짜장, 교차 조합(짜장곱+짬뽕곱) 다수 세분 운영

### 7. 음료 브랜드 명시화
- 콜라 → 코카콜라 355ml / 사이다 → 칠성사이다 355ml / 환타 → 환타 파인 355ml / 제로콜라 → 제로콜라 355ml

### 8. 2인 세트 식사 선택 옵션 통합
- 2가지 식사를 1개 옵션에서 조합(예: 등심짜장+새우볶음밥, 해물짬뽕+볶음밥) — 세트 구성 간소화

## 학습 인사이트

1. **중식 업종 카테고리 헤더 = 가게 메시지 노출 자리** — `[]카테고리[]` 장식을 카피로 전환
2. **메뉴명 브래킷 축**: 원산지([국내산 최고급 등심]), 용도([혼밥 맛집]/[무료배달]/[매운 야식]/[매콤]), 지위([중식당 시그니처]/[중국집 인기메뉴]/[대표메뉴]→[별미]/[광둥식]), 상태([겉바속촉]/[뽀얀 국물]/[고기 듬뿍]/[고기폭탄]/[바삭바삭]/[중독성 강한])
3. **음료는 반드시 브랜드명 + 용량 명시** (플랫폼 정확성)
4. **곱빼기 옵션 세분화** — 메뉴별·2개 조합 모두 별도 옵션 항목화
5. **세트메뉴 식사 선택 옵션(16번)이 단일 옵션에 28종 나열** — 자유도 높은 세트 운영의 전형
6. **요리류는 사이즈 3단(소/중/대)** — 면/식사/세트는 1인 단위
7. **"한 끼를 완성하는 필수 추가메뉴"** = 사이드 업셀 전략
8. **"사장님 강력 추천" 표기는 가안에서 제거** — 카피 정돈
