## 변경 포인트 (현안 → 가안)

### 1. 카테고리 2분할 (현안 통합 → 가안 함박/돈까스 분리)
- 함박 = "격이 다른 부드러움, 코끼리함박"
- 돈까스 = "국내산 최고급등심, 코끼리돈까스"

### 2. 함박 메뉴명에 "스테이크" 표기 추가
- "로제함박" → "로제 함박스테이크"
- "파스타함박" → "파스타 함박스테이크"
- 양식 정체성 강화

### 3. 포지셔닝 브래킷 추가
- [시그니처] / [가성비 혼밥] / [옛날 감성] / [양식당 느낌] / [매콤 야식]

### 4. 쫄면 역할 카피
- "쫄면" → "[느끼함 덜어주는/잡아주는] 쫄면"
- 메인(함박/돈까스)의 기름진 맛 대비 역할 제시 → 주문 유도

### 5. 옵션 일원화
- 모든 메인 메뉴에 옵션 1, 2, 4 동일 적용
- 파스타함박만 옵션 3(땡초 빼주세요) 추가

### 6. 리뷰이벤트 브랜딩
- "리뷰이벤트 (중복 시 랜덤발송)" — 중복 선택 방지 안내 포함

## 학습 인사이트 (이 사례에서 추출)

1. **함박/돈까스 카테고리 분리**: 한 가게에서 두 업종을 혼합 판매할 때 카테고리를 분리 + 각 카테고리별 정체성 헤더
2. **메뉴명 "스테이크" 접미사 정책**: 함박은 "함박스테이크"로 통일 → 경양식 정체성 강화
3. **쫄면 = 느끼함 해소 옵션**: 한식/양식 기름진 메뉴에 쫄면 옵션을 "[느끼함 덜어주는]" 역할 카피로 주문 유도 (재구매율 상승 기대)
4. **포지셔닝 브래킷 5축**: [시그니처] / [가성비 혼밥] / [옛날 감성] / [양식당 느낌] / [매콤 야식] — 경양식 업종 키워드
5. **옵션 4종 단순 구조**: 사이드/음료/리뷰/특정 메뉴 요청 — 경양식 업종의 간결한 옵션 설계
6. **브랜드명 반복**: "코끼리함박", "코끼리돈까스", "코끼리왕돈까스", "코끼리 수제 함박" — 메뉴명에 브랜드 반복 노출
7. **리뷰이벤트 중복 방지 고지**: "중복 시 랜덤발송" — 부정 주문 방지 안내 문구
