## 가격 구조
- 1인 도시락: 10,900(쭈꾸미·돼지불백) / 11,900(낙지)
- 1인 덮밥/비빔밥: 9,500~10,500
- 메인 메뉴 3단: 2인분 22,000~25,000 / 3인분 32,000~35,000 / 4인분 42,000~45,000
- 수육&무침회 2단: 소 15,000 / 중 25,000
- 사이드: 5,000~18,000

### 현안에만 있던 제철 추가메뉴 (가안에서 제거)
- 과메기 5미 +15,000 / 과메기 10미 +25,000 / 석화 8p +8,000 / 석화 15p +15,000
- → **제철 메뉴는 시즌성** 때문에 가안에서 삭제

## 변경 포인트 (현안 → 가안)

### 1. 상호명 의미 풀이
- "팔도한상" → 카테고리 헤더 **"전국 팔도의 맛을 한그릇에 담았습니다"**
- **"48시간 숙성된 특제 양념, 전국 팔도한상"** — 상호 + 기법 조합

### 2. 지역 브래킷 표준화
- 모든 메인 메뉴에 [충청도식] / [전라도식] / [경상도식] 지역 브래킷 통일

### 3. 도시락에 "1인분" 명시
- "낙지볶음 도시락" → "낙지볶음 1인분 도시락"

### 4. 할인 문구 삭제
- "[오픈 3,000원 할인]" 제거 — 이벤트성 문구는 배너/별도 노출
- 대신 포지셔닝 브래킷([불향가득 한식], 지역명)으로 대체

### 5. 비빔밥 → 덮밥 라벨 변경
- "낙지비빔밥" → "낙지덮밥 (비빔밥)"
- "쭈꾸미 비빔밥" → "쭈꾸미덮밥 (비빔밥)"
- 메인 라벨 "덮밥" + 보조 "(비빔밥)" 형식

### 6. "맛 선택" → "맵기 선택" 명확화

### 7. 옵션 그룹명 브랜딩
- "한상 추가메뉴 추가선택" → "팔도한상 추가메뉴"
- 상호명을 옵션 그룹명에 활용

### 8. 제철 메뉴(과메기/석화) 가안에서 제거
- 시즌성 메뉴는 고정 옵션에서 배제 — 재고/원가 관리 용이

## 학습 인사이트 (이 사례에서 추출)

1. **상호명 풀이 카피**: "팔도한상" → "전국 팔도의 맛을 한그릇에 담았습니다" — 상호의 의미를 카테고리 헤더에서 풀어주기
2. **지역 브래킷 시리즈**: [충청도식], [전라도식], [경상도식] — 지역 장르 라벨링으로 "골라 먹는 재미" 연출
3. **상호명을 옵션명에 재활용**: "팔도한상 추가메뉴" — 상호가 옵션 선택지 시점에도 노출
4. **할인 문구는 메뉴명에서 제거**: "[오픈 3,000원 할인]" 같은 이벤트 문구는 가안에서 배제 → 메뉴명은 항구적 정체성만
5. **숙성 시간 수치화**: "48시간 숙성된 특제 양념" — 숫자가 들어간 기법 어필
6. **비빔밥 → 덮밥 라벨 변경**: "덮밥 (비빔밥)" 이중 표기 — 검색 유입 확대 + 실제 형태 보조 표기
7. **제철 메뉴는 가안에서 배제**: 과메기/석화 등 시즌 의존 메뉴는 고정 메뉴판에서 삭제 → 재고·원가 리스크 감소
8. **맛 선택 라벨 표준화**: "맛 선택" → "맵기 선택" — 선택지 의미 명확화
