# 신만선횟집 — 솔루션 보고서 학습 노트

## 회사 표준 양식 (TWO CHOICE 투초이스)

**표지 슬로건**: "대표님의 성공이 저희의 성공입니다 / 사장님의 가게 노출부터 클릭, 그리고 주문전환까지"

**섹션 구조** (총 3장):
1. **적용 필수 솔루션** — 노출/클릭/주문 깔때기 (실행 액션)
2. **추가 수집 필요 내용** — 담당자 → 대표 대화형 질문 리스트
3. **상권·기존데이터 분석 결과** — 지도 + 인구통계

→ KPI 박스나 비교표가 아닌 **"깔때기 단계별 액션 체크리스트"** 형식

## 2. 추가 수집 필요 내용 (담당자 → 대표 질문)

1) 배달과 매장 메뉴가격 동일한지?
2) 가게 어필 포인트 / 운영 철학 (상세히)
3) 커스텀 메뉴 진행 — 셀프 2가지 모듬회 / 셀프 3가지 모듬회 (판매 가능 어종 전달 요청)
4) **배달 메뉴판 전체 구성 전달 필요** — 표준 가격표 형식 (예시):
   - 광어/우럭/모듬회: 1인/소 32,000/중 49,000/대 68,000/특대 90,000
   - 물회: 1인 18,000/소 30,000/중 50,000/대 70,000
5) 모듬회·광어·우럭 '소'짜는 혼술 메뉴인지?
6) 광어·우럭·물회·회덮밥·야채무침 타 업장 대비 장점

→ **계약 직후 대표에게 보내는 질문지** 성격. 대표 답변 받은 후 메뉴판 가안 작성.

## 학습 인사이트 (이 사례에서 추출)

1. **양식 구조**: KPI/비교표 아닌 **"노출 → 클릭 → 주문" 깔때기** + 추가수집 + 상권분석 3장 구조
2. **CPC 단가는 업종 표준**: 회 카테고리 = 551원 (프리미엄)
3. **3사 최소주문금액 통일**(19,000원) — 광고 세팅의 핵심
4. **첫주문 할인 3,000원**이 표준 첫 진입 쿠폰
5. **우리가게 NOW 수치 점검**: 주문접수율(33%)·운영지수가 노출 알고리즘에 직결
6. **추가 수집 질문 섹션 = 대화형 워크플로우**: 솔루션 보고서 자체가 "1차 진단 + 대표 인터뷰 가이드" 역할
7. **상권 분석**: 배후세대·연령대·주거형태가 광고 타겟·가격대 결정 근거
8. **3단 배너 제작비 55,000원** — 별도 비용 항목 명시
9. **메뉴 키워딩은 가안 전달로 분리** — 솔루션 보고서에는 키워드 변경 디테일 안 들어감 (별도 산출물)
10. **회사명 노출**: 표지에 "TWO CHOICE" 브랜드 일관 표기
