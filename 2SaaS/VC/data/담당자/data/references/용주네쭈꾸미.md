## 변경 포인트

### 1. "불맛" → "불향" 카피 통일
- 현안: 불맛쭈꾸미볶음 → 가안: 불향 쭈꾸미볶음 (메뉴명 주도 키워드 변경)
- 단, 대표 메뉴명 "불맛 쭈꾸미볶음"은 유지 / 콘치즈/보쌈은 "불향"으로

### 2. 메뉴명에 [용도] 브래킷
- [1인 한식] / [분식 느낌] / [야식/술안주 맞춤] / [한식당 감성] / [푸짐한 한식밥상]

### 3. 사이드 메뉴명 카피화
- 계란찜 → 매운맛 잡는 계란찜 (기능 설명)
- 튀김만두(8p) → 바삭바삭 튀김만두 8P

### 4. 카테고리 헤더 카피화
- "쭈꾸미 혼밥 메뉴" → "실속 가득, 용주네쭈꾸미 혼밥메뉴"
- "용주네 쭈꾸미 메뉴" → "쭈꾸미 하나만 바라본 용주네"
- "사이드메뉴" → "쭈꾸미볶음 필수 추가 사이드메뉴"
- "추가메뉴" → "한숟가락 부족할 때, 추가메뉴"
- "커플 메뉴" → "용주네쭈꾸미 2인 세트메뉴"

### 5. 커피 선택 옵션 분기
- 세트 기본 포함 커피(옵션 2, 필수) vs 별도 추가 커피(옵션 3, 선택, +2,500~3,500)

### 6. 맵기 옵션 3단계 (12번)
- 간장맛(매운맛 못 드시는 분/아이들용) / 양념 기본(신라면보다 조금 더 매운) / 매운맛 추가(땀 뻘뻘 화끈한) +1,000
- 설명이 드라마틱 + 타깃 명시

### 7. 에이드 탄산수 분리 제공
- "배달 중 탄산이 빠지는 것을 방지하기 위해 탄산수는 따로 제공됩니다" 필수 안내 옵션(6번)
- 에이드 메뉴(옵션 13)에서 탄산수 따로 제공 표기 (자몽/블루레몬/청포도 각 4,000)

### 8. 공기밥 개수 옵션 추가 (16번, 필수)
- 밥 X 기본 없음 / 1개 1,000 / 2개 2,000 / 3개 3,000

### 9. 리뷰이벤트 "참여하지않음" 기본 옵션
- 리뷰이벤트를 필수로 두되 "참여하지않음"으로 오팅 가능

## 학습 인사이트

1. **단일 메뉴 전문점(쭈꾸미)** — 카테고리 헤더에 "쭈꾸미 하나만 바라본 용주네" — 전문성 직접 호출
2. **미니카페 병행 운영** — 아메리카노~바닐라라떼 5종, 식사+커피 세트로 가치 제안
3. **옵션 그룹 대량(16개)** — 다수 필수 옵션(1,4,5,6,8,9,10,11,12,16 = 10개 필수)
4. **맵기 설명의 드라마틱 카피** — "간장맛(아이들용)/양념 기본(신라면+α)/매운맛 추가(땀 뻘뻘 화끈한)"
5. **에이드 탄산수 분리 제공 안내 필수 옵션** — 배달 품질 문제 사전 동의
6. **"불맛" vs "불향"** — 대표 메뉴는 "불맛" 유지, 파생 메뉴는 "불향"으로 카피 분리 (불맛이 더 강한 브랜드자산)
7. **용도 브래킷**: [1인 한식]/[분식 느낌]/[야식/술안주 맞춤]/[한식당 감성]/[푸짐한 한식밥상] — 각 메뉴를 구매 동기별로 라벨링
8. **세트메뉴 3단 구성** — 세트1(+튀김만두) / 세트2(+주먹밥) / 보쌈정식 — 업셀 단계 명확
9. **옵션명도 가게 카피** — "식후 카페인 한모금"(커피 추가), "한숟가락 부족할 때"(추가메뉴), "필수 추가 사이드메뉴" — 일관된 소비자 지향 말투
10. **음료 브랜드 풍부** — 펩시/사이다/탐스제로파인/코코팜(복숭아/화이트요구르트) + 에이드 + 주류 + 미니카페 = 음료 다양성 극대화
