# 도시피자 — 메뉴판 가안 학습 노트

## 핵심 정체성 (가게 메시지)
- **"72시간' 숙성된 흑미도우 [베이직 피자]"** — 도우 숙성 시간 + 흑미 차별화
- **"내 맘대로 고르는 [반반피자]"** — 커스터마이징 자유
- **"가성비 가득, 푸짐한 토핑 [프리미엄 피자]"** — 가성비 + 양
- **"도시피자만의 [시그니쳐 메뉴]"** — 시그니처 라인
- **"사이드메뉴로 '진짜' 완성되는 도시피자"** — 사이드 중요성 강조
- **"도시 = CITY"** 브랜드 정체성 (메뉴명 여러 곳 "CITY" 대체 표기)

## 변경 포인트 (현안 → 가안 전략)

1. **카테고리 헤더에 "'72시간' 숙성된 흑미도우" 차별화 핵심 삽입** — 도우 제조 강조
2. **"도시/시티/도심 → CITY" 브랜드 통일** — 영문 표기로 시그니처 구분
3. **"[리얼치즈] → [치즈폭탄]"** — "리얼"보다 "폭탄"으로 양 강조
4. **"한판가득 → 토핑폭탄"** 동일 의미 더 강한 표현
5. **[비건을 위한] 타겟 카피 추가** — 야채피자에 타겟 명시
6. **"사거리피자" 네이밍 유지** — 4가지맛 단순화로 이해도 상승
7. **스파게티 "[피자 사이드]" 프레임** — 단독 주문보다 피자 곁들임
8. **사이드 재정렬** — "버팔로윙 → [추천 조합] 버팔로윙"
9. **1+1 세트 옵션 강화** — 대량 할인 패키지
10. **메뉴가격 변동 없음** — 네이밍/카테고리/옵션 구조 개선만

## 학습 인사이트

1. **피자는 카테고리가 "베이직 < 프리미엄 < 시그니처" 3단 구조가 표준**
2. **도우 제조 방식(72시간 숙성 흑미)을 카테고리 헤더에 노출** — 품질 시그널
3. **"[폭탄]" 접미 패턴** (치즈폭탄/감자폭탄/토핑폭탄) — 양 어필 키워드
4. **브랜드 영문화(도시 → CITY)** — 시그니처 차별화 수단
5. **사이드 수량 3단 표기(1p/5p/10p)** — 피자 사이드 관행
6. **반반/사거리 피자 — 피자 업종 커스터마이징 핵심 라인**
7. **"1+1" 대규모 번들 (두판 세트)** — 피자 업종 특수 프로모션
8. **옵션이 피자별로 직접 연결되지 않고 커스텀 옵션 방대** — 반반·세트 선택 옵션이 수백 개
9. **"사이드메뉴로 '진짜' 완성되는"** 카피 — 피자 단독 부족 → 사이드 부스팅
10. **피자 사이즈 2단(M 10인치 / L 12인치)** — 피자 업종 표준
