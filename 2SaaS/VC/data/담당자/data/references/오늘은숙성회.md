## 변경 포인트 (현안 → 가안)

### 1. 카테고리를 TPO 카피로 재작성
- "사시미" → "매장에서 직접 숙성한 '오늘은숙성회'"
- "간단안주" → "가볍게 즐기는 간단안주"
- "구이류" → "불향 가득 구이요리"

### 2. 사시미 메뉴에 [콘셉트 키워드] 추가
- 포지셔닝([1인 혼술안주], [프리미엄 숙성 활어]) / 조리([다시마 숙성]) / 장르([오늘은 일식])

### 3. 사이드 메뉴를 옵션으로 흡수
- 현안: "사이드" 카테고리 별도 나열
- 가안: 사시미 메뉴 옵션 3번으로 통합 — 주문 시 자연스럽게 유도

### 4. 리뷰이벤트 옵션 "ri뷰 el벤트" 표기
- "리뷰/이벤트"를 "ri뷰 / el벤트"로 표기 — 금칙어 회피 + 시각적 포인트

### 5. 국물요리/간단안주 옵션으로 객단가 부스터
- 모든 사시미 메뉴 옵션에 국물요리(20,000~25,000) + 간단안주(12,000~18,000) 추가
- 사시미 1건 주문 시 +30,000~50,000 유도

### 6. 추가메뉴 분리(현안) → 통합(가안)
- 현안: "면추가" / "추가메뉴" / "사이드" 별도 카테고리
- 가안: 모두 옵션 그룹으로 흡수

## 학습 인사이트 (이 사례에서 추출)

1. **회/이자카야 업종**: 조각 수 기반 3단 사이즈(12p/20p/30p) 표준
2. **숙성 포지셔닝**: "매장에서 직접 숙성" 카테고리 헤더 + [다시마 숙성] 개별 메뉴 브래킷 — 차별화 자산
3. **TPO 브래킷**: [1인 혼술안주], [해장], [국민 술안주], [횟집 마지막 코스] — 어떤 상황에 어떤 메뉴인지 명시
4. **리뷰이벤트 표기**: "ri뷰 el벤트" — 플랫폼 금칙어 회피용 변형 표기
5. **사이드 → 옵션 흡수**: 독립 카테고리 사이드를 메인 메뉴 옵션으로 이동 → 객단가 상승
6. **국물요리/간단안주 이중 옵션**: 메인 메뉴 1건당 객단가 부스터 옵션 2개 세트(국물+안주)
7. **원재료 수식어**: "(특)대방어", "[프리미엄 숙성 활어]" — 등급 표기로 가격 수용성 확보
