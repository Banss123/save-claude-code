## 변경 포인트 (현안 → 가안)

### 1. 영문 메뉴명 → 한글 + [콘셉트 브래킷]

### 2. 하프앤하프 피자 신설 (24,000원)
- 현안에 없던 "반반 피자" 옵션 추가 — [24시간 숙성도우] 브랜드 메시지 부착

### 3. 1인(Personal 6인치) 세트 옵션 체계 구축
- 6인치 1인 피자 11종을 세트 옵션(20번)으로 통합 → 1인 주문 시 추가 피자 선택 가능

### 4. 하프앤하프 피자 선택 옵션 독립 운영
- 12인치 반반 피자 11종 조합을 옵션 13, 26번에 복제 배치 — 업셀 옵션으로 활용

### 5. 사이드 세트 28,000원 신설
- 피자 없이 사이드만으로 주문 가능한 세트 추가 (칠리 콘 카르네 등)

## 학습 인사이트

1. **양식/피자 업종은 영문+한글 병기 → 가안에서 한글 주도 + 영문 보조** (배달 가독성)
2. **지역명 브랜딩 피자 라인업** — 뉴욕/밀워키/텍사스/하와이안/인디애나/PA — 콘셉트 헤더로 스토리 부여
3. **1인 전용 카테고리(Personal 6인치) 별도 운영** — 혼밥 진입점 (8,300~9,800원)
4. **세트메뉴 옵션 복제 패턴** — 1인세트·2인세트에 동일 옵션셋(피자 선택, 사이드, 하프앤하프, 피클, 맵기, 소스, 하와이키키 토핑)을 복제 배치하여 각 세트 커스터마이징 자유도 확보
5. **할 & 할 피자**는 별도 메뉴가 아닌 옵션으로 운영 — 피자 선택 시 11종 반반 조합으로 확장
6. **[24시간 숙성도우]**같은 제조 공정 브래킷은 품질 신뢰 확보용
7. **[양식 혼밥]**은 카테고리가 아닌 메뉴명 브래킷으로 시그널 (1인용 리조또/파스타)
