## 변경 포인트 (현안 → 가안)

### 1. 카테고리 헤더 ":: ~ ::" 패턴
- "메인메뉴" → ":: 한국인 취향저격 :: 베트남쌀국수"
- 베트남 현지 포지셔닝 + 한국인 적응성 메시지

### 2. 쌀국수 메뉴명에 "베트남" 접미사
- "우삼겹 쌀국수" → "하노이 우삼겹쌀국수"
- "소고기 모듬 쌀국수" → "소고기 모듬 베트남쌀국수"
- 현지성 강화

### 3. 현안에만 있던 메뉴 가안에서 제거
- 분짜(14,900) / 껌땀 돼지갈비 덮밥(14,900) — 가안에서 삭제
- 메뉴 집중화 (라인업 단순화)

### 4. 사이드를 옵션 1번으로 우선 배치
- "수제 짜조 군만두 2p +7,000" 등 사이드 3종을 옵션 1번 "필수 사이드메뉴"로 이동 → 주문 시 자동 유도

### 5. "매콤 하노이 짬뽕쌀국수" → "[매운 야식] 얼큰 해장짬뽕쌀국수"
- TPO(야식) + 기능(해장) 브래킷 추가

### 6. 소스 "단짠" 수식
- "해선장소스" → "[단짠] 해선장소스" — 소스 맛 프로파일 명시

### 7. 반찬/소스 O/X 필수 옵션
- "반찬&소스 XX" / "반찬&소스 OO" (+500원) — 선택권 필수화 + 500원 업셀

### 8. "무료 면 추가" 홍보
- 쌀국수 옵션에 "무료 면 추가" 0원 — 양 어필 + 추가 주문 유도

## 학습 인사이트 (이 사례에서 추출)

1. **카테고리 ":: ~ ::" 패턴**: 콜론 2개를 구분자로 사용 — 헤더 가독성 향상 + 장르 구분
2. **현지성 + 로컬라이징 동시**: "한국인 취향저격" + "하노이식" — 현지감과 친숙함 동시 어필
3. **메뉴명 현지 접미사**: "쌀국수" → "베트남쌀국수" — 업종 정체성 반복 노출
4. **사이드 → 옵션 1번 이동**: 주문 시 첫 옵션으로 사이드를 보여 객단가 부스터
5. **"필수 사이드메뉴" 표기**: 옵션 헤더에서 "필수"라는 단어로 사이드 주문 유도 (실제 필수 아님에도 심리적 압력)
6. **메뉴 라인업 단순화**: 현안 일부 메뉴(분짜, 껌땀) 가안에서 제거 → 집중화
7. **"무료 면 추가" 마케팅**: 옵션에 0원 추가 항목 노출로 양 어필
8. **반찬 O/X 필수 + 업셀**: XX(0원) / OO(500원) — 선택을 강제하면서도 업셀
9. **소스 맛 프로파일 브래킷**: [단짠] 해선장 — 소스에도 맛 브래킷 부착
10. **TPO + 기능 중복 브래킷**: [매운 야식] 얼큰 해장짬뽕 — 브래킷과 메뉴명 둘 다에 기능 중복 어필
