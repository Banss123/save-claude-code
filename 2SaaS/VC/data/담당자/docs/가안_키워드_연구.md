# 메뉴판 가안 키워드 연구

> 작성일: 2026-04-11  
> 적용 대상: 육회·뭉티기·육사시미 전문 업장 (대흥육회 기준으로 최초 적용)

---

## 핵심 원칙

**배달 육회의 주문 장벽 = 신선도 불안**

소비자 구매결정 기준 (2024 오픈서베이):
1. 맛 (26.9%)
2. 가격 (19.2%)
3. 품질 (15.7%)
4. 안전·신선도 (11.8%) ← 배달 생고기는 이 항목이 주문 전환의 핵심 허들

→ 메뉴명에서 신선도·품질 신호를 최대로 발신해야 주문 전환율 올라감.

---

## 검증된 키워드 조합

### 1순위: `[당일도축]`
- **왜 효과적인가**: 배달 육회 소비자의 최대 불안 = "도착했을 때 신선한가?" → 당일도축이 이를 직접 해소
- **사실 기반 여부**: 뭉티기·육사시미 = 우둔·사태살로 법적 사전절취부위 → 당일도축 판매가 법적으로 가능. 과장 아님.
- **사용처**: 오아시스마켓 "맛있소 당일도축 암소한우 1등급 한우 육회 뭉티기", 장가한우 "당일도축 한우뭉티기 육사시미", 트리플에프 "당일 도축! 한우 육사시미 뭉티기"

### 2순위: `암소` / `암소한우`
- **왜 효과적인가**: 소비자 인식 = 암소 > 수소 (육질 더 부드럽고 고급스럽다는 인식)
- **조합 형태**: "암소한우" 또는 "한우 암소"로 사용
- **사용처**: 오아시스마켓 "1등급이상 한우 암소 우둔 육회", 팔도감 "신선한 1+ 한우 암소 육회", 엄마네한우 "1등급 한우 육사시미+뭉티기"

### 3순위: `우둔` (부위명 명시)
- **왜 효과적인가**: 부위명 명시 = 투명성·신뢰 강화. 알고 주문하는 느낌 제공.
- **적용 대상**: 뭉티기·육사시미 (우둔살 사용 부위)
- **육회는 부위 혼용** → 부위명보다 "암소한우" 조합이 적합

### 4순위: 용량 명시 (`200g`, `300g`)
- **왜 효과적인가**: 가격 대비 양 판단 가능 → 주문 결정 촉진
- **형태**: 메뉴명 말미에 용량 병기

## 메뉴별 가안 키워드 적용 기준

| 메뉴 | 적용 키워드 | 우선순위 이유 |
|------|------------|------------|
| 한상세트 | [당일도축] + 암소한우 + 구성 전면배치 | 세트 = 고관여 주문. 구성 명시로 가치 체감 |
| 뭉티기한상 | [당일도축] + 암소한우 + 뭉티기 앞배치 | 뭉티기가 당일도축 사실 기반 부위 → 신뢰도 극대화 |
| 뭉티기 단품 | [당일도축] + 암소 + 우둔 + 200g | 부위명까지 명시 → 투명성 최고 수준 |
| 육사시미 단품 | [당일도축] + 암소 + 우둔 + 200g | 동일 논리. 육사시미 인지도 보완 |
| 한우육회 단품 | [당일도축] + 암소 + 한우육회 + 200g | '한우육회'가 배달앱 내 1순위 검색어 → 그대로 유지하되 앞에 신선도 키워드 추가 |
| 음료·주류 | 변경 없음 | 신선도 키워드 무의미한 카테고리 |

## 적용하지 말아야 할 키워드

| 키워드 | 이유 |
|--------|------|
| `프리미엄` | 추상적. 소비자 신뢰 낮음 |
| `신선한` | 형용사 → 구체성 없음. [당일도축]이 대체 |
| `국내산` | 한우면 국내산 당연 → 중복 정보 |
| 등급 미확인 시 `1등급`, `1++` | 사실 불확인 시 표기 금지. 실제 등급 확인 후 사용 |
