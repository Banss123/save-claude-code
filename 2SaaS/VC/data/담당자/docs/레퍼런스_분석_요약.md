## Phase A: 구조 분석 결과

### 공통 구조 (6건 모두)

- **표지/요약**: KPI 4박스 + Before/After 비교표 + 핵심 메시지
- **섹션**: 원형 번호(①②③...) + 진녹색 배경 헤더 박스
- **내부**: 번호형 불릿 리스트 + 화살표(→) 설명
- **색상**: 녹색 계열 (#2B7A4B 강조, #1E3A32 제목)
- **수수료**: 별도 테이블 + "미달성 시 수수료 미발생" 강조

### 차이점

| 항목 | 구형 (갓꾸운닭, 강시떡) | 최신 (더피플버거) |
|------|----------------------|-----------------|
| 폰트 | 맑은 고딕 (추정) | Pretendard Variable |
| 비교표 열 | 2열 (지금/앞으로) | 3열 (구분/지금/앞으로) |
| 표지 | 없음 (바로 시작) | 별도 표지 페이지 |

→ **최신 기준(더피플버거)으로 통일 결정**
