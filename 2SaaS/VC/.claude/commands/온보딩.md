**매장**: $ARGUMENTS

위 매장에 대해 온보딩 전체 파이프라인을 실행해 주세요.

실행 지침:
1. 기본은 `onboarding-full` 스킬을 한 번만 호출하는 **원샷 경로**를 사용합니다.
2. 사용자가 `--skip-scrape`를 넘긴 경우 기존 스크래핑 결과를 재사용합니다.
3. `--cuisine`, `--location`이 주어지면 그대로 전달합니다. 없으면 기본값(`돈까스·회·일식` / 빈 문자열)을 사용합니다.
4. 실행 중 실패하면 즉시 멈추고, 에러 원인·복구 방법을 한국어 존댓말로 보고하세요.
5. 성공 시 최종 산출 파일 경로 + 검수 요약(passed/failed) + 총 소요 시간을 보고하세요.
