# 매장 현황 점검

특정 매장의 GBP 운영 현황을 종합 점검한다.

## 사용법

`/store-check [매장명]` 또는 매장명 없이 실행하면 목록에서 선택.

## 실행 흐름

1. 해당 매장 `store-info.md` 로드
2. 매장 폴더 내 파일 탐색 (리뷰답변/, 소식글/ 등)
3. 진단 항목 체크:
   - 마지막 리뷰 답변 날짜
   - 마지막 소식글 날짜
   - store-info.md 키워드/포맷 설정 완성도
   - 의료법 적용 매장: 규제 준수 이력
4. 현황 요약 + 즉시 실행 필요 항목 출력

## 출력 포맷

```
[매장명] 현황 점검

리뷰 답변: 마지막 YYYY-MM-DD (X일 전)
소식글: 마지막 YYYY-MM-DD (X일 전)
프로필 설정: 완료/미완료 항목 목록

즉시 필요:
- ...
```
