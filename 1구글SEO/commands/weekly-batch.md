# 주간 배치 작업

작업자 매장들의 주간 반복 업무를 일괄 처리한다.

## 실행 흐름

1. 사용자에게 이번 주 처리할 매장 목록 확인
2. 각 매장의 `store-info.md` 로드
3. 매장별 순차 처리:
   - 리뷰 답변 생성 (`review-reply` 스킬)
   - 리뷰 답변 검수 (`review-check` 스킬)
   - 주간보고 요약 생성 (`weekly-report` 스킬, 대상 매장만)
4. 전체 결과 요약 보고

## 대상 매장

작업자 매장 전체 (CLAUDE.md 매장 구분 참조).
주간보고 대상: 재주좋은치과, 벨라르시, 셀린의원 홍대점, 셀린의원 명동점.
