# 신규 매장 온보딩

`store-onboarding` 스킬을 실행하여 신규 매장 전체 패키지를 한번에 생성한다.

## 실행 흐름

1. 매장 기본 정보 수집 (매장명, 업종, 지역, 계약유형, 언어, 주력 서비스, 의료법 적용여부)
2. `store-onboarding` 스킬 실행 → 5개 항목 생성
3. 폴더 구조 생성 (`구글SEO/[계약유형] 매장명/`)
4. `store-info.md` 저장
5. CLAUDE.md 매장 구분 테이블에 신규 매장 추가 제안
6. 온보딩 체크리스트(`_templates/온보딩-체크리스트.md`) 기준 누락 확인
