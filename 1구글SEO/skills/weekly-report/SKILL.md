# 주간보고 특이사항 생성

## Overview

소식글 내용 + 사용자가 전달한 특이사항을 바탕으로 주간보고용 특이사항 요약을 생성한다.

## 대상 매장

재주좋은치과, 벨라르시, 셀린의원 홍대점, 셀린의원 명동점

## 입력

1. 소식글 내용 (해당 주에 발행한 소식글)
2. 사용자가 전달하는 특이사항 메모 (리뷰 대응, 방식 변경 등)

## 출력 형식

특이사항만 짧고 간결하게 작성. 아래 구조 준수:

```
📌특이사항
[소식글 핵심 내용 요약 1~2문장]

[발행 방식/추가 작업 내용 1~2문장]
```

## 작성 규칙

- 짧고 간결하게. 불필요한 수식어 배제
- 소식글에서 핵심 서비스명/메뉴명 + 소구 포인트만 추출
- 발행 방식(방문자 업데이트, 일반 리뷰 등) 언급 시 포함
- 리뷰 답글 대응 등 추가 작업이 있으면 별도 문장으로 추가
- 다국어 발행 시 몇 개 국어인지 명시
- 의료 매장은 의료법 광고 규제 표현 준수
- 줄바꿈 스타일: 한 호흡 단위로 짧게 끊어서 줄 넘김. 문단 사이 한 줄 띄움 (모바일 가독성)

## 톤

- 보고 톤: 간결한 서술체 (~했습니다)
- 업종별 톤 차이 없음 — 주간보고는 내부 보고용이므로 통일

## 예시

벨라르시:
```
📌특이사항
가일컷과 가일펌 스타일을 안내하는 소식글을 발행했습니다. 다운펌으로 옆 라인을 차분하게 정리하여 숱이 많거나 모발이 뜨는 고객님도 깔끔한 실루엣을 유지할 수 있음을 소구했습니다.

일반 리뷰보다 안정적인 방문자 업데이트 방식으로 발행하였습니다.
```

재주좋은치과:
```
📌특이사항
앞니 벌어짐 개선 시 불필요한 크라운 치료 대신 치아 삭제가 없는 레진 수복의 장점을 부각했습니다. 0.5mm 단위의 정밀한 진단과 치아 비율을 고려한 보존적 진료 철학을 3개 국어로 발행했습니다.

방문자 업데이트 방식을 채택하였으며, 새롭게 등록된 오가닉 리뷰에 대해 답글 대응을 완료하였습니다.
```

셀린의원 명동점:
```
📌특이사항
봄철 기후 변화에 따른 LDM 진정 관리와 여행객 맞춤형 슈링크 유니버스 등 소식글 2건을 발행했습니다.

일반 리뷰보다 안전한 방문자 업데이트 방식으로 발행하였습니다.
```

셀린의원 홍대점:
```
📌특이사항
슈링크·울쎄라·인모드 리프팅 1:1 맞춤 진료,
젠틀맥스프로 레이저제모 여성 의사 전담 케어
소식글 2건을 영어, 일본어, 중국어 3개 국어로
발행했습니다.
```
