# 자동화 트리거

> 아래 조건 중 하나라도 해당하면 **즉시 자동화 스크립트·스킬 사용 제안**. 사용자 요청 기다리지 말 것.

## 트리거 조건

1. 동일 수동 작업이 대화 내 **3회 이상 반복**
2. 파일 이동, 포맷 변환, 텍스트 복붙 패턴 감지
3. 언어별 썸네일 문구 분리 같은 **구조 고정 반복 작업**
4. "매번", "항상", "또", "다시" 표현 사용 시
5. 리뷰 원고 작성/검수/확인 언급 시 → **`review-copy-check` 자동 실행**

## 자동화 제안 거절 시

해당 조건을 기록하고 이후 같은 패턴에서 자동 제안하지 않음 (사용자 결정 존중).

## 관련 스킬

- `review-reply`, `review-check`, `review-copy-check`: 리뷰 업무
- `post-check`, `thumbnail-copy`, `thumbnail-figma`: 소식글·썸네일 업무
- `weekly-report`: 주간 일괄 업무
- `store-onboarding`: 신규 매장 온보딩 일괄 패키지
