# 구글맵 SEO 키워드 원칙

## 키워드 분류

- **업종 키워드**: 매장 업종 대표어 (예: 강남 피부과, 안국역 한식)
- **지역 키워드**: 행정구·지하철역·상권명 (예: 홍대, 연남동, 압구정)
- **서비스·메뉴 키워드**: 개별 시술·메뉴명 (예: 리쥬란, 닭한마리)
- **복합 키워드**: 업종+지역 조합 (예: 압구정 피부과, 안국역 맛집)

## 적용 원칙

- 모든 소식글·리뷰 답변·비즈니스 설명에 **매장 메인 키워드 + 지역명** 자연 삽입
- 키워드 밀도 과다 금지 (스팸 감지 위험)
- 문맥에 맞게 **분산 배치** — 같은 문단에 동일 키워드 2회 이상 금지
- 동일 키워드 2회째 등장 시 유의어·지시어로 교체

## 외국어 키워드 처리

- 외국어 키워드는 해당 언어권 구어체로 재작성 (번역 아님)
- 일부 한자·한글 병기 허용 (예: `韓國醫美(한국의미)`) — 한국어 독자에게 의미 전달 목적
- 제목·썸네일 문구에서는 병기 지양, 본문 내 문맥 설명 시 허용

## 리뷰 답변 내 키워드

- 매장명·지역명·핵심 서비스명 답변 내 1~2회 자연 삽입
- 키워드 전용 나열이 아닌 문장 흐름 내 녹여내기

## 썸네일 문구 내 키워드

- 제목줄: 서비스·메뉴명 직접 사용 (매장명은 지양)
- 설명줄: 맥락 설명, 키워드 강요 금지

## 매장별 지정 키워드

각 매장 `store-info.md`의 `키워드` 섹션 참조 (업종/지역/복합/외국어 키워드).
