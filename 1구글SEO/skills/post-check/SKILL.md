# 소식글 검수

## Overview

작성된 소식글 원고를 받아 SEO 정합성, 의료법, 모바일 가독성을 검수한다.
소식글 원고 작성 또는 수령 후 반드시 실행. 작업자 매장 자체 작성 원고에도 동일 적용.

## 실행 순서

1. 매장명으로 `store-info.md` 읽기 — 지정 키워드, 업종, 서론/결론 포맷 확인
2. store-info.md에 `영어 검수 체크리스트` 섹션이 있으면 → Step 4 실행 플래그 켬
3. Step 0 → Step 1 → Step 2 → Step 3 순서대로 검수
4. 플래그 켜진 경우 → Step 4 영어 정밀 검수 실행
5. 문제 항목을 심각도 순으로 정리해서 출력
6. 수정 원고 제공 (문제 항목만 재작성)

---

## Step 0 — 서론/결론 포맷 체크 (필수 선행, 언어별 전수 확인)

다국어 원고의 모든 언어에 대해 각각 확인. 하나라도 누락/불일치 시 즉시 수정 후 다음 단계 진행.

- [ ] 각 언어별 서론이 store-info.md 지정 포맷과 정확히 일치하는가
- [ ] 각 언어별 결론이 store-info.md 지정 포맷과 정확히 일치하는가
- [ ] 작업 언어 수만큼 서론/결론이 모두 존재하는가 (언어 누락 없는지)

---

## Step 1 — SEO 정합성 (최우선)

- [ ] 메인 키워드 (업종 키워드) 자연 삽입 여부
- [ ] 지역명 포함 여부
- [ ] 키워드 밀도 과다 여부 — 스팸 감지 위험, 문맥에 맞게 분산 배치
- [ ] CTA (행동 유도 문구) 포함 여부 — 예약, 방문, 문의 등

---

## Step 2 — 의료법 및 업종 규제

의료 업종(의원, 치과)에만 적용.

### 금지 표현 체크

- [ ] 최상급: "최고의", "최초", "가장 안전한", "업계 1위", "독보적"
- [ ] 시술 전후 비교: "before/after", "이렇게 달라졌어요" 류
- [ ] 효과 보장: "100% 효과", "확실한 결과", "부작용 없는"
- [ ] 과도한 할인 강조, 무료 시술 미끼 표현
- [ ] 특정 환자 사례를 일반화하는 표현

### 대체 방향

금지 표현 발견 시 아래 방향으로 수정 제안:
- 효과 보장 → 시술 과정과 특장점 중심 설명
- 최상급 → "경험 많은 의료진", "꼼꼼한 상담" 등 간접 신뢰 표현

---

## Step 3 — 모바일 가독성 및 톤

- [ ] 줄바꿈 스타일 — 한 줄에 하나의 짧은 의미 덩어리인가. 한국어 기준 한 줄 10~20자, 한 문장을 2~3줄로 나눠 리듬감 있게 구성. 각 문단 2~4줄.
- [ ] 문단 사이 한 줄 띄움 여부
- [ ] 업종별 톤 적절성:

| 업종 | 기준 톤 |
|------|--------|
| F&B | 친근, 감각적 |
| 의료 | 신뢰, 전문 |
| 뷰티 | 트렌디, 자연스러운 |
| 요리주점 | 친근, 분위기 |

- [ ] 서론/결론 포맷은 Step 0에서 처리 완료
- [ ] AI스러운 문장 없는가 — 수식어 남발, 대칭 나열, "~를 통해 ~를 경험해보세요" 류
- [ ] 대시(—, –) 사용 여부 — CLAUDE.md 규칙상 문장 연결/부연에 대시 금지. 쉼표, 마침표, 접속사로 대체해라.
- [ ] 다국어 원고 구어체 자연스러움 — 언어별로 "실제 브랜드가 이렇게 쓸까?" 기준으로 확인. 번역 티, 공문서체, 홍보 문체 없는지. 영어는 contraction 여부, 일본어는 문어체 여부, 중국어는 간체/번체 혼용 여부 기본 확인.

---

## Step 4 — 다국어 정밀 검수 (store-info.md에 다국어 정밀 검수 체크리스트 있는 매장만)

store-info.md의 체크리스트 항목을 언어별로 전수 확인하고 항목별 결과를 보고한다.

**영어**
- [ ] contraction: you are/we are/we will 등 축약 미사용 여부
- [ ] 단어 조합: texture·taste·aroma 각 영역에 맞는 수식어인지
- [ ] dangling modifier: 수식 대상이 주어와 어긋나는 구조
- [ ] 구어체 톤: present simple → progressive 전환 필요한 곳
- [ ] 원어민 감각: "이걸 실제로 쓸까?" 기준 최종 확인

**일본어**
- [ ] 키워드 범주 오류: "このレストラン" → "お店", "ここ" 여부
- [ ] 재방문 표현: また行きたい → また来たい
- [ ] 문어체: 지나친 격식체 없는지
- [ ] F&B 톤: 홍보 문체 아닌 자연스러운 브랜드 보이스인지

**중국어**
- [ ] 간체/번체 혼용: 전체 번체 통일 여부
- [ ] 직역체: "沒有失望" 류 한국어 직역 표현
- [ ] 단어 조합: 口感+風格 등 어울리지 않는 조합
- [ ] 구어체 자연스러움: 샤오홍슈·다중디엔핑 톤 기준
