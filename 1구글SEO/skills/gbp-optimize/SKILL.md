# GBP 프로필 최적화

## Overview

store-info.md를 기반으로 구글 비즈니스 프로필 내 콘텐츠 항목을 감사하고 최적화 원고를 생성한다.
랭킹 팩터 기준 고점수 항목 우선 처리.

## 입력 방법

GMB Everywhere 확장 프로그램에서 3개 리포트 PDF 제공:

| 파일 | 포함 데이터 |
|------|------------|
| Basic Audit | 카테고리(전체), 서비스 목록+내용, 운영시간, 리뷰 수/평점, Attributes |
| Website Audit | NAP 일치 여부, 카테고리→페이지 연결 현황, 타이틀/메타 분석, 운영시간 GBP vs 웹사이트 |
| Review Audit | 리뷰 응답률, 30일/6개월 유입량, 키워드 클라우드, 사진 있는 리뷰 수 |

3개 모두 제공 시 전체 감사 가능. Basic Audit만 있어도 프로필 항목 최적화 원고 생성 가능.

## 실행 순서

1. 매장명으로 `store-info.md` 읽기
2. 제공된 GMB Everywhere PDF에서 현황 데이터 추출
3. 항목별 갭 식별 (누락/미흡 항목)
4. 최적화 원고 생성
5. 항목별 출력 후 사용자에게 GBP 직접 입력 안내

## 커버 항목 (랭킹 팩터 순위 기준)

| 항목 | 랭킹 순위 | 점수 | 설명 |
|------|-----------|------|------|
| Primary GBP Category | 1 | 227 | 업종에 최적화된 메인 카테고리 추천 |
| Additional GBP Categories | 8 | 173 | 보조 카테고리 추천 (최대 9개) |
| GBP Completeness | 16 | 147 | 전체 프로필 완성도 체크 |
| GBP Predefined Services | 22 | 134 | 구글 제공 기본 서비스 항목 선택 안내 |
| Keywords in GBP Custom Service Titles | 30 | 122 | 커스텀 서비스 제목에 키워드 삽입 |
| Keywords in GBP Services | 34 | 118 | 서비스 설명에 키워드 삽입 |
| Photos Strategy | 45/74/96 | 108/95/84 | 사진 전략 체크리스트 |
| Google Q&A Section | 142 | 60 | 오너 시딩용 Q&A 세트 생성 |
| Keywords in Google Q&A | 154 | 54 | Q&A 내 키워드 자연 삽입 |
| GBP Description | 171 | 41 | 비즈니스 설명문 (750자 이내) |

## 출력 항목별 형식

### 1. 카테고리 추천
```
[메인 카테고리]
추천: (구글 GBP에서 실제 선택 가능한 카테고리명)
이유: (키워드 관련성 한 줄)

[보조 카테고리] (최대 3~5개 추천)
- (카테고리명): (이유)
```

### 2. 서비스 목록 (Custom Services)
```
[서비스 1]
제목: (키워드 포함, 30자 이내)
설명: (서비스 핵심 특장점 + 키워드, 300자 이내)

[서비스 2] ...
```

### 3. 비즈니스 설명문
```
(업종/지역/서비스 키워드 자연 포함, 750자 이내, 한국어)
```

### 4. Q&A 세트 (오너 시딩용)
```
Q: (예상 방문객 질문 — 키워드 포함)
A: (답변 — 업종/서비스/지역 키워드 자연 삽입, 구어체)
```
5~10세트 생성. 실제 자주 묻는 질문 기반.

### 5. 사진 전략 체크리스트
```
[ ] 외관 (낮/밤 2장 이상)
[ ] 내부 공간 (분위기 전달)
[ ] 대표 메뉴/서비스 (고해상도)
[ ] 직원/작업 장면
[ ] 파일명: 키워드 포함 (예: 안국역-닭칼국수-호이식.jpg)
[ ] 최근 3개월 내 신규 업로드 유지
```

## 작성 원칙

- 서비스 설명, 비즈니스 설명문: 업종/지역 키워드 store-info.md 기준 전부 포함
- 대시(—, –, --)를 문장 연결이나 부연 설명에 사용하지 마라. 마침표, 쉼표, 접속사로 대체.
- 키워드는 문장에 자연스럽게 녹임. 쑤셔 넣기 금지
- 구어체 금지 (소식글/리뷰 답변과 달리 프로필 설명은 공식 문어체 사용)
- 의료 매장: 구글작업 CLAUDE.md 의료법 광고 규제 섹션 준수
- Q&A는 실제로 방문객이 검색할 법한 질문만 — 가공의 질문 금지

## 항목 선택 모드

전체 생성이 아니라 특정 항목만 요청 가능:
- "설명문만 뽑아줘" → 비즈니스 설명문만 출력
- "Q&A 세트 만들어줘" → Q&A만 출력
- "서비스 목록 최적화해줘" → 서비스 항목만 출력
- "전체 감사해줘" → 현황 체크리스트 + 미흡 항목 전체 생성
