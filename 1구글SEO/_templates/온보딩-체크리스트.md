# 신규 매장 온보딩 체크리스트

## 1단계: 기본 정보 수집
- [ ] 매장명, 업종, 지역
- [ ] 계약 유형 (영업자/작업자)
- [ ] 다국어 운영 여부
- [ ] 의료법 적용 여부

## 2단계: 키워드 설정
- [ ] 업종 키워드 3개
- [ ] 지역 키워드 3개
- [ ] 복합 키워드(업종+지역) 3개
- [ ] 각 키워드 용도 명시 (프로필 설명 / 소식글 / 리뷰 답변)

## 3단계: 포맷 설정
- [ ] 소식글 서론/결론 문구 (한국어)
- [ ] 소식글 서론/결론 문구 (다국어: 영/중/일)
- [ ] 리뷰 답변 서론/결론 문구
- [ ] 본론 줄바꿈 규칙 확인

## 4단계: 프로필 설명문
- [ ] 구글 비즈니스 프로필 설명문 초안 (300자 이내)
- [ ] 메인 키워드 삽입 확인
- [ ] CTA 포함 확인

## 5단계: 폴더 생성
- [ ] store-info.md 작성
- [ ] 하위 폴더 생성 (소식글/리뷰답변/썸네일/키워드)
